## LF8分析（刺激される欲求）

### 1位: LF8-5（快適に暮らしたい）+ LF8-3（恐怖を免れたい）

**現状の不便（快適を求める）:**
- 「何度もまばたきしないと見えない」
- 「顔を近づけないと文字が読めない」
- 「画面との距離を調整しないと見えない」
- 「目をこすってピントを合わせようとする」
- 「首を前に出して画面を見る」

→ **「歳だから仕方ない」と諦めている不便**

**将来への不安（恐怖を免れたい）:**

**「元に戻らないかも...」という不安:**
- 「このまま悪化したら、もう元に戻らないかも...」
- 「今なら間に合うかもしれないけど、放置したら手遅れになるかも」
- 「友達は視力がどんどん落ちて、今はもう運転できない...私もそうなるのかな」

**日常生活に支障が出る不安:**
- 「買い物で値札が読めなくて、店員さんに聞くのが恥ずかしい」
- 「レストランのメニューが見えなくて、適当に注文してる」
- 「孫の顔がはっきり見えない...成長を見逃したくないのに」
- 「友達との旅行で、みんなに迷惑かけそうで不安」
- 「階段の段差が見えにくくて、転びそうで怖い」
- 「運転できなくなったら、自由がなくなる」

**自立できなくなる不安:**
- 「夫に頼りっぱなしになるのが嫌」
- 「子どもに迷惑かけたくない」
- 「自分のことは自分でやりたいのに...」

→ **「元に戻らなくなる恐怖 × 自立できなくなる恐怖」**

## LP構成パターン選定

**選定パターン:** パターンD（深刻な悩み系）

**理由:**
- LF8-3（恐怖を免れたい）が最も強い
- 健康・安全に関わる深刻な悩み
- 問題の教育を深く掘り下げる必要がある

**構成順:**
```
1. FV（体験型の未来シーン）
2. 問題の教育（深掘り）
3. 手段の教育（なぜこのサプリが良いのか）
4. 信用の教育（エビデンス・お客様の声）
5. 目的の教育（強化）
6. 投資の教育
7. 行動の教育
```

## ワクワクする未来ビジョン（目的の教育）

### 最も刺さる未来

**「諦めてた快適さが戻る」× 「将来の不安が消える」**

**理由:**
- LF8-5（快適）+ LF8-3（恐怖を免れる）
- 日常の小さなストレス解消 + 将来への安心感

**具体的な未来シーン:**
1. 何度もまばたきしなくても、文字がくっきり見える
2. 顔を近づけなくても、新聞が読める
3. スマホを遠ざけなくていい
4. 運転中、標識もはっきり見える → 不安が消える
5. 階段の段差がはっきり見える → 転ぶ心配がない
6. 「このまま悪化したら...」という不安が消える
