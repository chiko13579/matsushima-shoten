# クライアントヒアリングシート

このシートは、LP制作の前にクライアントから情報を集めるためのものです。
このヒアリングをしっかり行うことで、刺さるLPが作れます。

---

## 📋 ヒアリングの進め方

### ヒアリングの目的:
1. 商品について深く理解する
2. 既存顧客の特徴を知る
3. ターゲットを明確にする
4. 競合との差別化ポイントを見つける
5. クライアントの希望・目標を把握する

### ヒアリング時間の目安:
- 初回: 60-90分
- 追加質問: 30分程度

## 📊 ヒアリング後にやること

### 1. ヒアリング内容を整理する
- このシートを読み返す
- 重要なポイントをハイライト
- 不明点・追加質問をリストアップ

### 2. `input/01_商品情報入力.txt` に転記する
- 商品特徴
- ベネフィット
- 既存顧客の特徴
- 競合情報

### 3. ターゲットを絞る
- ヒアリング内容を元に、ターゲットを1人に絞る
- `input/01b_ターゲット分析.txt` に記入

### 4. 追加質問があれば、クライアントに確認
- メールやチャットで追加質問
- 不明点を潰す

## 💡 ヒアリングのコツ

### コツ1: 「なぜ?」を3回聞く
```
クライアント: 「この商品は、時短がウリです」
あなた: 「なぜ時短がウリなんですか?」
クライアント: 「お客様が忙しいからです」
あなた: 「なぜお客様は忙しいんですか?」
クライアント: 「仕事と家事で時間がないんです」
あなた: 「その時間がない人は、何を一番したいんですか?」
クライアント: 「家族との時間を増やしたいんです」← 本質!
```

### コツ2: 具体的なエピソードを聞く
```
× 「お客様は満足してますか?」
↓
○ 「お客様から、どんな声をもらいましたか? 印象に残ってるエピソードはありますか?」
```

### コツ3: 数字を聞く
```
× 「人気ですか?」
↓
○ 「月に何個売れてますか? どのくらいのペースで売れてますか?」
```

### コツ4: ネガティブなことも聞く
```
「競合に負けてる点はありますか?」
「お客様から、クレームや要望はありますか?」
「過去の広告で、失敗したことはありますか?」
```

**ネガティブな情報こそ、差別化のヒントになる!**
