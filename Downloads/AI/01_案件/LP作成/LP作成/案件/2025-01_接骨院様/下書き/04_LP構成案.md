### **4. 問題の教育(なぜ治らないのか)**

#### **目的**
- 「今までの治療が間違ってた」と気づかせる
- 「敵」を明確にする(間違った常識)
- 科学的根拠で納得させる

#### **構成**

**【見出し】**
```
なぜ、整体やマッサージでは
頭痛・腰痛が「戻る」のか？
```

**【パターンB: 不便列挙 + 科学的説明】**

**【NG例: 一般的な整体】**
```
一般的な整体・マッサージは
「表面」だけ治すから、すぐ戻る。

【こんな経験ありませんか?】
・整体行った直後は楽になる
・でも2,3日で元に戻る
・「また行かなきゃ…」の繰り返し
・結局、何年も通ってる
・お金も時間も使ったのに、治らない

なぜこんなことが起きるのか?
```

**【科学的説明】**
```
実は、頭痛・肩こり・腰痛の「本当の原因」は
筋肉のコリではありません。

【本当の原因】
自律神経の乱れ

自律神経が乱れると…
 ↓
血流が悪くなる
 ↓
筋肉が固まる
 ↓
頭痛・肩こり・腰痛

だから、
表面(筋肉)だけほぐしても
すぐ戻る。

内側(自律神経)を整えないと
根本的に治らないんです。
```

**【比較図】**
```
【一般的な整体】
表面(筋肉)だけ → すぐ戻る

【当院の根本治療】
表面(筋肉) + 内側(自律神経) → 戻りにくい
```

**【恐怖訴求(軽め)】**
```
「このまま放置すると…」

1年前: 数ヶ月に1回くらいの頭痛
 ↓
今: 月2,3回は頭痛

このまま放置すると、
慢性化して、毎日頭痛になるかもしれません。

【数値で見ると】
早期: 6回で改善
慢性化: 18回かかる(3倍)

「まだ我慢できる」と思ってる今が、
実は動くべきタイミング。
```

### **5. 手段の教育(当院の解決策)**

#### **目的**
- 独自性を見せる
- 「ここなら治る」と納得させる

#### **構成**

**【見出し】**
```
[院名]の
接骨院×鍼灸で「内側から治す」根本治療
```

**【3ステップ】**

**【STEP1: 接骨院で「表面」を整える】**
```
まずは表面(筋肉)の緊張をほぐします。

・首・肩・腰の筋肉の硬さを緩める
・体のバランスを整える
・即効性あり: 施術後すぐ楽になる
```

**【STEP2: 鍼灸で「内側」を整える】**
```
次に内側(自律神経)を整えます。

・鍼灸で自律神経を調整
・血流を改善
・体の「治る力」を引き出す

※鍼は髪の毛より細いので、痛くないです
```

**【STEP3: 定期的なメンテナンス】**
```
6回で「戻りにくい体」を作ります。

・1回目: 楽になる(即効性)
・3回目: 「あれ?頭痛減ったかも」
・6回目: 「最近頭痛ない」

その後は月1回のメンテナンスで
良い状態をキープ。
```

**【図解】**
```
【他院との違い】

一般的な整体:
表面だけ → すぐ戻る → 何年も通う

当院:
表面 + 内側 → 戻りにくい → 6回で改善
```
