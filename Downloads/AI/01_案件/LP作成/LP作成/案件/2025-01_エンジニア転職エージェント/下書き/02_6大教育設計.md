# 6大教育設計

## 📌 コンセプト確認

### ターゲット
**「このままでいいのか不安」を感じている中堅エンジニア(27-34歳、経験3-7年)**

### メインLF8
**LF8-3(恐怖回避): キャリアの停滞・市場価値の低下を避けたい**

### サブLF8
- LF8-8(社会的承認): エンジニアとして成長・評価されたい
- LF8-6(優越感): 同期や周りに負けたくない

### ゴール
**まずはつながりを持つこと(今すぐ転職しなくてもOK)**

## 🎯 6大教育の全体戦略

### 教育の流れ

```
① 目的の教育
   「市場価値を知ることが、キャリアの第一歩」
   ↓
② 問題の教育(恐怖喚起)
   「その"このまま"が、一番危ない」
   ↓
③ 手段の教育(解決策提示)
   「今すぐ転職じゃない。まずは情報を持つこと」
   ↓
④ 信用の教育(このサービスが最適な理由)
   「エンジニア専門だから、技術の話が通じる」
   ↓
⑤ 投資の教育(行動のハードルを下げる)
   「登録3分、相談無料、押し売りなし」
   ↓
⑥ 行動の教育(背中を押す)
   「まずは情報を知ってから、判断しよう」
```

## 1️⃣ 目的の教育

### 目的
**「転職するかどうかは、まず市場価値を知ってから決めればいい」**

### 伝えたいこと

#### ポイント1: 情報がない状態での決断は危険
```
今の会社に「残る」「転職する」を決めるのに、
一番大事なのは「情報」です。

自分の市場価値は?
他にどんな選択肢がある?
今動くべき? 待つべき?

情報がないまま決めると、後悔します。
```

#### ポイント2: 市場価値を知ることが第一歩
```
✅ 自分の市場価値を知る
✅ キャリアの選択肢を知る
✅ 今動くべきかを判断できる

これが揃って初めて、正しい決断ができます。

転職するかどうかは、その後決めればいい。
```

#### ポイント3: 今すぐ転職しなくてもOK
```
「今すぐ転職しなきゃいけない」
そんなことはありません。

大事なのは、
「情報を持った上で、自分で決める」こと。

情報収集だけでも、全然OK。
```

### キャッチコピー案
```
「転職するかどうかは、まず市場価値を知ってから決めよう」

「"このまま"か"転職"か。
 情報がない状態で決めていませんか?」

「今すぐ転職じゃない。
 まずは、自分の選択肢を知ることから」
```

## 2️⃣ 問題の教育(恐怖喚起)

### 教育パターン: パターンA(科学的・論理的説明)
LF8-3(恐怖回避)なので、パターンA

### 伝えたいこと

#### 問題1: 「このまま」が一番危ない
```
「今の会社、悪くはないから...」
「もう少し様子を見てから...」

その「このまま」が、実は一番危険です。

なぜなら、
キャリアには"動ける時期"があるから。

30代後半になると、
企業が求めるのは「即戦力+マネジメント経験」。

「経験年数だけ長い、でもマネジメント経験なし」
この状態になると、選択肢が一気に狭まります。

気づいたら、"選べる側"から"選ばれる側"になっている。
```

#### 問題2: 市場価値は、30代で決まる
```
エンジニアの市場価値は、30代で大きく決まります。

【20代】
ポテンシャル採用がある
→ 未経験でも挑戦できる

【30代前半】
経験+成長性で判断される
→ まだ選択肢が多い

【30代後半】
即戦力+マネジメント経験が求められる
→ 選択肢が狭まる

「あと数年待ってから...」
その数年が、キャリアの分岐点です。
```

#### 問題3: 技術トレンドに乗り遅れる
```
技術の変化は、想像以上に速い。

5年前のスタックが、もう古い。
「これができます」と思っていたスキルが、
企業からは「ちょっと古いね」と言われる。

レガシーな環境にいると、
気づいたら市場価値が下がっている。

「自分は大丈夫」
そう思っている人ほど、危ない。
```

#### 問題4: 情報がないまま判断している
```
「今の会社、まあまあかな」

でも、本当に?

他の会社の給料は?
開発環境は?
技術スタックは?
働き方は?

比較対象がない状態で、
「今の会社で良い」と思い込んでいませんか?

情報がないまま決めると、
「あの時動いておけば...」と後悔します。
```

### 問題の教育 - 共感フレーズ
```
😟 「今の会社、悪くはないけど...」
😟 「このままのスキルセットで大丈夫?」
😟 「同期が転職して年収が上がってるの、羨ましい」
😟 「でも、今動く理由が弱いんだよな」
😟 「何から始めればいいか分からない」

こんな風に思っていませんか?

それ、すごく分かります。
でも、その"漠然とした不安"を放置すると、
取り返しがつかないことになります。
```

## 3️⃣ 手段の教育(解決策提示)

### 核心メッセージ
**「今すぐ転職じゃない。まずは情報を持つことから」**

### 伝えたいこと

#### 手段1: まずは市場価値を知る
```
キャリアの第一歩は、
「自分の市場価値を知ること」。

✅ 自分のスキルは、他社でどう評価される?
✅ 今の年収は、適正?
✅ どんな企業が自分を欲しがる?

これを知るだけで、
「このままでいいのか」の答えが見えてきます。

転職エージェントに相談すれば、
プロの視点で市場価値を教えてくれます。
```

#### 手段2: キャリアの選択肢を知る
```
「今の会社」しか知らない状態と、
「他の選択肢も知った上で、今の会社を選ぶ」のは全然違う。

転職エージェントに登録すれば、
✅ どんな企業が募集してる?
✅ どんな技術スタックが使われてる?
✅ どんなキャリアパスがある?

これが分かるだけで、
漠然とした不安が、明確な判断に変わります。
```

#### 手段3: 転職するかどうかは、その後決める
```
誤解しないでください。

「今すぐ転職しなきゃいけない」
なんて言ってません。

まずは、情報を持つこと。
市場価値を知ること。
選択肢を知ること。

それから、
「今の会社に残る」or「転職する」
を決めればいい。

情報を持った上での決断なら、後悔しません。
```

#### 手段4: まずは「つながり」を持つ
```
エージェントに登録 = 転職
じゃないんです。

エージェントは、
「キャリアの壁打ち相手」として使えます。

✅ 今の市場価値を教えてもらう
✅ キャリアの相談をする
✅ 「今動くべきか」を一緒に考える
✅ いい求人が出たら教えてもらう

つながりを持つだけで、
キャリアの不安が減ります。

転職するかどうかは、その時に考えればいい。
```

### 手段の教育 - 具体的なステップ
```
【ステップ1】
エージェントに登録(3分)

【ステップ2】
面談で、市場価値・キャリアの選択肢を聞く

【ステップ3】
情報を持った上で、「今の会社に残る」or「転職する」を判断

【ステップ4】
転職する場合、求人紹介・面接対策をサポート
転職しない場合、つながりを持っておく(いつでも相談OK)

---

転職するかどうかは、ステップ3で決めればいい。
まずは、情報を持つことから始めましょう。
```

## 4️⃣ 信用の教育(このサービスが最適な理由)

### 核心メッセージ
**「エンジニア専門だから、技術の話が通じる」**

### 伝えたいこと

#### 信用1: アドバイザーが技術に詳しい
```
「総合型の転職エージェント、使ったことあるけど...」
「アドバイザーが技術のこと全然分かってなくて、話が通じなかった」

そんな経験、ありませんか?

当サービスのアドバイザーは、
✅ エンジニア経験者
✅ または技術知識が豊富

だから、技術の話が通じます。

「Reactの経験があります」
「Docker使ってます」
「マイクロサービスで開発してます」

こういう話が、きちんと理解できる。

技術に詳しいアドバイザーだから、
本当にマッチする求人を紹介できます。
```

#### 信用2: 開発環境・技術スタックまで把握
```
他社のエージェント:
「この会社、給料いいですよ!」(表面的)

当サービス:
「この会社、給料もいいし、
 技術スタックは○○、開発環境は○○、
 技術的負債は少なめ、チーム体制は○○です」

ここまで詳しく教えられるのは、
エンジニア専門だから。

入社後のミスマッチを防げます。
```

#### 信用3: エンジニア特化のサポート
```
当サービスでは、
一般的な面接対策だけでなく、
エンジニア特化のサポートを提供します。

✅ ポートフォリオ添削
✅ GitHubレビュー
✅ 技術面接対策
✅ 技術ブログの書き方アドバイス

「技術的に何をアピールすればいいか」
これを一緒に考えます。
```

#### 信用4: 押し売りしないスタンス
```
「転職エージェントって、押し売りされそう...」

安心してください。

当サービスは、
「今すぐ転職しなくてもOK」のスタンス。

無理に転職を勧めることはしません。

あなたのキャリアを一緒に考える、
壁打ち相手として使ってください。

「今は動かない方がいい」
そう判断したら、正直に伝えます。
```

### 差別化ポイント整理

| 項目 | 他社エージェント | 当サービス |
|------|--------------|----------|
| **技術理解** | 営業出身が多い、技術に詳しくない | エンジニア経験者・技術知識が豊富 |
| **求人紹介の深さ** | 給料・福利厚生などの表面的情報 | 開発環境・技術スタック・技術的負債まで把握 |
| **サポート内容** | 一般的な面接対策 | ポートフォリオ添削・技術面接対策 |
| **スタンス** | 早く転職させたい(成約重視) | 今すぐ転職しなくてもOK(相談重視) |

## 5️⃣ 投資の教育(行動のハードルを下げる)

### 核心メッセージ
**「登録3分、相談無料、リスクゼロ」**

### 伝えたいこと

#### 投資1: 登録は3分で完了
```
登録に必要なのは、
✅ 名前
✅ メールアドレス
✅ 簡単な職歴

たった3分で完了します。

履歴書・職務経歴書は後でOK。
まずは、つながりを持つことから。
```

#### 投資2: 相談は完全無料
```
✅ 登録無料
✅ 面談無料
✅ 求人紹介無料
✅ 面接対策無料

すべて無料です。

求職者から料金をいただくことは一切ありません。
```

#### 投資3: 転職しなくてもOK
```
「登録したら、転職しなきゃいけない?」

いいえ、全然そんなことありません。

✅ 情報収集だけでもOK
✅ 市場価値を聞くだけでもOK
✅ キャリア相談だけでもOK

転職するかどうかは、
情報を聞いてから決めればいい。
```

#### 投資4: 今動くリスク vs 動かないリスク
```
【今動くリスク】
✅ 登録に3分かかる
✅ 面談に1時間かかる

【動かないリスク】
❌ 市場価値を知らないまま、歳を重ねる
❌ キャリアの選択肢を知らないまま、今の会社に残る
❌ 気づいたら、"動ける時期"を逃している

どちらのリスクが大きいですか?

3分で始められることを、
「後でいいや」と先延ばしにすると、
取り返しがつかなくなります。
```

## 6️⃣ 行動の教育(背中を押す)

### 核心メッセージ
**「まずは情報を知ってから、判断しよう」**

### 伝えたいこと

#### 行動1: 今日から始められる
```
キャリアの第一歩は、
「情報を持つこと」。

それは、今日から始められます。

登録3分。
面談1時間。

それだけで、
「このままでいいのか」の答えが見えてきます。
```

#### 行動2: 1年後、後悔しないために
```
1年後の自分を想像してください。

【パターンA: 今動いた場合】
✅ 自分の市場価値を知っている
✅ キャリアの選択肢を複数知っている
✅ 「今の会社に残る」or「転職する」を、情報を持った上で決めた
✅ 後悔していない

【パターンB: 動かなかった場合】
❌ 市場価値を知らないまま、1年経った
❌ 「あの時動いておけば...」と後悔
❌ さらに歳を重ねて、選択肢が狭まった

どちらの1年後がいいですか?

選ぶのは、今のあなたです。
```

#### 行動3: 「このまま」か「情報を持つ」か
```
今、あなたには2つの選択肢があります。

【選択肢A】
「このまま」何もせず、様子を見る

【選択肢B】
まずは情報を持ってから、判断する

どちらを選びますか?

「このまま」は、ラクかもしれません。
でも、それが一番危ない。

情報を持った上での決断なら、後悔しません。
```

#### 行動4: 最後の背中押し
```
「まだ迷ってる...」

その迷いの正体は、
「情報がない」からです。

情報があれば、決断できます。

だから、まずは情報を持ちましょう。

登録3分。
それだけで、キャリアの不安が減ります。

転職するかどうかは、その後決めればいい。

---

今日から、キャリアの第一歩を踏み出しませんか?
```

## 📝 まとめ: 6大教育の全体像

### 教育の流れ(完成版)

```
【目的の教育】
「転職するかどうかは、まず市場価値を知ってから決めよう」
→ 「情報を持った上で決める」ことの重要性

【問題の教育】
「その"このまま"が、一番危ない」
→ キャリアの停滞・市場価値低下の恐怖を喚起

【手段の教育】
「今すぐ転職じゃない。まずは情報を持つことから」
→ まずはつながりを持つ、転職はその後決める

【信用の教育】
「エンジニア専門だから、技術の話が通じる」
→ 技術理解・エンジニア特化サポート・押し売りしないスタンス

【投資の教育】
「登録3分、相談無料、リスクゼロ」
→ 行動のハードルを極限まで下げる

【行動の教育】
「まずは情報を知ってから、判断しよう」
→ 「このまま」vs「情報を持つ」の対比で背中を押す
```

### ゴール
**登録してもらい、つながりを持つこと**
- 今すぐ転職しなくてもOK
- まずは市場価値を知る、キャリアの選択肢を知る
- その上で、転職するかどうかを判断
