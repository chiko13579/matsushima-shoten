# キャッチコピーの考え方

このファイルは、刺さるキャッチコピーを作るための思考プロセスをまとめたものです。

---

## 📋 キャッチコピー作成の5ステップ

### ステップ1: ターゲットを明確にする

**問い:**
「この特徴を好む人って、どんな人がいるんだろう?」

**やること:**
- 商品特徴から、ターゲット候補を3-5人リストアップ
- 年齢・性別・職業・生活スタイルを具体化
- 「誰に刺すのか?」を1人に絞る

**例: オールインワンファンデーション**
```
候補A: 忙しい40代女性、時短重視
候補B: めんどくさがりの30代女性
候補C: 敏感肌の50代女性

→ 選択: A(忙しい40代女性)
```

---

### ステップ2: LF8を軸に、その人の欲求を考える

**問い:**
「このターゲットは、どのLF8で動いてる?」

**やること:**
- メインLF8を特定(1つ)
- サブLF8を特定(1-2つ)
- なぜそのLF8なのか?を考える

**例: 忙しい40代女性**
```
メインLF8: LF8-5(快適でいたい)
理由: めんどくさいメイクから解放されたい

サブLF8: LF8-6(他人に後れを取りたくない)
理由: すっぴんでは外に出られない、でも時短したい
```

---

### ステップ3: ドロっとした本音を考える

**問い:**
「このターゲットは、普段何を思ってる?」

**やること:**
- 脳内会議を想像する
- 誰にも言えない本音を掘り下げる
- 「めんどくさい」「不安」「焦り」などのネガティブ感情を言語化

**例: 忙しい40代女性**
```
脳内会議:
「朝のメイク、めんどくさい...」
「でもすっぴんでは外に出られない...」
「下地、ファンデ、パウダー...面倒...」
「時短したいけど、手抜きに見られたくない...」
「プロ級の仕上がりが欲しい...」
```

**ドロっとした本音:**
- 「めんどくさい」
- 「でもすっぴんでは外に出られない」
- 「時短したいけど、手抜きに見られたくない」

---

### ステップ4: 理想の未来を考える

**問い:**
「どんなことを体験したら、嬉しいって思うのか?」

**やること:**
- 商品を使った後の理想のシーンを想像
- 「○○できた!」「○○になった!」を具体化
- ターゲットの感情(嬉しい・安心・誇らしい)を想像

**例: 忙しい40代女性**
```
理想の未来:
「朝のメイク時間が20分→3分に!」
「めんどくさいメイクから解放!」
「でもプロ級の仕上がり!」
「余った時間で、ゆっくりコーヒー飲める!」
「手抜きに見えない!」
```

---

### ステップ5: いい感じの言葉にする

**問い:**
「本音・解決策・理想の未来を、どう表現する?」

**やること:**
1. 本音を言語化
2. 解決策を提示
3. 理想の未来を見せる
4. テクニックを使う(擬音・セリフ・過去形・価格)

**例: 忙しい40代女性**

**パターンA: 本音→解決→理想**
```
「めんどくさいメイク...でもすっぴんでは外に出られない」

1本10秒で、プロ級の仕上がり。
朝のメイク時間が20分→3分に。
```

**パターンB: 過去形で理想を見せる**
```
「あれ、メイク時間3分で終わった!」

めんどくさいメイクから解放。
でもプロ級の仕上がり。
```

**パターンC: 価格+ベネフィット**
```
たった○○円で、
めんどくさいメイクから解放。

朝の20分が、自分の時間に変わる。
```

---

## 🎯 重要なポイント

### ポイント1: 「誰に刺すのか?」を1人に絞る
- ターゲットが曖昧だと、刺さらない
- 1人に絞ると、メッセージが強くなる
- 「みんなに刺そう」とすると、誰にも刺さらない

### ポイント2: ドロっとした本音を掘り下げる
- 表面的な悩みじゃダメ
- 「めんどくさい」「不安」「焦り」などの感情
- 誰にも言えない本音が、一番刺さる

### ポイント3: LF8を軸に考える
- ターゲットの欲求を理解する
- LF8が分かれば、訴求ポイントが決まる
- メインLF8とサブLF8を組み合わせる

### ポイント4: 理想の未来は「具体的なシーン」で見せる
- 抽象的な未来じゃダメ
- 「○○できた!」という体験
- 映像が浮かぶレベルで具体化

### ポイント5: テクニックを使って表現する
- 擬音(オノマトペ): パシャ、サッと、そっと
- セリフ形式: 「○○!」と言われる
- 過去形: 「○○だった」でリアリティ
- 価格: 日割り・月割りで安く見せる

## 📝 実践例

### 例1: 防水スマホケース

**ステップ1: ターゲット**
- アウトドア好きな30代男性

**ステップ2: LF8**
- メインLF8: LF8-3(恐怖回避)
- サブLF8: LF8-5(快適)

**ステップ3: ドロっとした本音**
```
「川で落としたら、スマホ終わり...」
「でも写真撮りたい...」
「スマホ壊れたら、データ消える...」
「新しいスマホ買うの、高い...」
```

**ステップ4: 理想の未来**
```
「落としても、慌てない」
「川でも、海でも、安心して写真撮れる」
「スマホも、データも、守られてる」
```

**ステップ5: いい感じの言葉に**
```
【防水スマホケース】

「落としても、慌てない。
スマホも、中身も。」

水没OK、2m落下OK
```

---

### 例2: 睡眠サプリ

**ステップ1: ターゲット**
- 不眠に悩む40代男性(仕事のストレス)

**ステップ2: LF8**
- メインLF8: LF8-3(恐怖回避)
- サブLF8: LF8-5(快適)

**ステップ3: ドロっとした本音**
```
「布団に入っても、眠れない...」
「時計を見る → まだ1時間しか経ってない...」
「明日仕事なのに...と焦る」
「この時間が、いちばんしんどい...」
```

**ステップ4: 理想の未来**
```
「いつの間にか眠りに入っていた」
「朝、スッキリ目覚めた」
「ちゃんと寝れた!」
```

**ステップ5: いい感じの言葉に**
```
夜、目が冴えてしまう。
あの時間が、いちばんしんどい。

布団に入ってから、
いつの間にか眠りに入っていた。

天然成分100%の睡眠サプリ
```

---

### 例3: オンライン英会話

**ステップ1: ターゲット**
- 英語を話せるようになりたい社会人(30代)

**ステップ2: LF8**
- メインLF8: LF8-8(社会的承認)
- サブLF8: LF8-6(優越感)

**ステップ3: ドロっとした本音**
```
「英語できたら、転職の幅が広がるのに...」
「海外で働く選択肢もあったのに...」
「英語できないから、諦めた仕事がある...」
「でも今さら勉強するの、めんどくさい...」
```

**ステップ4: 理想の未来**
```
「英語が当たり前に使える自分になる」
「転職も、海外も、選べる」
「将来の選択肢が広がる」
```

**ステップ5: いい感じの言葉に**
```
【オンライン英会話】

将来の選択肢を広げる。

たった1日200円で、
英語が「当たり前」にある毎日を。
```

## 🔥 よくある間違い

### 間違い1: 機能説明で終わる
```
❌ 「自動で温度調節するドライヤー」
↓
✅ 「温度調節のめんどくささから解放。ただ乾かすだけで、サロン帰りのツヤ髪に」
```

### 間違い2: ターゲットの本音を無視
```
❌ 「高性能なスマホケース」
↓
✅ 「落としても、慌てない。スマホも、中身も。」
```

### 間違い3: 抽象的な未来で終わる
```
❌ 「英語が上達する」
↓
✅ 「将来の選択肢を広げる。転職も、海外も、選べる人生へ」
```

### 間違い4: LF8を考えない
```
❌ 誰にでも刺さるように書く
↓
✅ 1人のLF8に絞って、深く刺す
```

## 📋 チェックリスト

キャッチコピーを書いたら、以下をチェック:

- [ ] ターゲットは1人に絞れてる?
- [ ] LF8は特定できてる?(メインLF8 + サブLF8)
- [ ] ターゲットの「ドロっとした本音」が入ってる?
- [ ] 理想の未来が「具体的なシーン」で見える?
- [ ] 機能説明じゃなくて、ベネフィットになってる?
- [ ] テクニック(擬音・セリフ・過去形・価格)を使ってる?
- [ ] 3秒で何の商品か分かる?
- [ ] 読んだ瞬間に「これ、自分のことだ!」と思わせられる?

## 💡 参考: 使えるテクニック集

### テクニック1: 擬音(オノマトペ)
- パシャ、サッと、パパッと、そっと、しっとり、ピリピリ、スーッと

### テクニック2: セリフ形式
- 「○○!」と言われる
- 「あれ、○○だ!」
- 「え、○○できた!」

### テクニック3: 過去形
- 「○○できた」
- 「○○だった」
- 「○○になった」

### テクニック4: 価格の見せ方
- 月額6,980円 → たった1日200円
- 39,800円 → 1日133円(×30日)
- 「コンビニコーヒー1杯分で」

### テクニック5: 韻を踏む
- 「そっとなじんで、しっとりだけが残る」
- 「たった○○で、ずっと○○」

### テクニック6: 「あの」「この」で共感
- 「あの時間が、いちばんしんどい」
- 「この悩み、分かる」

### テクニック7: 否定を肯定に変える
- 「ピリピリしない」→「しっとりだけが残る」
- 「もう○○しない」→「○○できる」

## 🎯 最後に

**キャッチコピーは「誰に・何を・どう伝えるか」**

1. **誰に**: ターゲットを1人に絞る
2. **何を**: ドロっとした本音・解決策・理想の未来
3. **どう伝えるか**: テクニックを使って、いい感じの言葉に

**このプロセスを繰り返せば、誰でも刺さるキャッチコピーが書けます。**
