# 引き込まれる文章の書き方ルール

---

## 🎯 **大原則: 3つの法則**

### **法則1: 口語・リアルな言葉を使う**
```
❌ NG: 書き言葉、堅い表現
✅ OK: 話し言葉、リアルなセリフ
```

### **法則2: 具体的なシーン描写**
```
❌ NG: 抽象的な説明
✅ OK: 映像が浮かぶ具体的なシーン
```

### **法則3: 感情を入れる**
```
❌ NG: 事実だけ
✅ OK: 感情・驚き・共感
```

---

## 📋 **ルール別 OK/NG 例文集**

### **ルール1: 口語・話し言葉で書く**

#### NG: 堅い書き言葉
```
「頭痛を我慢していませんか?」
「忙しくて後回しにしていませんか?」
「効果があります」
```

#### OK: 口語・リアルなセリフ
```
「また頭痛い…」
「整体行きたいけど、今日も残業だし...」
「え?嘘でしょ、これすごい」
```

**ポイント:**
- 「〜ませんか?」→「〜ですよね」「〜じゃないですか?」
- 「効果があります」→「え、すごい」「びっくり」
- 語尾に「...」「!」「?」を使う

---

### **ルール2: 抽象的ではなく、具体的なシーン**

#### NG: 抽象的な説明
```
「忙しくて後回しにしている」
「家族との時間を大切に」
「健康になる」
```

#### OK: 具体的なシーン描写
```
「整体行きたいけど、今日も残業だし...」
「日曜の朝、"パパ、公園行こう!" その誘いに、笑顔で"行こう!"と答えられる」
「朝スッキリ起きられる。階段を息切れせずに上れる」
```

**ポイント:**
- 時間を入れる(朝、夜、日曜、1年後)
- 場所を入れる(公園、リビング、階段)
- セリフを入れる(「パパ、公園行こう!」)
- 五感を入れる(触れた、見た、聞いた)

---

### **ルール3: 感情を入れる**

#### NG: 事実のみ
```
「効果がありました」
「頭痛が改善しました」
「良い商品です」
```

#### OK: 感情・驚き
```
「え?嘘でしょ、これすごい」
「びっくり。え...これすごい」
「思わず"嘘でしょ!"って鏡の前でニヤニヤしながら一人ごと」
「早く使っておけばと後悔しました」
```

**ポイント:**
- 「え?」「嘘でしょ?」「びっくり」
- 「思わず〜」「つい〜」
- 一人ごとを入れる
- 擬音を使う(ニヤニヤ、パッと、スッと)

---

### **ルール4: 「あるある」で共感させる**

#### NG: 一般論
```
「頭痛に悩んでいる人は多いです」
「忙しい人は多いです」
```

#### OK: 「あるある」で共感
```
「化粧ってほんとめんどくさい。でもすっぴんでは外に出れないし。」
「整体行きたいけど、今日も残業だし...」
「"パパ疲れてるから、今日は無理" また子供の誘いを断る...」
```

**ポイント:**
- ターゲットの脳内会議を代弁
- 「でも〜」「だって〜」のジレンマ
- 誰にも言えない本音

---

### **ルール5: セリフ形式で引き込む**

#### NG: 説明調
```
「効果を実感しました」
「家族に褒められました」
```

#### OK: セリフ形式
```
「"パパ、最近元気だね!" 子供にそう言われた時、嬉しかった」
「"え?何したの?" 友達に驚かれた」
「"あれ、最近頭痛なくなったかも" そう気づいた3回目の施術後」
```

**ポイント:**
- 他者の反応を入れる
- 「"〜"」でセリフを囲む
- 誰が言ったか明確に(子供、友達、自分)

---

### **ルール6: ビフォーアフターは「驚き」で**

#### NG: 淡々とした報告
```
「頭痛が改善しました」
「3回で効果がありました」
```

#### OK: 驚き・感動
```
「え?嘘でしょ。3回目で頭痛が半分になった」
「貼って寝て、翌朝起きたら感動」
「思わず"え?"って声を出してしまうほど全然違う」
「初回から衝撃」
```

**ポイント:**
- 「え?」「嘘でしょ?」で驚き
- 「感動」「衝撃」などの感情語
- 「思わず〜」で自然な反応

---

### **ルール7: 問題提起は「疑問形」で**

#### NG: 断定
```
「頭痛は自律神経の乱れが原因です」
「放置すると悪化します」
```

#### OK: 疑問形で引き込む
```
「なぜ、整体やマッサージでは頭痛が"戻る"のか?」
「そもそも頭痛が治らない原因は何なんでしょうか?」
「こんな経験ありませんか?」
```

**ポイント:**
- 「なぜ?」「どうして?」で好奇心を刺激
- 「〜ですか?」で読者に問いかける
- 「〜じゃないですか?」で共感

---

### **ルール8: 「実は〜」で気づきを与える**

#### NG: 普通の説明
```
「頭痛の原因は自律神経です」
「筋肉だけほぐしても治りません」
```

#### OK: 「実は〜」で気づき
```
「実は、頭痛・肩こり・腰痛の"本当の原因"は筋肉のコリではありません」
「実は今まで、治し方が間違っていただけ」
「調べてみると、実は今注目の対策のよう」
```

**ポイント:**
- 「実は〜」で新情報感
- 「本当の原因は〜」で気づき
- 「多くの人が知らない〜」で希少性

---

### **ルール9: 箇条書きで「視覚的に多い」を見せる**

#### NG: 文章で列挙
```
「化粧下地、ファンデ、コンシーラー、パウダーが必要です」
```

#### OK: 箇条書きで視覚化
```
なんとこれ1本でベースメイクは丸ごと完了。
- 日焼け止め
- 化粧下地
- ファンデーション
- コンシーラー
- パウダー
```

**ポイント:**
- 「これ1本で」「これ1台で」で全体像
- 箇条書きで視覚的に「多い」
- 数字を入れる(5役、3ステップ)

---

### **ルール10: ジレンマを見せる**

#### NG: 問題だけ
```
「頭痛がつらい」
「忙しい」
```

#### OK: ジレンマ
```
「頭痛を治したい。でも、忙しくて後回しにしちゃう」
「化粧ってめんどくさい。でもすっぴんでは外に出れないし」
「整体行きたいけど、今日も残業だし...」
```

**ポイント:**
- 「〜したい。でも〜」
- 「〜したいけど、〜できない」
- 矛盾する2つの感情

---

## 🔥 **実践: NG → OK 変換例**

### **例1: ファーストビュー**

#### NG
```
頭痛でお悩みの方へ
当院の根本治療で改善します
```

#### OK
```
「また頭痛い…」
そう言わない朝を、取り戻しませんか？

月2,3回の頭痛が、6回の施術で
「あれ?最近頭痛なくなったかも」に変わる
```

---

### **例2: 問題提起**

#### NG
```
頭痛を我慢していませんか?
放置すると悪化します
```

#### OK
```
こんなこと、ありませんか?

□ 月2,3回は頭痛がする
□ 「まぁいっか」と我慢している
□ 整体行きたいけど、忙しくて後回し

忙しいから、自分のことは後回し。
「まだ我慢できる」と思ってる。

でも、そろそろ限界じゃないですか?
```

---

### **例3: 体験談**

#### NG
```
40代女性
3回で頭痛が改善しました
```

#### OK
```
「3回目で、頭痛が半分になりました」

以前は月2,3回頭痛があって、
薬を飲んでも一時的にしか治らなかった。

でも、[院名]に通い始めて3回目くらいから、
「あれ?最近頭痛なくなったかも」と気づいて。

今は月1回も頭痛がないです。
朝スッキリ起きられるし、
家族にも「最近元気だね」と言われます。

(40代女性・主婦)
```

---

### **例4: 問題の教育**

#### NG
```
頭痛の原因は自律神経の乱れです。
筋肉だけほぐしても治りません。
```

#### OK
```
なぜ、整体やマッサージでは
頭痛・腰痛が「戻る」のか?

【こんな経験ありませんか?】
・整体行った直後は楽になる
・でも2,3日で元に戻る
・「また行かなきゃ…」の繰り返し

なぜこんなことが起きるのか?

実は、頭痛・肩こり・腰痛の"本当の原因"は
筋肉のコリではありません。

【本当の原因】
自律神経の乱れ
```

---

### **例5: 最後の背中押し**

#### NG
```
今すぐお申し込みください
予約枠に限りがあります
```

#### OK
```
申し込みは3分。
悩んでる時間の方が長い。

「また頭痛い…」
そう言わない朝を、取り戻しませんか?

P.S.
「最近元気だね」
家族にそう言われる日が来るかもしれません。
```

## 🎯 **最後に**

**引き込まれる文章 = 「読者の頭の中」を代弁する**

1. 読者の脳内会議を想像する
2. 本音・感情・ジレンマを言語化
3. 口語・具体的なシーン・感情で表現

**このルールを守れば、誰でも引き込まれる文章が書けます。**
