# 私のLP制作哲学（My Philosophy）
## 基本スタンス
* 私たちは常に「弱者・悩める者」の味方である。上から目線の教育ではなく、横に寄り添う「共感」を重視する。
* 読者を「客」として扱わず、「救うべき友人」として扱う。

## 禁止事項（NG）
* 「誰でも簡単に」「寝ていても稼げる」といった安っぽい煽りは禁止。
* 根拠のない「No.1」表記は禁止。
* 読者の恐怖を過剰に煽り、解決策を示さずに終わることは禁止。

---

## 中身開示の原則（最重要）

**「登録しないと分からない」ではなく「読むだけで価値がある」LPを作る**

### なぜ中身開示が重要か
- 先に価値を提供 → 信頼構築 → 登録ハードルが下がる
- 「すごいよ！詳しくは登録して」は怪しい、信用されない
- 中身を見せる = 自信がある証拠

### 開示すべき4要素
1. **相場・データ** - 業界の平均値、成功者の数値
2. **判断基準・チェックリスト** - 「こういう人は○○すべき」
3. **業界のリアル・裏側** - 知られていない真実
4. **成功事例の詳細** - Before/Afterの具体的な数値とプロセス

### 開示しすぎないバランス（重要）

**全てを開示しない。「一般論」と「あなた専用」を分ける。**

| 開示する（LP上で） | 開示しない（登録後） |
|------------------|-------------------|
| 一般的な相場・データ | あなた専用の診断結果 |
| 判断基準のチェックリスト | あなたに合った具体的アドバイス |
| 業界の裏側（一般論） | あなたのケースに当てはめた提案 |
| 成功事例のパターン | あなた専用のプラン |

**開示の黄金比:**
- LP: 70%（読むだけで価値がある）
- 登録後: 30%（あなた専用の価値）

**開示しすぎのNG例:**
- 全てのノウハウを教えてしまう → 登録する理由がなくなる
- 個別診断までLPでやる → サービスの価値がなくなる
- 秘密を全部バラす → 「もう十分」と思われる

**正しい流れ:**
```
LP: 「こういう情報があります」（一般論を開示）
    ↓
読者: 「へー、ためになる。ちゃんとしてる」（信頼構築）
    ↓
LP: 「でも、あなたのケースは『あなた専用』の診断が必要」
    ↓
読者: 「確かに、自分の場合どうなんだろう」（登録動機）
    ↓
CTA: 「あなた専用の○○を知る」
```

## 文章のリズム
* 一文は長くても60文字以内。
* 専門用語を使った場合は、必ず「中学生でもわかる比喩」で言い換えること。
* 語尾は「〜です。〜ます。」だけでなく、体言止めや「〜でしょう？」という問いかけを混ぜてリズムを作る。

# ナレッジファイルの使い方ガイド

## 📁 ファイル構成と使い分け

### 【構成・型を学ぶファイル】
#### 04_構成パターン.txt
**用途:** LP全体の構成を決める時に使う
**使い方:**
1. 商材とターゲットを確認
2. パターンA〜Dのどれが合うか選ぶ
3. 選んだパターンの基本構成をベースにする

**特に重要:** パターンD【記事LP型】は実証済みの勝ちパターン
- 美容・健康商材
- 40〜60代女性向け
- 新技術・新成分がある商品
に特に強い

**抽象度:** 高い（汎用性重視）
- 基本の6ステップで構成を理解
- 「この型が強い商材/弱い商材」で判断

#### 04-1_記事LP型_詳細テンプレート.txt
**用途:** パターンD【記事LP型】を選んだ場合の詳細版
**使い方:**
1. 04でパターンDを選んだ後に開く
2. 12ステップの詳細構成を確認
3. 各ステップの「書き方のコツ」「実例」を参考にする

**抽象度:** 低い（実践重視）
- 北達人コーポレーションの実例ベース
- セクションごとの具体的な書き方
- 美容・健康商材に特化

**関係性:**
```
04 (抽象化・汎用版)
 ↓ パターンDを選択
04-1 (詳細・実践版)
```

---

### 【反論処理を設計するファイル】
#### 05_反論処理フレームワーク.txt（抽象化版）
**用途:** 見込み客の疑問・不安を先回りして潰す設計をする時に使う
**使い方:**
1. 自分の商材について、14の反論のうちどれが該当するかチェック
2. 各反論に対する「証拠」を用意（権威、実績、データ等）
3. LP構成の中に配置（反論が出る「前」に潰すのがコツ）
4. 具体的な表現は06の具体例集を参考にする

**ポイント:**
- この型は商材を問わず応用可能
- 「何を証明すべきか」の思考フレームとして活用
- 証拠が足りない場合、何を補強すべきかが明確になる

#### 06_反論処理具体例集.txt（実例版）
**用途:** 反論処理の「書き方」を学ぶ時に使う
**使い方:**
1. 05で洗い出した反論処理を、どう表現するか迷った時に参照
2. 「私も○○でした」の共感パターンを学ぶ
3. 数字の使い方、感情の描写の仕方をお手本にする
4. **そのままコピペせず、自分の商材に合わせてアレンジする**

**注意:**
- あくまで「表現のお手本」として活用
- 実際の証拠がない場合は使わない（嘘・誇張は絶対NG）
- ターゲットの言葉遣いに合わせて調整する

## 🔄 実際のLP制作フロー（正しい思考順序）

**重要:** コンセプト設計 → LF8分析 → LP構成の順で考える

---

### ステップ0: コンセプト設計（最初にやること）

**コンセプトとは:** 誰に・何を・どう売るか

#### Q1: 誰に売るのか？（ターゲット設定）
- 年齢・性別
- 悩み・課題
- 生活スタイル
- 価値観

**✅ 良い例（具体的）:**
「40代後半〜50代女性、仕事・家事で忙しく、メイクに時間をかけたくないが、すっぴんでは外に出られない。手軽に綺麗になりたいと思っている」

**❌ 悪い例（抽象的）:**
「女性全般」

---

#### Q2: 何を売るのか？（商品・価値の明確化）
- 商品の機能
- 商品がもたらす結果
- 独自性（他と何が違うのか）

**✅ 良い例:**
「1本でベースメイク完了のオールインワンファンデーション。独自技術でバームからパウダーに変化。時短×プロ級の仕上がりを両立」

**❌ 悪い例:**
「オールインワンファンデーション」（だけ）

---

#### Q3: どう売るか？（切り口・ポジショニング）
- この商品を「どんな文脈」で紹介するか
- ターゲットの「どの悩み」に刺さるか
- 「どんな未来」を見せるか

**切り口の例:**
| 切り口 | ターゲット | 訴求ポイント |
|--------|----------|------------|
| 時短×高品質 | 忙しい40代女性 | 「1本10秒でプロ級」 |
| コスパ最強 | 節約志向の主婦 | 「5役で月○○円節約」 |
| 肌に優しい | 敏感肌の人 | 「○○フリー、肌ケアしながらメイク」 |
| 最新技術 | 美容マニア | 「バーム→パウダー変化の新技術」 |

**注意:** 同じ商品でも、切り口でLPの構成が変わる

---

### ステップ1: LF8分析

📄 **07_LF8逆算思考法.txt** を開く

#### やること:
1. コンセプトから「この商品を買った人は、結果的にどうなりたいのか？」を考える
2. メインLF8を特定（商品の核となる欲求）
3. サブLF8を特定（緊急度を高める追加の欲求）

**実例（ズボラ向けファンデ）:**
```
コンセプト:
- 誰に: 40代後半〜50代女性、忙しい、めんどくさがり
- 何を: 1本でベースメイク完了のファンデ
- どう売るか: 「時短×手軽に綺麗」

↓ LF8特定:
- メインLF8-5: 快適でいたい（めんどくさいから楽したい）
- サブLF8-6: 他人に後れを取りたくない（でもすっぴんでは外に出れない）
```

---

### ステップ2: LP構成パターンの自動選択

📄 **07_LF8逆算思考法.txt** の自動変換ルールを使う

#### やること:
1. メインLF8 → 問題の教育パターン（A/B）を決定
2. メインLF8 → トーン（深刻 or カジュアル）を決定
3. メインLF8 → ポジショニング（必需品 or 便利グッズ）を決定

**自動変換ルール:**
| メインLF8 | 問題の教育 | トーン | ポジショニング |
|----------|-----------|-------|-------------|
| LF8-1, 3（生存、恐怖） | パターンA（科学的説明） | 深刻、科学的 | 必需品 |
| LF8-5（快適） | パターンB（不便列挙） | カジュアル、共感 | 便利グッズ |
| LF8-6（社会的地位） | パターンA（科学的説明） | 真剣 | 自己投資 |

### ステップ3.5: 中身開示の設計（必須）

📄 **10_中身開示型LP設計法.md** を参照

#### やること:
1. **開示する内容を決める**
   - 相場・データ（業界の平均値）
   - 判断基準・チェックリスト
   - 業界のリアル・裏側
   - 成功事例の詳細

2. **開示しない内容を決める（重要）**
   - あなた専用の診断結果
   - 個別のアドバイス
   - カスタマイズされたプラン

3. **「一般論 → あなた専用」の流れを設計**
   - LP前半: 一般論を惜しみなく開示
   - LP後半: 「あなた専用は登録後」に誘導

**チェックポイント:**

- [ ] でも、登録する理由は残っているか？
- [ ] 「一般論」と「あなた専用」の境界は明確か？

## 🎯 思考順序まとめ

```
0. コンセプト設計 ← 最初にやること
   ↓
1. LF8分析（07_LF8逆算思考法.txt）
   ↓
2. LP構成自動選択（LF8 → パターン/トーン/ポジショニング）
   ↓
3. セクション詳細設計（04_構成パターン.txt）
   ↓
3.5. 中身開示の設計（10_中身開示型LP設計法.md）← 必須
   ↓
4. 反論処理組み込み（05_反論処理フレームワーク.txt）
   ↓
5. 品質向上（02_6大教育マニュアル.txt、03_参考フレーズ集.txt）
```

**重要ポイント:**
- コンセプトが決まらないと、LF8も決まらない
- LF8が決まらないと、LP構成も決まらない
- LF8を特定すれば、LP構成の80%が自動的に決まる
- **中身開示は必須**（読むだけで価値があるLPを作る）
- **開示しすぎない**（「あなた専用」は登録後に残す）

## ⚠️ 使う時の注意点

1. **型に縛られすぎない**
   - フレームワークはあくまでガイド
   - 商材・ターゲットに合わせて柔軟に調整する

2. **具体例をコピペしない**
   - 06の具体例集は「表現のお手本」
   - 自分の商材に合わせて必ずアレンジする

3. **証拠がないことは書かない**
   - 「医学誌掲載」「○○1位」等は実際にある場合のみ
   - 嘘・誇張は絶対NG

4. **ターゲットの言葉で書く**
   - 40代女性向けなら40代女性が使う言葉
   - 専門家向けなら専門用語もOK

5. **反論処理の「やりすぎ」に注意**
   - 全ての反論を詰め込むと長くなりすぎる
   - ターゲットが「本当に気にする」反論を優先する
