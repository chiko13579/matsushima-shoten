# 反論処理具体例集（北達人LP実例）

## このファイルの使い方
* 北達人コーポレーションの6本のLPから抽出した、実際に使われている反論処理の具体的な表現例
* 「こういう風に書けばいいのか」という表現のお手本として活用
* あくまで参考例。自分の商材に合わせてアレンジすること
* 抽象化された型は「05_反論処理フレームワーク.txt」を参照

## 【反論1】「本当に効果があるの?」への処理

### パターンA: 半信半疑の共感 → 驚きの体験
```
❌ 悪い例: 「効果抜群です!」

⭕ 良い例:
「本当にいいのかなって私も正直半信半疑でした。でも使ってみて本当にびっくり。」

「最初は私も半信半疑でした。でも楽天もAmazonも1位って聞くと、試す価値あるかもって思ったんです。」

「口コミが盛り上がっているとはいえ、半信半疑な私。さすがに劇的な変化があるわけないし、気休めよね、と思っていたら…びっくり。」
```

### パターンB: 感動の瞬間の描写
```
⭕ 良い例:
「嘘でしょ?あまりにも良すぎる」

「鏡の前で、え?綺麗すぎる!と思わず声が出ちゃいました。」

「思わず『嘘でしょ!』って鏡の前でニヤニヤしながら一人ごと。」

「え、すごい。こんなに簡単に対策できるなんて、まさに○○の救世主。」
```

### パターンC: 権威による証明
```
⭕ 良い例:
「世界で一番売れて、ギネス世界記録にも認定されているシリーズの最新作」

「この技術、医学誌に論文が掲載されるほどの実力なんです。」

「楽天・Amazon・Yahoo!ショッピングで3冠達成」

「プロのメイクアップアーティストも大絶賛」
```

### パターンD: 数字による証拠
```
⭕ 良い例:
「発売から2年も経たずに販売数が1,000万枚を突破」

「リピート率92%」

「なんと24秒に1個売れてる」

「発売から1日半で1万本が完売」

「おかげさまで170万個突破」
```

## 【反論8】「価格が高い」への処理

### パターンA: 価格の予想外し
```
⭕ 良い例:
「1本でパウダーまで完了できるから1万円以上するのかと思いきや、約2ヶ月分、通常価格4,422円。」

「どうせ高いんじゃないのって思いますよね。
長年の研究の成果が詰まっているだけあって、通常価格では4,422円。」

### パターンA: 価値の再定義
```
⭕ 良い例:
「それだけで1回あたり37円で使えるなんてすごくお得ですよね。」

「これはすごく節約になりそう。」

「日焼け止め、下地、ファンデ、コンシーラー、パウダー。
全部買ったら1万円超えるところ、これ1本で済むんです。」
```

### パターンC: 割引オファー
```
⭕ 良い例:
「なんとたった1分で終わる簡単なアンケートに協力すると55%オフ。」

「通常4,422円のハイテクファンデが最安値の1,982円」

「アンケートはたった5問で選択式なのですぐ終わりました。」

「モニター価格はこの公式サイト限定なので、他だと2倍以上します。」

## 応用のコツ

### コツ1: 感情の描写を具体的に
❌「良かったです」
⭕「鏡の前で思わず声が出ちゃいました」

### コツ2: 共感してから反論処理
❌「全額返金保証付きです」
⭕「私も不安でした。でも全額返金保証付きなので安心して試せました」

### コツ3: 数字は具体的に
❌「たくさん売れています」
⭕「発売から2年で1,000万枚突破」

### コツ4: 会話口調で自然に
❌「本商品は効果的です」
⭕「嘘でしょ?あまりにも良すぎる」

### コツ5: 「私も○○でした」の型を使う
「私も半信半疑でした」
「私も高いと思いました」
「私も不安でした」
「私も諦めていました」
→ 読者の気持ちを代弁してから解決策を提示

---

## 注意点

⚠️ これらの表現をそのままコピペするのではなく、自分の商材に合わせてアレンジすること
⚠️ 嘘や誇張は絶対にNG。実際の証拠がある範囲で書く
⚠️ ターゲットの言葉遣いに合わせる（40代女性なら40代女性が使う言葉）
⚠️ 「私も○○」の多用は逆効果。適度に使う
