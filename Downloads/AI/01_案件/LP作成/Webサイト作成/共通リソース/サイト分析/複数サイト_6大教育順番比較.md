# 複数Webサイトの6大教育順番比較

---

## 分析対象サイト

1. ビズリーチ（BtoB人材）
2. カオナビ（BtoBタレントマネジメント）
3. SmartHR（BtoB人事労務）
4. kintone（BtoBノーコード）
5. Dears（BtoC美容室）

---

## サイト1: ビズリーチ

### 6大教育の順番

```
1. 信用の教育（権威性） - 「No.1」バッジ
2. 目標の教育 + 信用の教育 - お客様の声スライダー
3. 信用の教育（実績） - 導入社数表示
4. 手段の教育 - 3つの成功理由
5. 信用の教育（強化） - 採用成功事例8社
6. 問題の教育（軽め） - 従来サービスとの比較表
7. 投資の教育（控えめ） - 料金プラン紹介
8. 行動の教育（控えめ） - お試し検索、お問い合わせ
```

### 特徴
- 信用の教育を4回入れている
- 問題の教育を後回しにしている
- お客様の声を2番目に配置（最速で目標イメージを見せる）

---

## サイト2: カオナビ

### 6大教育の順番

```
1. 信用の教育（権威性） - 「シェアNo.1」バッジ + 受賞歴
2. 信用の教育（社会的証明） - 導入企業ロゴスライダー（50社以上）
3. 手段の教育 - 製品説明（8機能の視覚化）
4. 目標の教育 - 活用シーン一覧（10項目）
5. 投資の教育（控えめ） - シンプルな料金体系
6. 信用の教育（強化） - 導入事例4社
7. 行動の教育（控えめ） - 資料DL、デモ申込
```

### 特徴
- 信用の教育を3回入れている（権威性 → 社会的証明 → 導入事例）
- 問題の教育がない（製品説明で直接解決策を提示）
- 目標の教育を4番目に配置（活用シーン=こうなれます）

---

## サイト3: SmartHR

### 6大教育の順番

```
1. 目標の教育 - 「業務効率化×戦略人事で事業を伸ばす」
2. 信用の教育（実績） - 「登録社数70,000社以上」+ No.1バッジ
3. 手段の教育 - 「SmartHRひとつで、すべて解決！」（3柱提示）
4. 信用の教育（強化） - 「サービス利用継続率99%」「工数88%削減」
5. 問題の教育（軽め） - 「8つの課題別ソリューション」
6. 信用の教育（導入事例） - ピクシブ、東京インキなど具体企業
7. 手段の教育（強化） - 「SmartHRが選ばれる理由」3点
8. 投資の教育（控えめ） - 料金ページへのリンク
9. 行動の教育（控えめ） - お問い合わせ、無料トライアル
```

### 特徴
- **目標の教育を最初に配置**（ビズリーチ・カオナビと違う）
- 信用の教育を3回入れている
- 問題の教育を中盤に配置（ビズリーチより早い）

---

## サイト4: kintone

### 6大教育の順番

```
1. 目標の教育 - 「ノーコードで業務アプリをつくろう」
2. 信用の教育（実績） - 3つのグラフィック（実績を視覚化）
3. 手段の教育 - 「ノーコードのキントーンが選ばれる理由」5つの特徴
4. 信用の教育（導入事例） - 日経電子版、News Picksの記事・インタビュー
5. 投資の教育（控えめ） - 月額1,800円/人、30日間無料
6. 信用の教育（強化） - 「93%の担当者が非IT部門、30,000社が導入」
7. 行動の教育（控えめ） - 資料DL、無料お試し、セミナー
```

### 特徴
- **目標の教育を最初に配置**（SmartHRと同じ）
- 信用の教育を3回入れている
- 問題の教育がない（手段の教育で直接解決策を提示）

---

## サイト5: Dears（美容室）

### 6大教育の順番

```
1. 目標の教育 - 「艶々サラサラがずっと続く」
2. 問題の教育（強め） - 5つの髪の悩み提示
3. 信用の教育（権威性） - 「施術実績No.1」「累計10万人以上」
4. 目標の教育（強化） - ビフォーアフター3事例「あなたの髪も、こんなに変わる」
5. 手段の教育 - 「Dearsが選ばれる4つの理由」
6. 信用の教育（お客様の声） - Case 1、Case 2「髪が変わったら、自分も変われました」
7. 手段の教育（強化） - アフターサポート、無料保証
8. 投資の教育（明確） - 料金表（明朗会計）
9. 行動の教育（強め） - 「予約する」ボタン複数配置
```

### 特徴
- **問題の教育を2番目に配置**（BtoBと大きく違う）
- 目標の教育を2回入れている（最初 + ビフォーアフター）
- 信用の教育を2回（実績 + お客様の声）
- **BtoCは問題の教育が早い**

## パターン分類

### パターンA: 信用→目標→手段→問題（ビズリーチ型）

**特徴:**
- 最初に信用（権威性）を見せる
- 2番目に目標+信用（お客様の声）
- 問題の教育は後回し

**適用:**
- BtoB SaaS
- 会社員がターゲット
- 上司への説明が必要

---

### パターンB: 目標→信用→手段→問題（SmartHR/kintone型）

**特徴:**
- 最初に目標（こうなれます）を見せる
- 2番目に信用（実績）
- 問題の教育は中盤または省略

**適用:**
- BtoB SaaS
- ポジティブな訴求を優先
- 問題より解決策を強調

---

### パターンC: 目標→問題→信用→手段（Dears型）

**特徴:**
- 最初に目標（理想の未来）
- 2番目に問題（悩み提示）
- 信用の教育は3番目以降

**適用:**
- BtoC
- 個人消費者がターゲット
- 悩みが明確

---

## 共通点

### 1. 信用の教育は必ず複数回入っている

全サイト共通で、信用の教育を2〜4回入れている。

**ビズリーチ:** 4回（権威性 → お客様の声 → 実績 → 導入事例）
**カオナビ:** 3回（権威性 → 企業ロゴ → 導入事例）
**SmartHR:** 3回（実績 → 数値 → 導入事例）
**kintone:** 3回（実績 → 記事 → 統計）
**Dears:** 2回（実績 → お客様の声）

---

### 2. 投資・行動の教育は控えめ

全サイト共通で、投資の教育・行動の教育は控えめ。

- 価格を強調しない
- 緊急性・限定性を使わない
- 「お気軽に」「まずは資料を」というトーン

---

### 3. 目標の教育は早い段階で入れている

全サイト共通で、目標の教育を1〜4番目に配置。

**1番目:** SmartHR、kintone、Dears
**2番目:** ビズリーチ
**4番目:** カオナビ

→ **「こうなれます」を早く見せることが重要**

---

## BtoBとBtoCの違い

### BtoB（ビズリーチ、カオナビ、SmartHR、kintone）

**共通点:**
1. 問題の教育を後回しにする、または省略
2. 信用の教育を3〜4回入れる
3. 投資・行動の教育は控えめ

**理由:**
- 比較検討段階
- 上司への説明が必要
- 不安を煽るより信頼を優先

---

### BtoC（Dears）

**特徴:**
1. 問題の教育を2番目に配置（早い）
2. 目標の教育を2回入れる（最初 + ビフォーアフター）
3. 投資の教育が明確（料金表を明示）

**理由:**
- 個人の悩みが明確
- 「私のケースに合ってるか？」を早く確認したい
- 料金が意思決定の重要要素

---

## ビズリーチの戦略が優れている理由

### 1. お客様の声を2番目に配置

**他サイト:**
- カオナビ: 6番目
- SmartHR: 6番目
- kintone: 4番目

**ビズリーチ:** 2番目

→ **最速で「こうなれる」イメージと「本当にこうなった証拠」を同時に見せる**

---

### 2. 信用の教育を4回入れている

**他サイト:** 2〜3回
**ビズリーチ:** 4回（権威性 → お客様の声 → 実績 → 導入事例）

→ **段階的に信用を強化している**

---

### 3. 問題の教育を6番目に配置

**他サイト:**
- SmartHR: 5番目
- kintone: なし
- Dears: 2番目（BtoC）

**ビズリーチ:** 6番目

→ **信用と目標を十分に伝えた後で、軽く課題を示す**

---

## 結論: Webサイトの6大教育の順番

### BtoB Webサイトの黄金パターン

```
1. 信用の教育（権威性） - No.1バッジ、受賞歴
2. 目標の教育 + 信用の教育 - お客様の声
3. 信用の教育（実績） - 導入社数、統計
4. 手段の教育 - サービスの特徴、選ばれる理由
5. 信用の教育（強化） - 導入事例、詳細な成功事例
6. 問題の教育（軽め） - 比較表、課題提示（省略も可）
7. 投資の教育（控えめ） - 料金ページへのリンク
8. 行動の教育（控えめ） - 資料DL、お問い合わせ
```

### BtoC Webサイトのパターン

```
1. 目標の教育 - 理想の未来
2. 問題の教育（強め） - 悩みの提示
3. 信用の教育（権威性） - 実績、No.1
4. 目標の教育（強化） - ビフォーアフター
5. 手段の教育 - 選ばれる理由
6. 信用の教育（お客様の声） - 具体的な事例
7. 投資の教育（明確） - 料金表
8. 行動の教育（強め） - 予約ボタン
```

---

## 重要ポイント

1. **信用の教育は2〜4回入れる**（段階的に強化）
2. **目標の教育は早い段階で**（1〜4番目）
3. **問題の教育はBtoBでは後回し、BtoCでは早め**
4. **投資・行動の教育は控えめ**（押し売り感なし）
5. **お客様の声は「目標+信用」を同時にやれる最強の教育**
