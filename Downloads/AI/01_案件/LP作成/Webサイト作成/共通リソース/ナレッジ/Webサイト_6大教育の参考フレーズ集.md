### 選ばれる理由（3〜5つ）

**BtoB SaaS:**

**ビズリーチの3つの成功理由:**
```
1. 審査を通過した会員だから採用したい即戦力人材が見つかる
   - 独自審査制度
   - 有料プラン導入による質の高さ

2. 企業の規模にかかわらず全国で即戦力採用成功の実績多数
   - 6拠点体制
   - コンサルタント支援体制

3. 料金プランが選べるからコストを最適化できる
   - 4つのプラン
```

**カオナビの選ばれる理由:**
```
1. 直感的な操作性
   - 現場の誰でも使える
   - ITスキル不要

2. 柔軟なカスタマイズ
   - 自社に合わせた項目設定
   - 評価制度に対応

3. 充実したサポート
   - 導入支援
   - 活用サポート
```

**kintoneの5つの特徴:**
```
1. ノーコードで開発
2. 豊富なテンプレート
3. 外部サービス連携
4. モバイル対応
5. セキュリティ対策
```

**表現パターン:**
- 「○○が選ばれる理由」

## 使い方のコツ

### Webサイト型の特徴

1. **冷静なトーン**
   - 感情を揺さぶらない
   - 論理的に説明

2. **比較検討を意識**
   - 他社との違いを明確に
   - 比較表を使う

3. **信用を段階的に強化**
   - 権威性 → 社会的証明 → 詳細な事例

4. **押し売り感を出さない**
   - 投資・行動の教育は控えめ
   - 「お気軽に」「まずは資料を」

---

## 注意事項

**このフレーズ集を使う時の注意:**

1. **そのままコピペしない** - 自分の商材に合わせて必ずアレンジする
2. **証拠がないことは書かない** - 「No.1」「継続率○○%」等は実際にある場合のみ
3. **BtoB と BtoC でトーンを変える**
4. **商品が分かりにくい場合は説明を先に入れる**
5. **お客様の声は「目的+信用+社会的承認」の3役を意識**
