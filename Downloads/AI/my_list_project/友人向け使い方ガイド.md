# 🚀 Web制作会社調査ツール - 使い方ガイド

## 📋 このツールでできること
- Web制作会社の情報を自動で収集
- 社長名、FVテキスト、制作実績URLなどを抽出
- Google Sheetsに自動で保存
- CSVファイルでダウンロード

## 🎯 セットアップ方法（初回のみ）

### 1. プロジェクトを受け取る
```bash
# フォルダをコピーまたはダウンロード
cd my_list_project
```

### 2. 必要なパッケージをインストール
```bash
npm install
```

### 3. サーバーを起動
```bash
node server.js
```

### 4. ブラウザでアクセス
```
http://localhost:3000
```

## 📊 Google Sheetsを使う場合

### ステップ1: スプレッドシートを作成
1. [Google Sheets](https://sheets.google.com) にアクセス
2. 新しいスプレッドシートを作成
3. タイトルを「Web制作会社リスト」などに変更

### ステップ2: 共有設定
1. 右上の「共有」ボタンをクリック
2. 以下のメールアドレスを追加：
   ```
   web-company-scraper@x-auto-poster-466609.iam.gserviceaccount.com
   ```
3. 権限は「編集者」を選択
4. 「送信」をクリック

### ステップ3: スプレッドシートIDを取得
- URLから取得します
- 例: `https://docs.google.com/spreadsheets/d/【ここの部分】/edit`
- `【ここの部分】`がスプレッドシートIDです

## 🔍 使い方

### 1. 検索を実行
1. 場所を入力（例：東京 Web制作会社）
2. 会社数を選択（5〜30社）
3. 「検索実行」をクリック

### 2. 結果を保存

#### A. CSV形式でダウンロード
- 「CSV形式」ボタンをクリック
- Excelで開けます

#### B. Google Sheetsに保存
1. スプレッドシートID欄に自分のIDを入力
   - 空欄の場合：デフォルトのシートに保存
   - ID入力時：自分のシートに保存
2. 「Google Sheetsに保存」をクリック
3. 成功したらシートが自動で開きます

## ⚠️ 注意事項

### Google Sheets保存時のエラー
**「権限エラー」が出た場合**
- 共有設定を再確認
- メールアドレスが正確に入力されているか確認
- 「編集者」権限になっているか確認

**「スプレッドシートが見つかりません」が出た場合**
- スプレッドシートIDが正しいか確認
- URLからコピーし直す

### 検索時の注意
- 1回の検索で2〜5分かかります
- 検索中はブラウザを閉じないでください
- 「実際のWeb検索を使用」にチェックを入れると本物のデータを取得

## 💡 便利な使い方

### 複数地域を一度に検索
場所欄に複数入力できます：
```
東京 Web制作会社, 大阪 Web制作会社, 名古屋 Web制作会社
```

### データの活用
1. **営業リスト**として使用
2. **競合調査**に活用
3. **パートナー探し**に利用

## 🆘 困ったときは

### サーバーが起動しない
```bash
# プロセスを終了してから再起動
pkill -f "node server.js"
node server.js
```

### ブラウザでアクセスできない
- http://localhost:3000 を確認
- ファイアウォールの設定を確認
- 別のブラウザで試す

### データが取得できない
- インターネット接続を確認
- しばらく時間をおいて再試行
- 会社数を減らして試す
