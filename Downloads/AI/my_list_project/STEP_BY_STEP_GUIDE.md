# スプレッドシート連携の詳細手順

## 🎯 目標
Google Sheetsにスクレイピングしたデータを自動出力できるようにする

## 📋 Step 1: Google Sheetsでスプレッドシートを作成

1. **Google Sheetsにアクセス**
   ```
   https://sheets.google.com
   ```

2. **新しいスプレッドシートを作成**
   - 「+」ボタンまたは「空白」をクリック
   - タイトルを「Web制作会社データ」などに変更

3. **スプレッドシートIDを取得**
   - URLをコピーします
   - 例: `https://docs.google.com/spreadsheets/d/1ABC123XYZ789/edit#gid=0`
   - この場合、`1ABC123XYZ789` がスプレッドシートIDです

## 📤 Step 2: サービスアカウントと共有

1. **共有ボタンをクリック**
   - 右上の「共有」または「Share」ボタン

2. **メールアドレスを入力**
   ```
   web-company-scraper@x-auto-poster-466609.iam.gserviceaccount.com
   ```

3. **権限を設定**
   - ドロップダウンから「編集者」を選択
   - 「送信」をクリック

4. **通知を無効にする**
   - 「メールで通知」のチェックを外す（推奨）

## 🧪 Step 3: テストの実行

### 方法A: コマンドラインで直接テスト
```bash
# あなたのスプレッドシートIDに置き換えてください
node test-existing-spreadsheet.js 1ABC123XYZ789
```

### 方法B: ウェブアプリから使用
サーバーを起動して、ウェブインターフェースから操作できます。

## 🔍 トラブルシューティング

### エラー: "The caller does not have permission"
**原因**: サービスアカウントがスプレッドシートにアクセスできない

**解決策**:
1. スプレッドシートの共有設定を再確認
2. メールアドレスが正確に入力されているか確認
3. 「編集者」権限が設定されているか確認

### エラー: "Spreadsheet not found"
**原因**: スプレッドシートIDが間違っている

**解決策**:
1. URLからスプレッドシートIDを再確認
2. スプレッドシートが削除されていないか確認

## 🚀 実際の使用方法

1. **初回設定**: 上記の手順1-2を実行
2. **データ追加**: スクレイピング実行時に自動でスプレッドシートに追加
3. **確認**: Google Sheetsでリアルタイムにデータを確認可能
