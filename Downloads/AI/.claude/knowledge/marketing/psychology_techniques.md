# 心理学コピーライティング技法 50選

## 使い方
各テクニックには「定義」「prompt」「Before/After実例」がある。
目的に合ったテクニックを選び、promptを実行してコピーを生成する。

---

## No.1 バンドワゴン効果 (Bandwagon Effect)
**定義**: 「みんなが持っている」という事実を突きつけ、思考停止状態で行列の最後尾に並ばせる集団催眠術。

**prompt**:
```
# Role: Bandwagon Master
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと強調したい数字・実績を聞く
IF 2: 商品の「流行感」の根拠を聞く（予約待ち/SNS話題/インフルエンサー愛用/発売即完売）
→ 読者が「乗り遅れる恐怖」を感じるコピーを出力
```

**実例**:
- Bad: 多くの企業様に導入されています
- Good: 上場企業の3社に1社が導入済み。もはや業界標準のセキュリティ対策です

---

## No.2 スノッブ効果 (Snob Effect)
**定義**: 「私はその他大勢とは違う」というエゴへの媚薬。入手困難・限定性を餌に優越感という名の麻薬を高値で売りつける技術。

**prompt**:
```
# Role: Snob Appeal Specialist
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと「あえて断るべき顧客層」を聞く
IF 2: 商品の「手に入らなさ」の理由を聞く（生産数限定/会員制/招待のみ/審査あり）
→ 「選ばれたあなただけ」という特別感を極限まで高めたコピーを出力
```

**実例**:
- Bad: 誰でも相談してください
- Good: 年商10億未満お断り。本気で業界トップを狙う経営者以外、このノウハウは劇薬です

---

## No.3 アンカリング効果 (Anchoring Effect)
**定義**: 最初に意図的に「極端な数字（アンカー）」を見せ、その後の提示価格を「安い」「お得」と錯覚させる脳のバグ利用。

**prompt**:
```
# Role: Price Anchor Strategist
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと比較対象となる「高額なアンカー」を聞く
IF 2: 最終的に売りたい価格と最初に見せる見せ金（高値）を聞く
→ 最初の数字が強烈な基準点となり、提示価格が破格に見えるコピーを出力
```

**実例**:
- Bad: 月額3万円で利用可能です
- Good: 人件費なら月30万かかる業務が、これなら月額たったの3万円

---

## No.4 損失回避性の法則 (Loss Aversion)
**定義**: 「得すること」より「損すること」への異常な恐怖を利用する。利益の喜びより損失の痛みを2倍強く感じる心理を突く。

**prompt**:
```
# Role: Loss Aversion Copywriter
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと「行動しなかった場合の具体的な損失」を聞く
IF 2: 避けるべき「最悪の未来」を聞く
→ 「得られる未来」ではなく「回避すべき損失」に焦点を当てたコピーを出力
```

**実例**:
- Bad: 安全なデータを守ります
- Good: 情報漏洩で1億円の損害賠償を払いますか？それとも月額1万でリスクをゼロにしますか？

---

## No.5 カクテルパーティー効果 (Cocktail Party Effect)
**定義**: 「自分のことだ！」と強制的に振り向かせる名指しの技術。「皆さん」ではなく「○○に悩む30代のあなた」と呼びかける。

**prompt**:
```
# Role: Target Sniper
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストとターゲットの「属性」や「特有の悩み」を聞く
IF 2: 誰に向けたメッセージか（ペルソナ詳細）を聞く
→ 雑多な情報の中でもターゲットが「私のことだ」と即座に反応するコピーを出力
```

**実例**:
- Bad: 痩せたい方におすすめ
- Good: 「去年のズボンが入らない」とクローゼットで溜息をついたあなた。その悩み、3週間で解決できます

---

## No.6 ハロー効果 (Halo Effect)
**定義**: 「一点豪華主義」で全体を美化する錯覚魔法。権威、外見、肩書きに引きずられて商品全体の評価を歪めさせる。

**prompt**:
```
# Role: Halo Effect Architect
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと使える「権威」「受賞歴」「有名固有名詞」を聞く
IF 2: 商品の「最も輝かしい一点」を聞く
→ 一つの際立った権威性が商品全体の信頼度を底上げするコピーを出力
```

**実例**:
- Bad: 業務効率化を支援します
- Good: 元マッキンゼーの戦略家が教える、無駄を削ぎ落とす経営術

---

## No.7 ウィンザー効果 (Windsor Effect)
**定義**: 「本人が言うより第三者の噂話」が最強の証拠。売り手からの言葉は疑われるが、第三者からの言葉は無条件に信じられる。

**prompt**:
```
# Role: 3rd Party Voice Creator
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと「誰の声として伝えると最も響くか」を聞く
IF 2: 顧客が信じる「客観的な評価」は何かを聞く
→ 売り込み臭を消し、第三者視点からの称賛として信頼を勝ち取るコピーを出力
```

**実例**:
- Bad: 弊社は風通しの良い職場です
- Good: 「辞めたくなかった」と退職者が語る会社。元社員がSNSでこっそり勧めるホワイト企業

---

## No.8 カリギュラ効果 (Caligula Effect)
**定義**: 「見るな」と言われると見たくなる天邪鬼ハック。禁止されることで逆に欲求が高まる心理的リアクタンスを利用。

**prompt**:
```
# Role: Forbidden Fruit Offer

IF 1: 対象テキストと「禁止・制限すること」を聞く
IF 2: 商品が持つ「強すぎる効果」を聞く
→ 「ダメ」と言いつつ相手の手を強引にボタンへ伸ばさせる禁止コピーを出力
```

**実例**:
- Bad: 効果的な営業手法を公開
- Good: 【悪用厳禁】競合他社には教えないでください。市場バランスを崩す恐れがあるため地域限定公開です

## No.9 認知的不協和 (Cognitive Dissonance)
**定義**: 「常識」と「事実」の矛盾で脳をバグらせる。信じている常識を否定し不快感を与え、矛盾の解消を求めてクリックさせる。

**prompt**:
```
# Role: Paradox Creator
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと「否定すべきターゲットの常識」を聞く
IF 2: 商品が実現する「あり得ない組み合わせ」を聞く
→ 脳に不快な矛盾を与え、答えを知りたくてたまらなくなるコピーを出力
```

**実例**:
- Bad: 運動して痩せましょう
- Good: まだジムで消耗してるの？運動すればするほど痩せにくくなる、人体の矛盾について

---

## No.10 返報性の原理 (Reciprocity Principle)
**定義**: 「タダより高いものはない」を地で行く恩売りの技術。まずは無料で与えまくり、断れない心理状態を作ってから本命を売る。

**prompt**:
```
# Role: Giver's Strategy
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと「先に提供できる過剰なプレゼント」を聞く
IF 2: 相手に「申し訳ない」と思わせるほどの提供価値を聞く
→ 「ここまでしてもらったんだから買わないと悪い」と思わせるコピーを出力
```

**実例**:
- Bad: 資料をダウンロードできます
- Good: 有料セミナー（5万円相当）の動画を、今だけ無料で全公開

---

## No.11 おとり効果 (Decoy Effect)
**定義**: 「松竹梅」の魔術。本命（竹）を選ばせるために、あえて誰も選ばない「割高な梅」を並べる強制選択。

**prompt**:
```
# Role: Decoy Strategist
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 一番売りたい本命商品と比較用の「捨て駒（おとり）」を聞く
IF 2: 顧客を誘導したい「真ん中の選択肢」を聞く
→ 「これしかない」と思わせる比較構造とおとり選択肢を出力
```

**実例**:
- Bad: プランA（1万）、プランB（2万）
- Good: C（5万・全部入り）、B（2万・基本機能）、A（1.8万・機能制限）→AとBの差を縮めBのお得感を演出

---

## No.12 希少性の原理 (Scarcity Principle)
**定義**: 「残り1個」に群がる飢餓本能の刺激。時間・数量・条件を極限まで絞り「今動かないと二度と手に入らない」という焦りを焚きつける。

**prompt**:
```
# Role: Scarcity Engineer
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと設定可能な「限界」を聞く
IF 2: 嘘にならずに言える「手に入らない理由」を聞く
→ 心拍数を上げ、思考停止でクリックさせる切迫感のあるコピーを出力
```

**実例**:
- Bad: 参加者募集中
- Good: 【残席2】会場の定員に達し次第、予告なく締め切ります。増席の予定はありません

---

## No.13 権威バイアス (Authority Bias)
**定義**: 「白衣を着た人」には逆らえない思考停止スイッチ。内容の正しさより「誰が言ったか」で判断する心理。肩書きという印籠で相手をひれ伏させる。

**prompt**:
```
# Role: Authority Builder
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと借りられる「虎の威」は誰かを聞く
IF 2: 商品に「お墨付き」を与える要素を聞く
→ 読者が無条件で信じざるを得ない、圧倒的権威をまとうコピーを出力
```

**実例**:
- Bad: 業務効率化を支援します
- Good: 経産省認定「DX認定事業者」が監修。国の基準をクリアした、失敗のないメソッド

---

## No.14 単純接触効果 (Mere Exposure Effect)
**定義**: 興味がない物でも、何度も目にするうちに警戒心が薄れ好意に変わる。接触頻度を上げ、相手の脳内シェアを占拠する。

**prompt**:
```
# Role: Exposure Looper
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと「これは何度目の接触用か」を聞く
IF 2: 顧客の生活導線に「割り込む」手段を聞く
→ しつこさを消し、親近感だけを残す「また会いましたね」コピーを出力
```

**実例**:
- Bad: 新製品のご案内
- Good: （件名なし）昨日の件ですが...（業務連絡を装い日常的に会話している錯覚を与える）

## No.15 初頭効果 (Primacy Effect)
**定義**: 第一印象が全て、後は誤差。最初の情報が最も強く記憶に残り、その後の情報の解釈基準になる。

```
→ 冒頭の1行で勝負を決め、続きを読ませる強制力を持ったコピーを出力
```

**実例**:
- Bad: 新しい美白化粧水が誕生
- Good: 「もう、コンシーラーはいりません。」

## No.16 親近効果・終末効果 (Recency Effect)
**定義**: 「終わり良ければ全て良し」の記憶改竄術。最後に与えられた情報は判断に最も大きな影響を与える。

**prompt**:
```
# Role: The Closer
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと最後にダメ押しする「特典」や「保証」を聞く
IF 2: 読者の背中を押す「最後の一言」を聞く
→ 読み終わった瞬間に「買う」という選択肢しか残さないクロージングコピーを出力
```

**実例**:
- Bad: よろしくお願いいたします
- Good: 追伸：もし効果が出なければ、全額返金いたします。リスクは弊社が背負いますので、ご安心ください

## No.17 フレーミング効果 (Framing Effect)
**定義**: 「物は言いよう」で価値を歪める詐術。同じ事実でも切り取り方次第で「最高」にも「最悪」にもなる。

**prompt**:
```

モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストとネガティブに聞こえがちな「事実」を聞く
IF 2: 弱点を「強み」に見せるフレームを聞く
→ ネガティブな事実をポジティブな価値に変換するコピーを出力
```

**実例**:
- Bad: 年間12万円かかります
- Good: 「1日あたり缶コーヒー1本分（330円）」で導入可能です

## No.18 ハード・トゥ・ゲット効果 (Hard-to-Get Effect)
**定義**: 「障害がある恋ほど燃える」ロミオとジュリエット脳。わざとハードルを設け、それを乗り越えさせることで商品への執着心を増幅させる。


**実例**:
- Bad: 代理店募集中
- Good: 弊社の理念に共感し、厳しい審査を通過した企業様のみと提携します

## No.19 保有効果 (Endowment Effect)
**定義**: 「自分のもの」になった瞬間、価値が跳ね上がる。一度手にした物を手放すことに強烈な苦痛を感じる心理を利用する。


**実例**:

## No.20 ディドロ効果 (Diderot Effect)
**定義**: 「一つ買ったら全部揃えないと気が済まない」統一への執着。新しい服を買うと靴や鞄が欲しくなる連鎖反応を利用。

## No.21 ツァイガルニク効果 (Zeigarnik Effect)
**定義**: 「続きはWebで」の生殺し拷問術。未完了のことを強烈に覚えている心理。情報を寸止めし、「知りたい」という空白ストレスを与えクリックを強制する。

**実例**:

## No.22 バーナム効果 (Barnum Effect)
**定義**: 「私のことをわかってくれている！」という勘違い誘発。誰にでも当てはまる曖昧な記述を、自分だけへの予言だと信じ込ませる。

## No.23 コントラスト効果 (Contrast Effect)
**定義**: 「ドブ水を見せてから水道水を売る」比較の暴力。極端に悪いものや高価なものを先に見せ、普通の商品を「最高」に見せる。

```

```

**実例**:
- Bad: 月額1万円です
- Good: 専門家を雇えば月50万。外注なら月20万。本ツールなら月1万

## No.24 選択のパラドックス・ジャムの法則 (Paradox of Choice)


**実例**:
- Bad: 10種類のプランから選べます
- Good: 「迷ったらスタンダードプラン。」9割の企業様がここからスタートしています

## No.25 IKEA効果 (IKEA Effect)
**定義**: 「苦労した子供は可愛い」の手間賃搾取。人は手間暇をかけたものに客観的な価値以上の評価を下す。あえて「仕上げ」をユーザーにやらせて愛着と価値を捏造する。

**prompt**:

## No.49 フレーミング効果・属性フレーミング (Attribute Framing)
**定義**: 「中身は同じでも、ラベルで味が変わる」言い換え術。「牛肉80%」と「脂身20%」は同じだが受ける印象は真逆。ポジティブな属性を強調するフレームで評価を上げる。

**prompt**:
```
# Role: Frame Master
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと言い換えたい「事実」を聞く
IF 2: 最も好印象を与える「角度」を聞く
→ 嘘はつかずに視点を変えるだけで魅力を最大化するフレーム書き換えコピーを出力
```

**実例**:
- Bad: 失敗率は5%です
- Good: 「95%の確率で成功します。」安心してお任せください

## No.50 メタ認知・客観視 (Metacognition)
**定義**: 「売り込みだとバレている」ことを逆手に取る。「どうせ広告でしょ？」と思っている読者に「これは広告です」と認めた上で、防御壁を内側から解除する高度な技。

**prompt**:
```
# Role: Meta Copywriter
モード選択: 1.既存テキストのリライト / 2.新規文章の作成
IF 1: 対象テキストと読者が感じている「不信感」を聞く
IF 2: ぶっちゃけるべき「売り手の事情」を聞く
→ 「ここまで正直に言うなら本物かも」と思わせる本音コピーを出力
```

**実例**:
- Bad: 弊社のサービスはNo.1です
- Good: 「正直に言います。このサービスには向いていない人がいます。」（先に弱点を言うことで信頼を獲得）
