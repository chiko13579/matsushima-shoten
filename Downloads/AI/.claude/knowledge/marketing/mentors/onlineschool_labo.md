## N1インタビュー → コピー変換の完全プロセス

### ステップ①：本音を引き出す「場の設計」（冒頭5分）

言うべき言葉：
- 「**正解・不正解はありません。ラフな本音をください**」
- 「データは社外秘で個人特定されません」
- 「答えにくいことはパスで。いつでも中断OK」

取るべきキャラクター：
- **「無知な弟子」**：「教えてください」スタンスで相手を先生役にする
- **「共犯者・共感者」**：軽い失敗談を自己開示して感情の土台を共有する
- **リアクションは普段の2倍以上**：全身で聴いていることを示す

### ステップ②：建前を剥がす3つの武器

**武器A：極端化質問（エクストリーム・クエスチョン）**
```
「魔法の薬で同じ効果が得られたら、○○（その行動）をやめますか？」

→「やめる」= 機能的欲求（健康管理・時短）
→「やめない」= 情緒的欲求（努力する自分が好き・帰属感）
```

**武器B：5Whys（なぜを5回）**

クレンジングの実例：
```
Why1「洗浄力が強く突っ張らない」
Why2「夜のケアが短くなり早く寝られる」
Why3「疲れているとイライラする」
Why4「明日、子供に優しく接したい」
Why5「完璧じゃなくても良いお母さんでいたい自分への自信」
    ↓
コピー軸：「メイク落とし」→「母親としての自信回復」
```

**武器C：本音直撃のぶっちゃけ質問**
> 「正直、痩せたらモテるし、ワンチャン何かいいことあると思いませんか？」
→ この一言でダイエットサプリ購入者が「実は異性にモテたかった」という本音を話し始めた

### ステップ③：形容詞 → 情景変換（最重要）

**ルール：形容詞（機能評価）を「具体的な人生の場面」に置き換える**

| 形容詞（NG） | 情景・ストーリー（OK） | コピー軸の変化 |
|---|---|---|
| 「掃除が楽になった」 | 「日曜朝に自分だけ掃除機をかけてイライラ。今は夫と2人でコーヒーを飲める。夫婦喧嘩がなくなった」 | 「時短」→「夫婦関係の修復」 |
| 「保湿力が高い」 | 「夕方4時、疲れた顔を見て自信を失っていた。今は『私、まだ戦える』とデスクに戻れる」 | 「美肌」→「働く自信の回復」 |
| 「節約できる」 | 「無駄なコストを払わない賢い消費者である自分への満足感」 | 「お得」→「賢者の自己肯定」 |

> 「形容詞は機能的価値の評価に過ぎず、顧客を動かす真の動機を隠す」

### ステップ④：成功のサイン3つ
1. **沈黙と自己発見**：スラスラ答える時点では建前。「考え込んだ後にポロっと出た一言」が本音
2. **仮説の好意的な裏切り**：「節約」が「賢者の自己満足」に変わった瞬間
3. **「こんな話、初めてしました」**：この発言が出たら成功

### ステップ⑤：得られた本音の全施策への転用
```
N1の「本音の言葉」（最小限の変換で使う）
├─ 広告の冒頭一言（MOL）
├─ LPのファーストビューコピー
├─ LINE配信の書き出し
├─ 営業ヒアリングの質問設計
└─ セールストークの「顧客が気づいていない理想の言語化」
```

## コンセプト圧縮の完全プロセス

> 「コンセプトとは、一言で伝えて『それいいね！』と言わせられるメッセージ」

### 2段階圧縮

**第1段階（シンプル言語化）：**
「どんな悩みを抱えた人を、どんな未来に連れていくサービスか？誰が読んでも分かる言葉にする。煽りなし、キャッチーさ不要」

**第2段階（キャッチー変換）：**
「ベネフィットや内容が一瞬で伝わる言葉に変換する」

### 圧縮成功事例（実体験）
```
【失敗①】「ホワイト企業転職可能」
【失敗②】「年収○○万円アップ」
    ↓ N1インタビュー10人実施
    ↓ 気づき：「転職したいけど踏み出せない」が主流
    ↓ 根本欲求：「将来の安全保障」「現状維持リスクの回避」

【成功】「在職中に次を検討しましょう」
    ↓ 「2〜3年後の安全確保」という本能に直撃
```

### タカが引用する圧縮の見本

|---|---|---|
| 「お母さんに嫌われないゲーム機」 | 任天堂Wii | 母親の購買決定権という現実の本能 |

## MOL・MOAフレームワーク

- **MOL（Moment of Lifting）**：顧客が興味を持ち始める「入口」。潜在ニーズが浮上する瞬間
- **MOA（Moment of Action）**：購買決定までの積み重ね段階。決定打で購買に至る

### 具体例：ブルーライトメガネ
```
状況：顧客は「暇つぶしでメガネ屋に入った」状態

MOL：「仕事するなら疲れないメガネ」
     → 「疲れないメガネってあるの？」と潜在ニーズを喚起

MOA①：「軽い」（機能情報）
MOA②：「手頃な値段」（価格障壁の除去）
MOA決定打③：「鼻の部分がぶれない」（体感できる具体的機能）→ 購買決定
```

### MOL設計のコピー原則
> 「道を歩いているターゲットにどう言葉を投げかけたら足を止めてもらえるか」

MOLで使う言葉の条件：
- 潜在ニーズを「顕在化」させる（知っていたようで知らなかった問いかけ）
- 「自分のことだ」と気づかせる
- スキップできない好奇心か意外性を含む

### MOA後の離脱防止コピー（タカ独自の気づき）
> 「顧客の頭の中では、敷居を一度超えても気が変わることがある」

組み込む言葉：
- 「いつでもブロックできます」
- 「迷惑メールには届きません」
- 「今登録すると○○が手に入ります」（即時報酬の提示）

## LPコピーライティングの哲学

> 「LPは顧客との『会話のキャッチボール』。24時間働くオンライン上の営業マン」

### 制作の思考テンプレート
制作の8割は事前思考。以下を繰り返してワイヤーフレームを作る：
```
①提示：「この情報をユーザーに投げかける」
②想像：「ユーザーはこう思う：『本当に？』『自分に関係ある？』」
③応答：「その疑念に答える次のコンテンツを置く」
→ これを繰り返すとワイヤーフレームが完成する
```

### 核心のコピー哲学：「主張するな、連想させよ」
> 「『確実にできます』ではなく『税金対策まで学べます』と背景で語ることで、顧客に自ら結論を導かせる」

| 主張型（NG） | 連想させる型（OK） |
|---|---|
| 「確実に月100万稼げます」 | 「税金対策まで学べます」（→自分で「稼げる」と結論づける） |
| 「今すぐ行動しないと損します」 | 「Aさんは在職中に始めて、6ヶ月後に独立しました」 |

### RTB（Reason to Believe）
> 「なぜ効くのかという仕組みレベルの納得感構築が成約率を左右する」

成果の主張（what）だけでなく、仕組みの説明（why/how）がRTBを形成する。

### LP7構成要素

| 要素 | 突破すべき壁 | 核心 |
|---|---|---|
| ①ヘッダー | 「見ない」の壁 | N1のインサイトを一言に凝縮。シンプルかつ強烈に |
| ②セカンドビュー | 「本当か？」の疑念 | 期待を信頼で補強する |
| ③本編 | 「自分に関係ある？」 | N1基準のベネフィット + 仕組み説明（RTB） |
| ④オファー/CTA | 行動への惰性 | 心理ハードルを下げ、不安を事前に解消する言葉 |
| ⑤やらない理由潰し | 最後の抵抗 | やらない理由を全部先回りして潰す。今すぐやる理由を明示 |
| ⑥社会的証明 | 「自分と同じ人がいるか？」 | 「**無料カウンセリング感想**」が効果的（相談への期待値が上がり申込ハードルが下がる） |
| ⑦権威付け | 「専門家が認めているか？」 | 推薦・メディア掲載・公的承認 |

### LPO基準値
| 指標 | 目標値 | 理想値 |
|---|---|---|
| ファーストビュー到達率 | 60〜70% | 80% |
| 初回CTA到達率 | 30% | — |

## 診断LP設計の心理工学

3つの心理メカニズム：
1. **自己理解欲求**：「自分はどのタイプか」という問いに誰でも答えたくなる
2. **ゲーミフィケーション**：「申し込み」より「無料診断」はゲーム感覚でハードルが低い
3. **ザイガルニク効果**：「結果をまだ見せない」→完了していない行動への強い引きつけ

### 質問設計の鉄則
| 設計要素 | 最適値 | 理由 |
|---|---|---|
| 質問数 | 16〜24問 | 少ない→信憑性低下。多い→離脱 |
| 回答形式 | イエス/ノー中心 | 直感で答えられ離脱防止 |
| 序盤 | ライトな質問 | リズムを作り没入感を高める |
| タイトル | 可能性訴求 or 危険性訴求 | 「失う未来」の方が反応しやすい |

### 最重要設計：結果を見せない間に登録させる
```
1. 診断LP上で回答（ゲーム感覚）
2. LINE誘導ページ「結果はLINEでお送りします」← ここで結果を見せない
3. LINE登録 → 外部リザルトページへ転送
4. 結果提示 + 「解決策として無料相談」をオファー
5. 申込フォーム → LINE連携で予約完了
```

実績：LINE獲得CPA約500円・月間数万件を数年継続

## 広告クリエイティブの実践コピー技術

### 冒頭の一言に全労力を注ぐ
> 「最初の2〜3秒が極めて重要。道を歩いているターゲットにどう言葉を投げかけたら足を止めてもらえるか」

冒頭パターン：
- **好奇心型**：「え、それどういう意味？」と思わせる逆説・意外な前提
- **意外性型**：「普通の前提を否定する」（「○○はやめてください」）
- **直接呼びかけ型**：「○○で悩んでいる人だけ見てください」

### 「親へのLINE」メソッド
> 「親にLINEするように、言い方を意識せず言いたいことだけをプロットに詰める」
> 「プロットができた時点で9割完了」
> 「途中で手を止めるとロジックがずれる。書き始めたら休憩禁止」

### 「広告感を消す」クリエイティブ技法
| 技法 | 内容 | 効果 |
|---|---|---|
| iPhoneメモ帳スクショ型 | 手書き風・メモアプリ風のビジュアル | オーガニック投稿に見える |
| X投稿風クリエイティブ | 投稿のスクリーンショット風デザイン | 広告と認識されにくい |
| 道を歩く動画 | 編集なしの素の映像 | SNSの日常投稿と区別がつかない |
| AIアニメ化バナー | 当たりバナーをAIでアニメ加工 | 視覚的新鮮さで獲得数増加 |

### YouTube広告の3フォーマット
| フォーマット | 特徴 |
|---|---|
| ホルダー語り型 | 顔出し・信頼構築・長期運用向け |
| 紙芝居型 | コピーのみ・顔出し不要・制作スピード重視 |
| 通販型（テレビ通販スタイル） | 「最高の獲得効率」（タカ評価）・大量獲得向け |

> 「1本作れば1年間ずっと回せる。4億円の広告費を投下した実績」

### CPA改善の数値基準
| 指標 | 目標値 |

## セールストークの具体的な言葉

### ヒアリングで使う言葉
- 「的確なアドバイスのためにたくさん質問しますが、大丈夫ですか？」（主導権確保）
- 「それって本音で言えてる感覚ありますか？」（建前を剥がす）
- 「理想が手に入る時、どんなシーンが目に浮かびますか？」（映像化）
- 「どんな感情を手に入れたいですか？」（情緒的欲求を引き出す）

### クロージングで使う言葉
- 「お互いの時間のためにも、今日やるかやらないか、どっちでもいいので決めてくださいね」（「検討します」を封じる）
- 「私に任せてください。後悔させません」（最後の一押し）

### 3つのNOを突破する設計
| NOの壁 | 突破の言葉・設計 |
|---|---|
| 見ない（スキップ） | 冒頭2秒の意外性・好奇心・直接呼びかけ |
| 信じない（疑念） | RTB（仕組み説明）・社会的証明・第三者の声 |
| 行動しない（惰性） | やらない理由潰し・今すぐやる理由・不安の先回り解消 |

> 「営業の成功はスキルよりも『マインド』によって決まります」
> 「このサービスは絶対にこのお客様の役に立つという確信が、最後の土俵際で勝敗を分ける最も大事な要素」
> 「成約に至らない原因の多くはクロージングの弱さではなく、前半のヒアリングが浅いこと」

## 右脳マーケティングの実践体系

> 「爆発的な売上（億単位）を生むアイデアは左脳の論理分析ではなく、右脳の直感創造性から生まれる」
> 「右脳は左脳より100倍以上賢い。1日5,000回の意思決定の99%を処理している」

### 右脳起動3ステップ
1. **「脳みそがちぎれるくらい」インプット**：ターゲット・競合・市場を徹底的に詰め込む
2. **「潔く寝かせる」**：20分集中後、執着を捨てて放置（3〜7日）。脳の主導権が右脳へ移行
3. **「降りてきたものをキャッチ」**：アイデアは一瞬で消えるため常にスマホを手元に

### 左脳オフ環境の設計
- 入浴・散歩：外部刺激遮断
- 瞑想：ノイズキャンセリング + 528Hz + 「綿あめ機」で雑念を払うイメージ（10分）→ 感謝ワーク（5分）
- 作業中BGM：40Hz音源 → 左右脳ハイブリッド状態
- ポジティブな雑談（食べ物・笑い）：感情が動くと15分前に出なかった企画案が浮上する

## コピーを書く時の実践順序

```
1. N1を実在する1人に絞る（「ドンピシャ客」）
2. 建前を剥がす（形容詞禁止・5Whys・極端化質問・ぶっちゃけ質問）
3. 根本欲求（本能レベル）に到達する
4. 形容詞 → 情景・ストーリーに変換する
5. コンセプトを「一言で『それいいね』と言わせる」まで圧縮
```
