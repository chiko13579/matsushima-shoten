# Webサイト構成の黄金パターン

---

## 基本構造

**多くのWebサイトは、この順番で構成されている**

```
【FV (ファーストビュー)】
├─ 目的の教育（メインメッセージ）
└─ 手段の教育（さらっと）

【FV直下】
1. 信用の教育（権威性） - No.1バッジ、受賞歴
2. （商品が分かりにくい場合）説明
3. 目的の教育（詳細）
4. 問題の教育（軽め）
5. 手段の教育（詳細）
6. 目的の教育（強化） + 不安払拭 - お客様の声・導入事例
7. 投資の教育（控えめ）
8. 行動の教育（控えめ）
```


## 重要ポイント

1. **FVには必ず「目的 + 手段」が入っている**
2. **商品が分かりにくいものは、先に説明が入る**
3. **目的の教育は2回入っている**（FV → 手段の後に強化 + 不安払拭）
4. **手段の教育は2回入っている**（さらっと → 詳細）
5. **信用の教育は権威性のみ（1回）** ※コラム・ブログで長期的に信頼構築している場合も多い
6. **お客様の声 = 目的の教育（強化） + 不安払拭**
7. **BtoBは問題の教育を後回し、BtoCは早め**
8. **投資・行動の教育は控えめ**（押し売り感なし）

---

## お客様の声の役割

**配置:** 手段の教育の後

**お客様の声が果たす役割:**
1. **目的の教育（強化）**: 「本当にこうなれる」証拠を見せる
2. **不安払拭**: 「自分と同じ悩みを持った人が成功してる」安心感を与える
3. **社会的承認**: 「社内で認められる」というイメージを与える（BtoB）

**流れ:**
```
手段の教育: なぜこのサービスが良いのか
  ↓
  「でも本当にこうなれるの？」（不安）
  ↓
お客様の声: 実際にこうなった人がいます
```

→ **手段を説明した後に、「本当にこうなった証拠」を見せる**

---

## BtoB と BtoC の違い

### BtoB

**特徴:**
1. 問題の教育は後回し、または省略
2. 信用の教育は権威性のみ（1回）
3. 投資・行動の教育は控えめ

**理由:**
- 比較検討段階
- 上司への説明が必要
- 不安を煽るより信頼を優先

---

### BtoC

**特徴:**
1. 問題の教育を最初に配置（早い）
2. 目的の教育を2回入れる（最初 + お客様の声で強化）
3. 投資の教育が明確（料金表を明示）

**理由:**
- 個人の悩みが明確
- 「私のケースに合ってるか？」を早く確認したい
- 料金が意思決定の重要要素

---

## 信用の教育の2パターン

### パターン1: Webサイト内で完結（黄金パターン通り）

```
【FV直下】
1. 信用の教育（権威性） - No.1バッジ、受賞歴
```

**例:**
- ビズリーチ: No.1バッジ
- カオナビ: シェアNo.1 + 受賞歴
- SmartHR: 登録社数70,000社以上

→ **即座に信頼を獲得**

---

### パターン2: コラム・ブログで長期的に信頼構築

**やり方:**
- 専門知識を発信するブログ
- 業界トレンドを解説するコラム
- ノウハウ記事（SEO対策）

**狙い:**
- 「この会社は専門家だ」と思わせる
- Google検索から流入
- 信頼を積み重ねてからWebサイトへ誘導

**例:**
- kintone: 「業務効率化ノウハウ」記事多数
- SmartHR: 「人事労務の基礎知識」コラム充実

→ **専門性で信頼構築（長期戦略）**

---

### 両方やると最強

```
【Webサイト外】
コラム・ブログで専門性を発信 → 信頼構築（長期）
  ↓
【Webサイト内】
権威性バッジで即座に信頼獲得（短期）
```

→ **信頼の二段構え**
