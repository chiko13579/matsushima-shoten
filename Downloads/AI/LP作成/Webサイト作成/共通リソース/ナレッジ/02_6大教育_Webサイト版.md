# 6大教育 - Webサイト版

---

## LPとWebサイトの6大教育の違い

| 教育 | LP | Webサイト |
|------|-----|-----------|
| **信用の教育** | 社会的証明（お客様の声、実績数） | 会社情報、代表プロフィール、資格、受賞歴 |
| **目標の教育** | 理想の未来（「こうなれます！」） | サービス紹介（「こうなれます」落ち着いたトーン） |
| **問題の教育** | 深掘り（感情を揺さぶる） | **解決策の提示（課題→解決策の選択肢→自社の強み）** |
| **手段の教育** | 独自性・仕組み・エビデンス | サービスの特徴・他との違い・選ばれる理由 |
| **投資の教育** | 価格・オファー・特典 | **控えめ**（料金ページに記載、押し売り感なし） |
| **行動の教育** | 緊急性・限定性・リスクリバーサル | **控えめ**（お問い合わせボタン、押し売りしない） |

---

## 1. 信用の教育（Webサイト版）

### 目的
「この会社は信頼できる」と思わせる

### やること

**会社情報ページ:**
- 会社概要（設立年、所在地、代表者名）
- 代表プロフィール（経歴、想い、顔写真）
- 資格・認定（業界団体への加盟、認定資格）
- 受賞歴・メディア掲載

**実績:**
- お客様の声（具体的な成果）
- 事例（ビフォーアフター、導入事例）
- 取引先企業（ロゴ掲載）
- 数字で見せる（○○件の実績、満足度○○%）

**例:**
```
### 会社情報

【設立】2015年
【代表】山田太郎
【資格】○○認定講師、○○資格保持

### 実績

- お客様満足度95%
- 年間500件以上の施術実績
- リピート率80%
```

## 2. 目標の教育（Webサイト版）

### 目的
「こうなれる」という理想の未来を見せる

### やること

**サービス紹介ページ:**
- このサービスを使うと、どうなれるか
- お客様の声で、実際にどうなったかを見せる
- ビフォーアフターで視覚的に見せる

**トーン:**
- LPのように「こうなれます！」と強く言わない
- 「こうなれます」と落ち着いたトーンで伝える

**例:**
```
### 当店の縮毛矯正で、こうなれます

- 毎朝のスタイリング時間が10分短縮
- 雨の日でも髪が広がらない
- ツヤのあるまっすぐな髪に

### お客様の声

「毎朝のスタイリングが楽になりました。雨の日でも髪が広がらなくて、本当に助かっています」
（30代女性）
```

## 3. 問題の教育（Webサイト版）- 最重要

### LPとの違い

**LP:**
- 問題を深掘りして、感情を揺さぶる
- 「やばいかも...」「誰か助けて...」と思わせる

**Webサイト:**
- **課題に対してどんな解決策があるかを明記**
- 選択肢を提示して、自社の強みを伝える

---

### やること

**ステップ1: お客様の課題を明確にする**

```
### こんなお悩みはありませんか？

☑ ○○で困っている
☑ ○○がうまくいかない
☑ ○○したいけど、できない
```

**ステップ2: 解決策の選択肢を提示**

```
### 解決策は3つあります

1. **○○をする**
   → メリット
   → デメリット

2. **○○をする**
   → メリット
   → デメリット

3. **○○をする**
   → メリット
   → デメリット
```

**ステップ3: 自社の強みを伝える**

```
### 当社では、○○という方法で解決します

従来の○○は、○○が課題でした。

しかし、当社では○○という技術を使うことで、
○○を実現できます。

だから、
→ ○○ができる
→ ○○が解消される
→ ○○になれる
```

---

### 具体例（美容院の縮毛矯正）

```
### こんなお悩みはありませんか？

☑ 縮毛矯正を繰り返していて、髪が傷んでいる
☑ 毎朝のスタイリングに時間がかかる
☑ 雨の日は髪が広がってしまう

### 解決策は3つあります

1. **縮毛矯正をする**
   → 髪がまっすぐになり、スタイリングが楽になる
   → ただし、髪へのダメージが心配

2. **トリートメントで髪質を改善する**
   → 髪が扱いやすくなる
   → ただし、根本的な解決にはならない

3. **ストレートパーマをかける**
   → 縮毛矯正より自然な仕上がり
   → ただし、効果が長持ちしない

### 当店の縮毛矯正は、髪へのダメージを最小限に

従来の縮毛矯正は、強い薬剤を使うため、髪が傷みやすいのが欠点でした。

しかし、当店では○○という技術を使うことで、
髪へのダメージを最小限に抑えながら、しっかりとクセを伸ばすことができます。

だから、
→ 髪が傷まない
→ ツヤが出る
→ 長持ちする（平均6ヶ月）
```

## 4. 手段の教育（Webサイト版）

### 目的
「なぜこのサービスが良いのか」を伝える

### やること

**サービスの特徴:**
- 他社との違い
- 独自の技術・手法
- こだわり

**選ばれる理由:**
- お客様の声で裏付け
- 実績で裏付け

**例:**
```
### 当店が選ばれる3つの理由

1. **髪へのダメージを最小限にする技術**
   独自の○○技術で、髪へのダメージを最小限に抑えます。

2. **年間500件以上の実績**
   縮毛矯正を年間500件以上担当しているため、安心してお任せいただけます。

3. **アフターケアのアドバイス**
   施術後、自宅でのケア方法をアドバイスします。
```

## 5. 投資の教育（Webサイト版）- 控えめ

### LPとの違い

**LP:**
- 価格を強調（「今だけ○○円！」）
- オファー・特典を前面に

**Webサイト:**
- 料金ページに記載
- 押し売り感を出さない

---

### やること

**料金ページ:**
- 料金表を明記
- 「なぜこの価格なのか」を説明
- オプションがあれば記載

**トーン:**
- 「お得です！」と強調しない
- 「適正価格です」と伝える

**例:**
```
### 料金

| メニュー | 料金 |
|---------|------|
| カット | 5,000円 |
| カット + カラー | 10,000円 |
| 縮毛矯正 | 15,000円〜 |

※髪の長さによって料金が変わります。
※初回の方は、カウンセリング無料です。
```

## 6. 行動の教育（Webサイト版）- 控えめ

### LPとの違い

**LP:**
- 緊急性・限定性（「今だけ！」「残り○個！」）
- リスクリバーサル（「全額返金保証！」）

**Webサイト:**
- お問い合わせボタンを設置
- 押し売り感を出さない

---

### やること

**お問い合わせボタン:**
- 各ページに設置
- 「お気軽にお問い合わせください」
- 「まずはご相談ください」

**トーン:**
- 「今すぐ！」と急かさない
- 「お気軽に」と柔らかく

**例:**
```
【お問い合わせボタン】

「まずはお気軽にご相談ください」
「無料カウンセリング実施中」
```

## まとめ: Webサイトでの6大教育

| 教育 | やること | トーン |
|------|---------|--------|
| **信用** | 会社情報、実績、資格 | 信頼できることを伝える |
| **目標** | サービス紹介（こうなれます） | 落ち着いたトーン |
| **問題** | **課題→解決策の選択肢→自社の強み** | **冷静に、選択肢を提示** |
| **手段** | サービスの特徴・違い | なぜ良いのかを伝える |
| **投資** | 料金ページに記載 | **控えめ、押し売りしない** |
| **行動** | お問い合わせボタン | **控えめ、急かさない** |

**重要ポイント:**
- 問題の教育では、課題に対する解決策を明記
- 投資・行動の教育は控えめ
- 押し売り感を出さず、信頼を構築する
