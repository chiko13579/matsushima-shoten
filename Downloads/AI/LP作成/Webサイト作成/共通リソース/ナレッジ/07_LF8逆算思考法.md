# LF8逆算思考法: 商材からLP構成を自動設計する

## 本質

**「この商材を欲しいと思う人は、どのLF8が刺激されているか?」**
→ この問いに答えるだけで、LP構成・トーン・問題の教育のアプローチが自動的に決まる。

---

## LF8とは（Life Force 8: 人間の8大欲求）

人間が生まれながらに持っている、抗えない8つの欲求。
マーケティングはこの欲求を刺激することで購買行動を引き起こす。

### LF8の一覧

| LF8 | 欲求 | 刺激例 |
|-----|------|-------|
| LF8-1 | 生き残りたい、人生を楽しみたい | 健康、安全、QOL向上 |
| LF8-2 | 美味しいものを食べたい・飲みたい | グルメ、高級食材 |
| LF8-3 | 恐怖・痛み・危険を免れたい | コンプレックス解消、病気予防 |
| LF8-4 | 性的に交わりたい | 異性にモテたい、魅力的に見られたい |
| LF8-5 | 快適でいたい | 時短、便利、ストレス軽減 |
| LF8-6 | 他人に勝る、世の中に後れを取りたくない | 承認欲求、社会的地位 |
| LF8-7 | 愛する人を気遣い、守りたい | 家族のため、自分を大切にする |
| LF8-8 | 社会で認められたい | 評価、尊敬、ステータス |

**LP制作で特に重要なのは:**
- **LF8-1**: 健康・QOL商材
- **LF8-3**: コンプレックス・美容商材
- **LF8-5**: 時短・利便性商材
- **LF8-6**: 見た目・社会的地位に関わる商材
