# LP作成プロジェクト

## 📁 フォルダ構成

```
LP作成/
│
├── 共通リソース/              # 全案件で共有するリソース
│   ├── ナレッジ/             # LP制作の知識体系
│   │   ├── 00_私の哲学.md
│   │   ├── 02_6大教育マニュアル.txt
│   │   ├── 04_構成パターン.txt
│   │   └── ... (その他ナレッジファイル)
│   ├── LP分析/               # 参考LP分析
│   │   ├── 北達人_ファンデ.txt
│   │   ├── 北達人_瞼.txt
│   │   └── ...
│   └── テンプレート/          # 新規案件用テンプレート
│       ├── ヒアリングシート.md
│       └── 使い方.md
│
├── 案件/                     # 案件別フォルダ
│   ├── 2025-01_初回案件/
│   │   ├── 入力/            # ヒアリング、商品情報など
│   │   ├── 下書き/          # 競合調査、構成案、ドラフトなど
│   │   ├── 完成/            # LP完成版
│   │   └── 案件概要.md
│   │
│   ├── 2025-XX_顧客名/      # 今後の案件はここに追加
│   └── ...
│
├── ココナラ・ランサーズ/      # 営業用
└── トレーニング/             # 学習用
```

## 🚀 新規案件の始め方

1. **案件フォルダを作成**
   ```bash
   mkdir -p "案件/2025-XX_顧客名/入力"
   mkdir -p "案件/2025-XX_顧客名/下書き"
   mkdir -p "案件/2025-XX_顧客名/完成"
   ```

2. **テンプレートをコピー**
   ```bash
   cp "共通リソース/テンプレート/ヒアリングシート.md" "案件/2025-XX_顧客名/入力/"
   ```

3. **ヒアリングシートを記入**
   - `案件/2025-XX_顧客名/入力/ヒアリングシート.md` を開いて顧客情報を入力

4. **LP制作開始**
   - `共通リソース/ナレッジ/` を参照しながら制作
   - 下書きは `下書き/` フォルダに保存
   - 完成版は `完成/` フォルダに保存

## 📚 ナレッジの使い方

詳細は `共通リソース/テンプレート/使い方.md` を参照してください。

## 💡 フォルダ命名規則

案件フォルダは以下の形式で命名:
```
YYYY-MM_顧客名
```

例:
- `2025-01_自動車学校様`
- `2025-02_ケーキ屋様`
- `2025-03_システム会社様`
