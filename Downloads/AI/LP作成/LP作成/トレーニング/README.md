# LP制作スキルアップ訓練ガイド

このフォルダは、あなたのLP制作スキルを高速で伸ばすための訓練場です。

---

## 📁 ファイル構成

```
training/
├── README.md                    ← このファイル(訓練ガイド)
├── 00_添削テンプレート.md        ← 自分が書いたLPの添削用
├── 00_LP分解テンプレート.md      ← 他人のLPを分解する用
├── 01_添削ログ_YYYYMMDD.md      ← 添削記録(日付ごと)
└── LP分解ログ_001.md            ← 分解記録(番号順)
```

## 🎯 訓練の流れ(3ヶ月プラン)

### 【1ヶ月目】インプット強化: LP分解100本

**目標:** 売れてるLPの「型」を体に染み込ませる

**やること:**
1. 同じジャンルのLP(美容・健康・ビジネス等)を集める
2. `00_LP分解テンプレート.md` をコピーして分解
3. 自分で分解 → AIに分解依頼 → 答え合わせ

**訓練量:**
- 最初の10本: じっくり分解(1本30分) → テンプレート使用
- 次の40本: 速読+構成メモ(1本10分) → 簡易メモでOK
- 残り50本: LF8とパターン判定だけ(1本5分) → 箇条書きでOK

**ポイント:**
- 良い表現は `knowledge/03_6大教育の参考フレーズ集.txt` に追加
- LF8の推測精度を上げることに集中

---

### 【2ヶ月目】思考力強化: 逆算訓練12本

**目標:** 「もし自分なら」を考える逆算力を鍛える

**やること:**
1. 競合LPを1本選ぶ
2. 分解する
3. 「ターゲットを変えたら?」「LF8を変えたら?」を考える
4. 実際に別バージョンのFVや問題の教育を書いてみる

**訓練量:**
- 週3本ペース(月12本)
- 各LPで3パターン書く(合計36パターン)

**ポイント:**
- パターンA/Bの使い分けを体で覚える
- 同じ商品でも切り口で全然変わることを実感

---

### 【3ヶ月目】実践: 架空商品のLP執筆6本

**目標:** ゼロから書く力をつける

**やること:**
1. 架空の商品を設定(例: 副業講座、ダイエットサプリ等)
2. ターゲット・LF8を決める
3. 2時間で1本書く
4. `00_添削テンプレート.md` で添削依頼

**訓練量:**
- 月2本ペース(合計6本)

**ポイント:**
- 完璧を目指さない。「書ききる」ことが目標
- 添削で改善 → また書く のループを回す

## 🔥 添削の受け方(今すぐできる)

### ステップ1: LPを書く

`00_添削テンプレート.md` をコピーして、あなたのLPを貼り付け

### ステップ2: AIに添削依頼

```
プロンプト例:
「training/01_添削ログ_20250126.md を読んで、
【あなたのLP】セクションのLPを添削してください。

knowledge/02_6大教育マニュアル.txt
knowledge/05_反論処理フレームワーク.txt
knowledge/07_LF8逆算思考法.txt
を参照して、【AI添削結果】セクションを埋めてください。」
```

### ステップ3: 修正する

AIの指摘を元に修正 → 【修正後のLP】セクションに貼り付け

### ステップ4: 振り返り

【振り返り】セクションに学びをメモ

## 💡 訓練のコツ

### コツ1: フィードバックループを回す
```
書く → 添削 → 修正 → また書く
```

### コツ2: アウトプット前提で読む
- 「この表現、使える!」→ メモ
- 「このLF8の使い方、応用できる」→ メモ

### コツ3: 完璧を目指さない
- 最初は下手でいい
- 書ききることが大事
- 添削で改善すればOK
