> **AIは「型」を作るのが得意。あなたは「中身」を決めるのが得意。**

| 項目 | 担当 | 理由 |
|------|------|------|
| **LF8特定** | 👤 あなた | AIには肌感覚がない |

# 📝 各ステップの詳細

## ステップ-1: クライアントへのヒアリング 🎤

### 💡 重要度: ⭐⭐⭐⭐⭐

> **LP制作の8割は、ヒアリングで決まる!**

---

### ✅ やること

1. `input/00_ヒアリングシート.md` を開く
2. クライアントに質問して、シートを埋める
3. ヒアリング内容を `input/01_商品情報入力.txt` に転記

---

### 📋 ヒアリングで聞くべきこと

| カテゴリ | 質問内容 |
|---------|---------|
| **🎁 商品について** | 特徴・ベネフィット・独自性・価格 |
| **👥 既存顧客について** | 誰が買ってる? なぜ買った? どう変わった? |
| **🎯 ターゲットについて** | 誰に売りたい? どんな悩みがある? |
| **⚔️ 競合について** | 競合は? 違いは? 負けてる点は? |
| **🎯 クライアントの希望** | 目標CV数・売上・絶対に入れたい要素 |

---

### 🔑 ヒアリングのコツ

```
💬 「なぜ?」を3回聞く
   → 本質的な理由を掘り下げる

📖 具体的なエピソードを聞く
   → 数字や実例を集める

🔢 数字を聞く
   → 月何個売れてる? リピート率は?

⚠️ ネガティブなことも聞く
   → 競合に負けてる点、過去の失敗
```

---

### 📄 詳細マニュアル

👉 [`input/00_ヒアリングシート.md`](input/00_ヒアリングシート.md) を参照

## ステップ0: コンセプト設計 🎯

### 💡 重要度: ⭐⭐⭐⭐⭐

> **誰に・何を・どう売るか** を決める

---

### 📌 Q1: 誰に売るのか? (ターゲット設定)

#### ステップ1️⃣: ターゲット候補をリストアップ (3-5人)

まず、「誰が買いそうか?」を **一言でサクッと** リストアップ

```
- 候補A: _________________
- 候補B: _________________
- 候補C: _________________
- 候補D: _________________
- 候補E: _________________
```

**💡 例: ロボット掃除機の場合**
```
- 候補A: 共働き夫婦
- 候補B: ペット飼ってる人
- 候補C: 高齢者
- 候補D: 一人暮らしの忙しい人
```

---

#### ステップ2️⃣: 1人に絞る

以下の基準で選ぶ:

- ✅ 既存顧客の中で、一番多い層か?
- ✅ クライアントが増やしたい層か?
- ✅ 悩みが深刻か? (緊急性が高いか?)
- ✅ 市場規模は十分か?

**選んだターゲット:** 候補_____

---

#### ステップ3️⃣: 選んだターゲットを具体化する

選んだターゲットを、詳しく書き出す:

```
年齢・性別: _________________
悩み・課題: _________________
生活スタイル: _________________
価値観: _________________
```

---

#### ✅ 良い例 vs ❌ 悪い例

| | 内容 |
|---|---|
| ✅ **良い例** | 「40代後半〜50代女性、仕事・家事で忙しく、メイクに時間をかけたくないが、すっぴんでは外に出られない。手軽に綺麗になりたいと思っている」 |
| ❌ **悪い例** | 「女性全般」 |

---

### 📌 Q2: 何を売るのか? (商品・価値の明確化)

以下を明確にする:

```
✅ 商品の機能: _________________
✅ 商品がもたらす結果: _________________
✅ 独自性 (他と何が違うのか): _________________
```

---

#### ✅ 良い例 vs ❌ 悪い例

| | 内容 |
|---|---|
| ✅ **良い例** | 「1本でベースメイク完了のオールインワンファンデーション。独自技術でバームからパウダーに変化。時短×プロ級の仕上がりを両立」 |
| ❌ **悪い例** | 「オールインワンファンデーション」(だけ) |

---

### 📌 Q3: どう売るか? (切り口・ポジショニング)

> **= どんな未来につれていくのか?**

ターゲットのLF8に沿って、理想の未来を提示する

---

#### ステップ1️⃣: ターゲットのLF8を特定する

選んだターゲットは、どのLF8で動いてる?

```
メインLF8: LF8-○ (        )
サブLF8: LF8-○ (        )
```

---

**📋 LF8リスト (参照用)**

| LF8 | 内容 | 理想の未来 |
|-----|------|----------|
| **LF8-1** | 生き延びたい | 健康になる、病気を防ぐ、命を守る |
| **LF8-2** | 食べたい | 美味しいものを食べられる |
| **LF8-3** | 恐怖回避 | もう○○しない、不安から解放される |
| **LF8-4** | 愛する人を守りたい | 家族を守れる、子供の未来が明るくなる |
| **LF8-5** | 快適でいたい | めんどくさいから解放、ラクになる、時間が生まれる |
| **LF8-6** | 優越感 | 他人より上、周りと差がつく、勝てる |
| **LF8-7** | 愛されたい | 愛される、大切にされる、認められる |
| **LF8-8** | 社会的承認 | 尊敬される、キャリアアップ、社会的地位 |

---

#### ステップ2️⃣: LF8に沿った「理想の未来」を複数出す

ターゲットのLF8に対して、「どんな未来を見せたら、欲しくなるか?」を **3-5個** リストアップ

```
- 未来A: _________________
- 未来B: _________________
- 未来C: _________________
- 未来D: _________________
- 未来E: _________________
```

**💡 例: ロボット掃除機 (ターゲット: 共働き夫婦、LF8-5 + LF8-4)**
```
- 未来A: 掃除の時間を、家族の時間に
- 未来B: 週末を掃除に使わなくていい
- 未来C: 仕事から帰ったら、もう部屋が綺麗
- 未来D: 毎日綺麗な部屋で過ごせる
```

---

#### ステップ3️⃣: 切り口を1つに決める

未来の中から、一番刺さるものを選んで、**切り口** にする

**選んだ切り口:** _________________

---

**💡 切り口の例**

| 切り口 | ターゲット | LF8 | 見せる未来 |
|--------|----------|-----|----------|
| 掃除の時間を、家族の時間に | 共働き夫婦 | LF8-5 + LF8-4 | 時間が生まれる、家族と過ごせる |
| 時短×プロ級 | 忙しい40代女性 | LF8-5 + LF8-6 | めんどくさいから解放、でも綺麗 |
| 将来の選択肢を広げる | 英語を学びたい人 | LF8-8 + LF8-6 | キャリアアップ、転職の幅が広がる |

---

⚠️ **注意:** 同じ商品でも、ターゲットとLF8が変われば、切り口(見せる未来)も変わる

# 📊 AIと人間の役割分担まとめ

| 項目 | 👤 あなたの役割 | 🤖 AIの役割 |
|------|------------|---------|
| **コンセプト案の作成** | ✅ 最終決定 | 💡 複数パターンを提案 |
| **ターゲット設定** | ✅ 具体的なペルソナを決める | 💡 候補を提案 |
| **LF8特定** | ✅ ターゲットの本当の欲求を洞察 | 💡 分析・推測 |
| **問題の教育の中身** | ✅ どんな不便・原因を列挙するか決める | 表現の改善 |
| **手段の教育の中身** | ✅ 独自性をどう伝えるか決める | 表現の改善 |
| **+αの価値の選定** | ✅ ターゲットが喜ぶポイントを選ぶ | 💡 提案のみ |
| **LP構成の設計** | - | ✅ LF8から自動選択 |
| **表現のバリエーション** | - | ✅ 同じ内容を別の言い方で |
| **反論処理の網羅** | - | ✅ 14項目から抜け漏れチェック |

# ⚠️ よくある間違い

## ❌ 間違った使い方

```
「以下の商品のLPを作成してください」と丸投げ
```

### 問題点

- ❌ AIが勝手にLF8を判断 → **精度が低い**
- ❌ AIが勝手に問題の教育の内容を決める → **ターゲットに刺さらない**
- ❌ AIが勝手に独自性を解釈 → **商品の魅力が伝わらない**

---

## ✅ 正しい使い方

1. ✅ あなたがコンセプト・LF8・中身を決める
2. ✅ AIに構造化・表現改善を依頼する
3. ✅ AIの出力を確認して、中身を修正する

---

# 💡 重要ポイント

```
✅ 各Stepの後、draft/ のファイルを確認して修正を入れてからNext Stepへ

✅ knowledge/ のファイルは育成用。良い表現が見つかったら追記していく

✅ コンセプトが決まらないと、LF8も決まらない

✅ LF8が決まらないと、LP構成も決まらない

✅ LF8を特定すれば、LP構成の80%が自動的に決まる
```

---

1. ✅ `output/final/LP完成版.md` を確認
