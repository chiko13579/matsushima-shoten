# 6大教育 定義マニュアル

LPにおいては、以下の6つの教育をすべて満たす必要がある。
ターゲットのLF8を特定した後、そのLF8に刺さる形で6大教育を設計する。

---

## 1. 信用の教育

**本質：**
「なぜあなたから買うのか」を納得させる。
読者は「この人、本当に信用できるの？」と疑っている。

**信用を作る4要素：**

| 要素 | 内容 | 例 |
|------|------|-----|
| 実績 | 数字で証明できる結果 | 「3,000人が受講」「売上2億円」 |
| 苦労話 | 過去の失敗・どん底経験 | 「借金500万から這い上がった」 |
| 理念 | なぜこの仕事をしているか | 「過去の自分を救いたい」 |
| 専門性 | 信頼に値する知識・情報を持っている | 「業界15年」「資格保有」「独自データ」 |

**専門性の2つの側面：**

| 側面 | 内容 | 例 |
|------|------|-----|
| 権威 | 「誰が言ってるか」で信頼させる | 資格、業界歴、肩書き |
| 納得感 | 「何を言ってるか」で信頼させる | コラム的な解説、論理的な説明、「なるほど」と思わせる内容 |

**「納得感」で信頼を作るパターン：**
- 「実は○○の原因は△△なんです」（原因の解説）
- 「よくある間違いが○○です」（誤解を正す）
- 「なぜ○○がダメかというと…」（理由の説明）
- 「業界では常識ですが、一般には知られていない○○」（裏話）
- 「○○と△△の違いは…」（比較・整理）

**4要素の組み合わせ方：**
「昔は自分もダメだった（苦労）→ だから同じ悩みの人を救いたい（理念）→ 実際に3,000人を救った（実績）→ だからこそ言える○○（専門性）」

**ポイント：**
- 実績だけ → 「すごい人」で終わる。距離ができる
- 苦労話だけ → 「大変だったね」で終わる。信頼にならない
- 理念だけ → 「きれいごと」に聞こえる。嘘くさい
- 専門性だけ → 「先生」になる。距離ができる
- **4つを組み合わせる**のが正解

## 2. 目的の教育

**本質：**
お客さんが無意識に思い描いている理想のゴールを具体的にイメージさせ、「手に入れたい！」と憧れさせる。

**なぜ必要か：**
- 人は素敵な未来を見せてくれる人についていく
- 理想を持っていなくても、具体的にイメージさせれば「欲しい」が生まれる
- 何度もイメージさせると、お客さん自身が勝手に妄想し始める
- 「理想を手に入れるには何をすればいい？」とセールスに興味を持ち始める

**目的の教育でやること：**

| やること | 内容 | 例 |
|----------|------|-----|
| 理想の未来をイメージさせる | 「あなたはこうなれます」 | 「月収50万円の自由な生活」 |
| リアリティを演出 | 五感で想像させる | 「朝、目覚ましなしで起きる生活」 |
| 繰り返しイメージさせる | LP全体で何度も見せる | 冒頭・中盤・終盤で未来を描写 |

**具体的にイメージさせるテクニック：**
- 時間を入れる（「朝起きたら」「1年後には」）
- 場所を入れる（「カフェで」「自宅のリビングで」）
- 感情を入れる（「ニヤニヤが止まらない」「肩の荷が降りた」）
- 他者の反応を入れる（「家族に"変わったね"と言われる」）
- 数字を入れる（「月収50万円」「週3日だけ働く」）

**「自分ごと」にさせる：**
抽象的な理想ではなく、**読者の日常に存在するシーン**で描く。

| NG（抽象的） | OK（日常のシーン） |
|-------------|-------------------|
| 「自由な生活」 | 「平日の昼間にカフェでゆっくりできる」 |
| 「収入が増える」 | 「値段を見ずに買い物できる」 |
| 「健康になる」 | 「階段を息切れせずに上れる」 |
| 「自信がつく」 | 「鏡を見るのが楽しくなる」 |

---

**用途・シーンの広がりで「便利さ」を見せる：**

メイン機能の「範囲の広さ」を箇条書きで見せることで、「こんなに使える！」という驚きと便利さを伝える。

**※これは理想の未来セクション(LP前半)で使う。+αの価値セクション(手段の教育の後)とは別物。**

**2つの切り口:**

| 切り口 | 内容 | 例 |
|--------|------|-----|
| 機能の範囲の広さ | メイン機能が「何役もこなす」 | 「1本で5役（日焼け止め、下地、ファンデ、コンシーラー、パウダー）」 |
| シーン・用途の広がり | 「どんな場面でも使える」 | 「リビング、寝室、階段、車の中、布団」 |

**実例（ズボラ向けファンデ）:**
```
なんとこれ1本でベースメイクは丸ごと完了。
- 日焼け止め
- 化粧下地
- ファンデーション
- コンシーラー
- パウダー
```
→ **機能の範囲の広さ**を箇条書きで見せている

**実例（掃除機の場合）:**
```
これ1台で家中の掃除が完了
- リビング
- 寝室
- 階段
- 車の中
- 布団クリーナーとして
```
→ **シーン・用途の広がり**を箇条書きで見せている

**書き方のコツ:**
- 箇条書きで視覚的に「多い」と感じさせる
- 「これ1本で」「これ1台で」のワンフレーズで全体像を先に見せる
- 「○役」「○ヶ所」など数字を入れる

**+αの価値との違い:**

| 項目 | 用途・シーンの広がり（理想の未来） | +αの価値（手段の教育の後） |
|------|--------------------------------|------------------------|
| タイミング | LP前半（理想の未来セクション） | LP後半（手段の教育の後） |
| 内容 | メイン機能の範囲の広さ | メイン機能とは別の追加価値 |
| 例（ファンデ） | 1本で5役（機能の範囲） | 美容成分配合（追加の成分） |
| 例（掃除機） | リビング・車・布団に使える（用途の広がり） | 静音・省エネ・デザイン（追加機能） |

---

**「自分でもできる」と思わせる：**

1. **同じ境遇の人の事例を見せる**
   - 「私と同じ状況だった人ができてる」→ 自分もできそう
   - 年齢、職業、家族構成、悩みが近いほど刺さる

2. **ハードルの低さを見せる**
   - 「1日5分だけ」「スマホがあればOK」「特別なスキルは不要」

3. **最初の一歩を具体的に見せる**
   - 「まずは○○するだけ」「届いたらこれをやるだけ」
   - 何をすればいいか分からない不安を消す

4. **失敗しても大丈夫を見せる**
   - 「最初はみんな失敗します」「サポートがあるから安心」「返金保証があるからリスクゼロ」

5. **才能・センス不要を見せる**
   - 「再現性がある方法」「やり方さえ知れば誰でもできる」「私も最初はできなかった」

※特にLF8-8（認められたい）の人には「あなたと同じようにダメだった人ができた事例」が最強

**NG：**
- 抽象的すぎる（「幸せになれます」→ 何も想像できない）
- 自分の実績自慢になる（お客さんの未来を見せるのが目的）

## 3. 問題の教育

**本質：**
理想を邪魔している壁・障害物を自覚させる。
**※解決はしない。自覚させるだけ。**

**なぜ解決しないのか：**
- 解決したら「これで大丈夫」と満足して離脱する
- 問題を自覚させて「克服したい」という強い欲求を起こさせるのが目的
- 解決策は「手段の教育」で商品と一緒に提示する

---

### 🔥 問題の教育を読ませる最強の心理設計

**従来のダメなパターン（ネガティブ先行）:**
```
❌ 問題の教育（原因・恐怖）
→ 「加齢で○○が失われて...」
→ 「自力では無理で...」
→ ネガティブな感情が先
→ 「やっぱり無理なんだ...」と絶望
→ 離脱
```

**北達人の勝ちパターン（ポジティブ→好奇心→問題の教育）:**
```
✅ ポジティブ体験談
→ 「すごい効果！」
→ ポジティブな感情で高揚

↓ 疑問を作る

❓ 「でもなんでこんなに効くの？知りたい！」
→ 知的好奇心で前のめり（LF8-2: 知りたい）

↓ 問題の教育

📚 「実は○○が原因で...」（科学的説明）
→ 「なるほど！だから効くんだ！」と納得
→ むしろ信頼度アップ
```

**なぜこの順番が強力なのか:**
- ポジティブな結果を先に見せる → 希望を持たせる
- 「なぜ？」の疑問を作る → 知的好奇心を刺激
- 問題の教育を**自発的に**読みたい状態にする
- ネガティブ情報でも絶望させない（すでに解決策を見せているから）

**読者の心理:**
- ❌「怖い...無理なんだ...」（従来型）
- ✅「なるほど！だから効くんだ！科学的根拠がある！買おう！」（北達人型）

---

### 問題の教育への3つの切り口パターン

体験談で好奇心を喚起した後、問題の教育に入る「切り口」は3パターン:

#### パターン1: 「なぜ？」疑問型（最も多い）
**使い方:** 体験談の後に疑問を投げかけ、インタビュー形式で解明

**実例（瞼LP）:**
```
体験談: 「でも、さっと塗っただけなのに、なんでこんなに目の大きさが変わるの？」
↓
問題の教育: 「Q: そもそも目が小さく見える原因は何でしょうか？」
```

**実例（眉間シワLP）:**
```
体験談: 「でもこれまでもいろいろ対策は試したのに、どうしてここまでの違いが?」
疑問: 「このパッチ、何者?」
↓
問題の教育: 「インタビューで発覚。眉間に出る年齢感は自力で解決できない?」
```

#### パターン2: 「調べてみた」探求型
**使い方:** 「もっと知りたい」という探求心を見せ、インタビューに繋げる

**実例（瞼LP）:**
```
「ネットで調べてもわかりにくい。もうプロに聞くしかない。」
↓
「思い切って開発元に連絡してみたところ、なんとまぶたをリフトアップさせられる秘密を特別にインタビューさせていただきました。」
```

**実例（眉間シワLP）:**
```
「ただネットで調べても分かりにくい。」
↓
「そこで開発元に連絡してみたところ、なんとすごさの秘密を特別インタビューさせてくれました。」
```

#### パターン3: 「ご存知ですか？」知識提示型
**使い方:** 「実は○○なんです」と知識を提示し、教育に入る

**実例（膝ヒアルロン酸LP）:**
```
体験談: 「え、すごい。こんなに簡単に対策できるなんて」
↓
問題の教育: 「調べてみると、実は今注目の対策のよう」
→ 「ヒアルロン酸シートの人気の秘密は?」
→ 「ヒアルロン酸の特徴は、なんといってもその保水力」（知識提示）
```

**どの切り口を使うか:**
- パターン1「なぜ？」疑問型: 原因が複雑で科学的説明が必要な場合（美容・健康商材）
- パターン2「調べてみた」探求型: 開発ストーリーや技術的背景を見せたい場合
- パターン3「ご存知ですか？」知識提示型: 知識不足が問題の場合（新素材、新技術）

---

### 商材によって2つのパターンがある

問題の教育は、**悩みの深刻度**によって2つのアプローチがある:

---

### パターンA: 深刻な悩み(コンプレックス系、健康問題)

**対象商材:**
- 美容(シワ、たるみ、シミなど)
- 健康(膝の痛み、不眠など)
- コンプレックス(薄毛、体臭など)

**目的:**
悩みの「原因」を科学的に説明して、自力では解決できないことを理解させる
→ 商品の必要性を高める

**やること:**

| やること | 内容 | 例 |
|----------|------|-----|
| 原因を科学的に説明 | 「なぜその悩みが起きるのか」 | 「加齢によるヒアルロン酸の減少が原因です」 |
| 自力では無理を証明 | 「今の科学では不可能」 | 「肌のヒアルロン酸減少を自力で止めることは、今の科学では不可能だったんです」 |
| 放置の恐怖 | 「このままだとどうなるか」 | 「放っておくとさらに深くなります」 |

**テクニック:**
- 「実は○○が本当の原因なんです」（気づきを与える）
- 「多くの人が○○と思ってますが、違います」（誤解を正す）
- 「○○していませんか？それが原因です」（自分ごと化）
- 「自力で○○することは不可能です」（商品の必要性を高める）

**実例(北達人LP):**
```
「Q: そもそも眉間に年齢感が出る原因は何なんでしょうか?
A: ズバリ、加齢によるヒアルロン酸の減少が原因です

減少は誰も避けられない。でも肌のヒアルロン酸減少を自力で止めることは、今の科学では不可能だったんです。

Q: じゃあ解決はできないんですね
A: そこで全く別のアプローチ方法を開発したんです」
```

---

### パターンB: 共感型(うっすら思っていることを自覚させる)

**正しい理解:**
❌ 軽い不便を列挙するだけ
✅ **ターゲットが普段うっすら思っている身近な不安・課題・不快を自覚させる**

**対象商材:**
- 時短ツール(オールインワン化粧品など)
- 利便性向上(家事を楽にする商品など)
- ストレス軽減(めんどくさいを解決する商品)
- **健康食品(深刻ではないが、うっすら不安がある)**

**目的:**
今抱えている「不便・煩わしさ・うっすらとした不安」を言語化して、「これ解決したら楽だよね」と思わせる
→ 解決への欲求を喚起

---

### 🔥 パターンBの核心: 本音と葛藤

**パターンBの本質は「知らないことを教える」のではなく、「知っていることを自覚させる」**

#### **3つのレイヤー:**

**レイヤー1: あるある(共感)**
- 「こんなことしてませんか?」
- → **「そうそう!」と言わせる**

**レイヤー2: 本音(自覚)**
- 「本当はこう思ってるよね?」
- → **「やばいかも...」と思わせる**

**レイヤー3: 葛藤(無力感)**
- 「やらなきゃと思ってるけど、できないよね?」
- → **「そう!それが悩みなんだよ!」と共感させる**
- → **「誰か助けて...」と救いを求めさせる**

---

### やること:

| やること | 内容 | 例 |
|----------|------|-----|
| 不便を列挙 | 「今どんな手間がかかってるか」「今どんな不便があるか」 | 「化粧下地、コンシーラー、パウダー…」「スマホを1日10回探す」 |
| 時間の浪費を見せる | 「これに何分かかってるか」 | 「毎朝20分もベースメイクに使ってる」 |
| 「あるある」で不便や不快の共感 | ターゲットの言葉であるあるを代弁(分かるって言いたくなるもの) | 「めんどくさいですよね」 |
| ストレスの言語化 | 「いつも感じてる煩わしさ」 | 「あれこれ塗り重ねるのがストレス」 |
| **本音と葛藤** | 「分かってるけどできない」ジレンマを見せる | 「運動しなきゃいけないのは分かってる、でも...」 |

---

### 🔥 本音と葛藤の表現パターン

#### **パターン1: 「分かってるけど...」**
```
分かってはいるけど、○○できない
やらなきゃいけないのは分かってる、でも○○
```

**例:**
- 「運動しなきゃいけないのは分かってる、でも時間がない」
- 「野菜を食べなきゃいけないのは分かってる、でも続かない」
- 「分かってはいるけど、生活習慣、運動、食事、何から手をつけていいのか」(Wの青汁)

#### **パターン2: 「○○したいけど、○○しないといけない」**
```
○○したい、でも○○しないといけない
```

**例:**
- 「手軽にしたいけど、効果も欲しい」
- 「時短したいけど、手抜き感は出したくない」
- 「楽したいけど、すっぴんでは外に出れない」
- 「化粧ってほんとめんどくさい。でもすっぴんでは外に出れないし」(北達人LP)

#### **パターン3: 「何から手をつけていいのか...」**
```
何から手をつけていいのか分からない
どうすればいいのか分からない
```

**例:**
- 「生活習慣、運動、食事、何から手をつけていいのか」
- 「情報が多すぎて、何が正解か分からない」

---

### なぜ葛藤が重要なのか?

#### **葛藤がないと:**
```
「健康診断の結果が気になる」
「不規則な食生活してる」
↓
「だから青汁飲もう」
```
→ **浅い。刺さらない。**

#### **葛藤があると:**
```
「健康診断の結果が気になる」
「不規則な食生活してる」
↓
「分かってる。運動も野菜も必要なのは分かってる」
「でも...何から手をつけていいのか分からない」
↓
「誰か助けて...」(無力感)
↓
「そんなあなたに、この商品」
```
→ **深い。刺さる。救いを求めている状態。**

---

### テクニック:
- 箇条書きで手間を列挙(視覚的に「多い」と感じさせる)
- ターゲットの口語で書く(「めんどくさい」「でもすっぴんでは外に出れないし」)
- 日常のシーンを描写(「毎朝」「バタバタ」など時間帯を入れる)
- **「でも○○しないといけない」のジレンマを見せる**
- **「分かってるけど...」の葛藤で無力感を演出**

**実例1(北達人LP - ズボラ向けファンデ):**
```
「化粧ってほんとめんどくさい。でもすっぴんでは外に出れないし。

あれこれ塗り重ねるのがめんどくさくてストレス。
- 化粧下地を塗って
- コンシーラーも
- パウダーもしなきゃ
- プライマーいるかなぁ
- ファンデ塗って

普段20分かけてベースメイクしていた私でもあっという間にベースメイクが終わるなんて思いもよりませんでした。」
```

**実例2(Wの青汁 - 健康食品):**
```
健康診断を受けた後は結果が気になりますか？95%がはい。
特に体重、ウエスト、中性脂肪、血圧、気になる人が多い傾向に。

日々の生活で思い当たることはありませんか？
- 不規則な食生活
- お酒を飲むようなものが好き
- あまり動かない生活
- 日々のストレス

これらの積み重ねの結果、どうしよう？
最近ちょっとお腹が出てきたかな？
生活習慣が乱れているかも？

国から推奨されている1日1万歩レベルの運動
国から推奨されている1日350gの野菜を食べる

分かってはいるけど、生活習慣、運動、食事、何から手をつけていいのか。
```

→ **「95%がはい」で共感→「あるある」で自覚→「分かってはいるけど...」で葛藤**

### パターンA vs パターンB の違い(再定義)

| 項目 | パターンA(教育型) | パターンB(共感型) |
|------|---------------------|-------------------|
| アプローチ | 知らない→知る | うっすら思っている→明確に自覚 |
| トーン | 深刻、科学的 | 共感、カジュアル |
| 説明の深さ | 原因を科学的に説明 | あるある+本音+葛藤 |
| ゴール | 「自力では無理」と諦めさせる | 「分かってるけどできない」と救いを求めさせる |
| 欲求の種類 | 「解決しないとヤバい」(恐怖) | 「解決したら楽だよね」(快適) |
| 商品の位置づけ | 必需品 | あったら便利、でも欲しい |
| 読者の反応 | 「へぇ!そうなんだ!」(驚き) | 「そうそう!分かる!」(共感) |

---

### どちらを使うか判断する方法

**パターンAを使う条件:**
✅ ターゲットが深刻に悩んでいる(LF8-3: 恐怖から逃れたい)
✅ 健康・コンプレックスに関わる
✅ 放置すると悪化する
✅ 科学的根拠で説明できる

**パターンBを使う条件:**
✅ ターゲットは困ってるけど深刻ではない(LF8-5: 快適でいたい)
✅ 時短・利便性向上が目的
✅ 「あったら便利」レベル
✅ 共感・あるあるで刺さる

---

### 共通のNG

**NG:**
- 解決策まで言う(「だから○○すればいい」→ 満足して離脱)
- 問題を軽く見せる(「大したことない」→ 欲求が生まれない)
- ターゲットを責める(「あなたのせい」→ 反発される)
- パターンAとBを混同する(深刻な悩みに「あるある」はNG、軽い不便に「恐怖」はNG)

## 4. 手段の教育

**本質：**
問題を克服するための手段として、この商品が最良だと納得させる。
**※競合批判はNG。**

**なぜ競合批判NGか：**
- 陰口を言う人から買いたくない
- 批判で買う客はクレーマーになりやすい、リピーターにならない
- 「あっちよりこっちが自分に合ってそう」と思わせるのが正解

---

### 手段の教育の6つの特徴

#### 1. 「唯一性」の演出が核心
**やること:**
- この商品でしか解決できない理由を示す
- 独自技術、独自成分、独自メソッドを強調

**表現例:**
- 「世界初」「業界初」「日本初」
- 「特許取得」「独自開発」
- 「○○年の研究の末、ついに完成」
- 「他では真似できない○○技術」

#### 2. 開発ストーリーで感情移入させる
**やること:**
- 開発者の情熱・苦労を描く
- なぜこの商品が生まれたのかの物語

**表現例:**
- 「気が遠くなるような試作を繰り返し」
- 「○○年の研究の末、ついに完成」
- 「諦めかけた時、転機が訪れた」
- 「開発者自身が○○に悩んでいた」

#### 3. 科学的根拠 + わかりやすい比喩
**やること:**
- 専門用語を使いつつ、中学生でもわかる説明
- 「まるで○○のように」の比喩表現
- 図解、イラストでの補足

**表現例:**
- 「専門用語で言うと○○。簡単に言えば、△△のようなものです」
- 「まるでスポンジが水を吸うように、肌に浸透する」
- 「○○大学の研究で証明されています」

#### 4. 問題の教育からの自然な流れ
**やること:**
- 「じゃあどうすればいいの？」→「そこで開発したのが」
- 問題提起 → 絶望 → 希望（手段の教育で提示）

**表現例:**
- 「Q: じゃあ解決はできないんですね」
- 「A: そこで全く別のアプローチ方法を開発したんです」

#### 5. 競合との違いを"暗に"示す
**やること:**
- 直接批判はNG
- 「従来の○○では△△だったが、これは〜」の表現
- 「一般的な○○と違い」の表現

**テクニック:**
- 「○○も良い方法ですが、△△な人には向きません」（否定せず棲み分け）
- 「○○で成果が出なかった人は、□□が原因かもしれません」（相性の問題にする）
- 「あなたのような○○な人には、この方法が最適です」（ターゲットを絞る）
- 「○○と△△、どちらが正解かはあなた次第です。ただ、□□を重視するなら…」（選択肢を提示）

#### 6. 「なぜ今までなかったのか？」の説明
**やること:**
- 技術的ハードルを示す
- コストの問題
- 開発の難しさ

**表現例:**
- 「実は○○を実現するには、△△という課題がありました」
- 「コストがかかりすぎて、今まで商品化できなかった」
- 「異なる2つの技術を融合させるのは至難の業でした」

---

### 手段の教育でやること（まとめ）

| やること | 内容 | 例 |
|----------|------|-----|
| 手法の正しさを証明 | 学術的根拠・データ | 「○○大学の研究で証明」 |
| 独自性を示す | この商品だけの特徴 | 「世界初の○○技術」 |
| 開発ストーリー | なぜ生まれたか | 「○○年の研究の末」 |
| わかりやすく説明 | 専門用語 + 比喩 | 「まるで○○のように」 |
| 他の手法との違い | 批判せず比較 | 「従来の○○では△△だったが」 |
| この商品が最適な理由 | なぜあなたに合うか | 「初心者でも○○だから安心」 |

---

### NG表現

**NG:**
- 「○○はダメです」「○○は詐欺」（直接批判）
- 「うちが業界No.1」（根拠なき自画自賛）
- 専門用語だけで説明（わかりにくい）
- 開発ストーリーが嘘っぽい（具体性がない）

## 4-2. +αの価値（手段の教育の後に追加）

**本質:**
「解決策だけでも十分なのに、さらにこんな良いことも！」という追加価値を示す。
メイン機能で納得させた後、副次的メリットで購買意欲を高める。

**目的:**
- 「お得感」を演出する
- 商品の魅力を多角的に見せる
- 「これだけでいいの？」という驚きを与える

---

### +αの価値の4つの特徴

#### 1. 「そしてもう一つ」の追加感が重要
**やること:**
- メイン機能を説明した後に、追加で良いことを伝える
- 「さらに」「そしてもう一つ」「実は」などの接続詞で繋ぐ

**表現例:**
- 「そしてもう一つ、私はこれが嬉しすぎると思ったのが」
- 「さらに驚いたことに」
- 「実はこれだけじゃないんです」
- 「加えて、○○も配合されているんです」

#### 2. 箇条書きで視覚的に「豊富感」を出す
**やること:**
- 成分、機能、特典などを箇条書きで列挙
- 視覚的に「こんなにたくさん！」を演出

**表現例:**
```
なんと潤い成分が贅沢に配合されていること。

- セラミド
- CICA
- パールエキス
- シルクエキス
- ヒアルロン酸
```

#### 3. メイン価値との相乗効果を示す
**やること:**
- メイン機能 + +α = さらに良い結果
- 単なる追加ではなく、相乗効果があることを強調

**表現例:**
- 「化粧している時間がまさかのスキンケアタイムに大変身」
- 「○○しながら、同時に△△もできる」
- 「使うたびに肌が整っていく」

#### 4. ターゲットにとって「嬉しすぎる」ポイントを選ぶ
**やること:**
- ターゲットが「これは嬉しい！」と思う副次的メリットを選ぶ
- メイン価値とは違う角度から攻める

**表現例:**
- 時短商材 → 美容成分配合（時短 + 美容）
- 美容商材 → 使いやすさ（効果 + 利便性）
- 健康商材 → 美味しさ（健康 + 快適さ）

---

### +αの価値の典型的な内容

| 商材タイプ | メイン価値 | +αの価値 |
|----------|----------|---------|
| 時短ファンデ | 1本で完了 | 美容成分配合、スキンケア効果 |
| 美容液 | シワ改善 | 時短、使いやすさ、香り |
| サプリメント | 健康効果 | 美味しい、続けやすい、コスパ |
| 家事代行ツール | 時短 | 省エネ、静音、デザイン |


### +αの価値を書く時のコツ

1. **メイン価値との対比を意識する**
   - メイン: 機能的価値
   - +α: 情緒的価値、副次的価値

2. **「意外性」を演出する**
   - 「まさか○○まで？」という驚き
   - 「ここまでやるの？」という感動

3. **「贅沢感」を出す**
   - 「贅沢に配合」「惜しみなく」
   - 箇条書きで視覚的に豊富さを見せる

4. **ターゲットの言葉で「嬉しさ」を表現**
   - 「これが嬉しすぎる」
   - 「ここまでしてくれるの？」
   - 「至れり尽くせり」

---

### NG表現

**NG:**
- +αが多すぎて焦点がぼやける（メイン価値が何か分からなくなる）
- メイン価値より+αの方が強調される（本末転倒）
- 根拠のない成分羅列（「なぜ良いのか」が不明）
- +αセクションが長すぎる（簡潔に）

## 5. 投資の教育

**本質：**
「なぜこの価格が投資なのか」を納得させる。
読者は「高いな…」と思っている。

**価格を納得させる4つの技術：**

| 技術 | 内容 | 例 |
|------|------|-----|
| リターン提示 | 得られる価値を金額換算 | 「月5万円稼げれば、1年で60万円」 |
| コスト分解 | 日割り・月割りで見せる | 「1日たった330円」 |
| 比較 | 他の出費と比べる | 「飲み会1回分で一生モノのスキル」 |
| 機会損失 | 買わないコストを見せる | 「今のまま1年過ごすと失う時間は○時間」 |

**「高い」と言われる前に潰す：**
- 価格を出す前に価値を積み上げる
- 「普通なら○○円かかるところ…」で基準を作る
- 「これは費用ではなく投資です」と定義を変える

**NG：**
- 価値を積み上げる前に価格を出す
- 「安いです！」と言い切る（安売り感が出る）

## 6. 行動の教育

**本質：**
「なぜ今申し込むのか」を納得させる。
読者は「もうちょっと考えてから…」と思っている。

**行動させる3要素：**

| 要素 | 内容 | 例 |
|------|------|-----|
| 限定性 | 今じゃないと手に入らない | 「先着○名」「○月○日まで」「今回限り」 |
| 特典 | 今だけもらえるもの | 「今なら○○もプレゼント」 |
| 保証 | リスクをゼロにする | 「全額返金保証」「効果なければ無料」 |

**「考えます」を潰す：**
- 限定性で「後では手に入らない」を見せる
- 保証で「失敗しても大丈夫」を見せる
- 特典で「今がお得」を見せる

**NG：**
- 嘘の限定性（「毎月"今月限定"」→ 信頼を失う）
- 過剰な煽り（「今すぐ！」連発 → 胡散臭くなる）

**【LF8別】最後の背中押しフレーズ：**

限定性・特典・保証は共通だが、**最後の一押しフレーズはLF8で変える**。

【LF8-1（生き残り・人生を楽しみたい）】
- 「1年後、"あの時始めてよかった"と思う自分になる」
- 「退屈な毎日は、今日で終わりにしませんか」
- 「人生を変えるのに、遅すぎることはない」

【LF8-3（恐怖・痛みから逃れたい）】
＜ネガティブ：恐怖を突く＞
- 「"あの時やっておけば"と思う前に」
- 「不安を抱えたまま、あと何年過ごしますか」
- 「今動けば、最悪の未来は避けられる」

＜ポジティブ：始めた自分を肯定＞
- 「"あの時始めてよかった"と思える未来へ」
- 「1年後、"やっておいてよかった"と笑っている自分」
- 「"あの決断が正解だった"と振り返れる日のために」
- 「未来の自分に"ありがとう"と言われる選択を」

【LF8-5（快適・楽したい）】
- 「申し込みは3分。悩んでる時間の方が長い」
- 「今日やれば、明日から考えなくていい」
- 「この面倒を最後にしませんか」

【LF8-6（優越・遅れたくない）】
- 「迷ってる間に、周りは動いている」
- 「同じ悩みを持ってた人は、もう始めてます」
- 「1年後、"なんであの時やらなかったんだろう"と思いたくないなら」

【LF8-7（守りたい）】
- 「大切な人を守れるのは、今のあなただけ」
- 「"パパのおかげ"と言われる未来のために」
- 「家族のために動けるのは、今だけかもしれない」

【LF8-8（認められたい）】
- 「"やっぱり自分には無理だった"で終わりたくないなら」
- 「変われた人と変われなかった人の違いは、最初の一歩だけ」
- 「自分を信じてあげられるのは、自分だけ」

---

## 6大教育の流れ

LPでは以下の流れで教育を行う：

```
1. 目的の教育 → 理想の未来を見せて「欲しい」を作る
　　↓
2. 問題の教育 → 理想を邪魔する壁を自覚させる
　　↓
3. 手段の教育 → この商品が解決策だと納得させる
　　↓
4. 信用の教育 → この人から買う理由を作る（※全体に散りばめる）
　　↓
5. 投資の教育 → この価格が妥当だと納得させる
　　↓
6. 行動の教育 → 今申し込む理由を作る
```

※信用の教育は特定のセクションではなく、LP全体に散りばめる
