# 中身開示型LP設計法

## 核心

**「登録しないと分からない」ではなく「読むだけで分かる」**

LPを読むだけで価値がある。
その上で「もっと詳しく、あなた専用の情報が欲しい人は登録」という流れ。

---

## なぜ中身開示型が売れるのか

### 従来のLPの問題点

```
【従来のLP】
「すごい結果が出ます！」
「詳しくは登録してね」
→ 怪しい、信用できない
→ 登録のハードルが高い
```

### 中身開示型のメリット

```
【中身開示型LP】
「こういう情報があります」（先に見せる）
「読むだけで、ここまで分かります」
「あなた専用の情報は登録後に」
→ 信頼できる、ちゃんとしてる
→ 登録のハードルが下がる
```

### 心理的な仕組み

1. **返報性の原理**
   - 先に価値を提供される → お返ししたくなる
   - 「こんなに教えてくれるなら、登録してもいいか」

2. **信頼構築**
   - 中身を見せる = 自信がある証拠
   - 「ちゃんとしたサービスだ」と感じる

3. **判断材料の提供**
   - 読者が「自分に合うか」判断できる
   - ミスマッチが減る → 質の高いリード

## 中身開示型LPの設計パターン

### パターン1: 知識開示型

**「このLPを読めば、○○が分かる」**

開示する内容：
- 業界の相場・データ
- 判断基準・チェックリスト
- 業界の裏側・リアル
- 成功事例の詳細

例：
```
【転職エージェント】
✅ エンジニアの年収相場・市場価値
✅ 「動くべきか、待つべきか」の判断基準
✅ 業界の裏側・転職市場のリアル
✅ 年収を上げた人のキャリアパス
```

```
【美容サロン】
✅ 施術の種類と効果の違い
✅ 「あなたに合う施術」の選び方
✅ 業界の裏側（効果のない施術とは）
✅ ビフォーアフター事例の詳細
```

---

### パターン2: 体験開示型

**「理想の未来を具体的に見せる」**

開示する内容：
- 施術・サービスの詳細な流れ
- 利用者のリアルな体験談
- 具体的な変化のプロセス
- Before/Afterの詳細

例：
```
【整体院】
施術の流れを完全公開
1. カウンセリング（10分）→ 何を聞かれるか
2. 検査（5分）→ どこを見るか
3. 施術（30分）→ 何をするか
4. アフターケア（5分）→ 何を教えてもらえるか
```

---

### パターン3: ノウハウ開示型

**「これだけで実践できる」レベルまで教える**

開示する内容：
- 具体的な方法論
- ステップバイステップの手順
- 自分でできるセルフケア
- 失敗しないためのポイント

例：
```
【ダイエットコーチング】
自分でできる食事管理の基本
✅ カロリー計算の具体的な方法
✅ 食べていいもの・避けるものリスト
✅ 1週間の献立例
✅ よくある失敗パターンと対策

「これだけでも効果はある。でも、あなた専用のプランが欲しいなら相談を」
```

---

## 「一般論 → あなた専用」の流れ

### 基本構造

```
【LP前半】
一般論を開示する
- 業界の相場
- 判断基準
- 成功事例

【LP後半】
「でも、これは一般論です」
「あなたのケースは、あなた専用の診断が必要」
「あなた専用の○○を知りたい方は、登録を」
```

### 具体例（転職エージェント）

```
【開示する一般論】
- 職種別の年収相場表
- 転職判断のチェックリスト（15項目）
- 業界のリアル

【あなた専用へ誘導】
「でも、あなたのスキル・経験は『あなた専用』」
「あなたの市場価値は、個別に診断が必要」
「無料で、あなた専用の市場価値診断ができます」
```

---

## 開示すべき4つの要素

### 1. 相場・データ
- 業界の平均値
- 成功者の数値
- 比較表

**効果**: 「自分はどこにいるか」が分かる

### 2. 判断基準・チェックリスト
- 「こういう人は○○すべき」
- 「こういう場合は待つべき」
- セルフチェック項目

**効果**: 「自分は該当するか」が分かる

### 3. 業界のリアル・裏側
- 企業が本当に求めていること
- 知られていない真実
- よくある失敗パターン

**効果**: 「この人は本当のことを言ってる」と信頼される

### 4. 成功事例の詳細
- Before/Afterの具体的な数値
- どんなプロセスで変わったか
- その人の背景・状況

**効果**: 「自分も同じ結果が得られるかも」と思える

## 中身開示型LPの構成テンプレート

### FV（ファーストビュー）

```
【キャッチコピー】
悩み・課題を言語化

【サブコピー】
このLPで分かること:
✅ ○○（相場・データ）
✅ ○○（判断基準）
✅ ○○（業界のリアル）
✅ ○○（成功事例）

3分で読める。まずは情報を持とう。
```

---

### セクション1: 相場・データの開示

```
【見出し】
○○の相場、知っていますか？

【内容】
- 具体的な数値データ（表形式）
- 条件別の比較
- あなたの位置確認

【締め】
あなたの○○、相場と比べてどうですか？
```

---

### セクション2: 判断基準・チェックリスト

```
【見出し】
「○○すべきか」の判断基準

【内容】
- チェックリスト（□ 項目1、□ 項目2...）
- 「5つ以上該当したら○○」
- 逆に「こういう人は待ってもいい」

【締め】
あなたは、いくつ該当しましたか？
```

---

### セクション3: 業界のリアル・裏側

```
【見出し】
○○が教えない、業界のリアル

【内容】
- 表向き vs 本音
- 知られていない真実
- よくある失敗パターン

【締め】
これを知っているかどうかで、結果が変わります。
```

---

### セクション4: 成功事例の詳細

```
【見出し】
○○した人は、何をしたのか？

【内容】
- Before（具体的な状況）
- 転機（何がきっかけで動いたか）
- After（具体的な結果）
- ポイント（何が決め手だったか）

【締め】
共通点は「情報を取りに行った」こと。
```

---

### セクション5: 一般論 → あなた専用

```
【見出し】
ここまで読んで、どう感じましたか？

【内容】
✅ 「○○かも...」
✅ 「チェックリスト、該当した...」
✅ 「成功事例の人と状況が似てる」

もしそう感じたなら、
あなた専用の○○を知りませんか？
```

---

### セクション6: サービス紹介

```
【見出し】
あなた専用の○○、無料でできます

【内容】
このLPの情報は「一般論」。
でも、あなたのケースは「あなた専用」。

✅ あなた専用の○○
✅ あなた専用の○○
✅ あなた専用の○○

【CTA】
[登録○分、無料]
まずはあなた専用の○○を知る →
```

## 開示のバランス（最重要）

### 開示しすぎない原則

**全てを開示しない。「一般論」と「あなた専用」を分ける。**

```
【開示の黄金比】
LP上で開示: 70%（読むだけで価値がある）
登録後に開示: 30%（あなた専用の価値）
```

### 開示する vs 開示しない

| 開示する（LP上で） | 開示しない（登録後） |
|------------------|-------------------|
| 一般的な相場・データ | あなた専用の診断結果 |
| 判断基準のチェックリスト | あなたに合った具体的アドバイス |
| 業界の裏側（一般論） | あなたのケースに当てはめた提案 |
| 成功事例のパターン | あなた専用のプラン・戦略 |
| 「こういう人は○○すべき」 | 「あなたは○○すべき」 |

### 開示しすぎのNG例

❌ **全てのノウハウを教えてしまう**
→ 登録する理由がなくなる

❌ **個別診断までLPでやる**
→ サービスの価値がなくなる

❌ **秘密を全部バラす**
→ 「もう十分」と思われる

❌ **具体的な手順を全て書く**
→ 自分でできてしまう

### 正しいバランスの例

```
【転職エージェント】
✅ 開示: 職種別の年収相場表（一般論）
❌ 開示しない: あなたのスキルでの適正年収（あなた専用）

✅ 開示: 転職判断のチェックリスト（一般論）
❌ 開示しない: あなたは今動くべきかの判断（あなた専用）

✅ 開示: 成功事例のパターン（一般論）
❌ 開示しない: あなたに近い事例の詳細紹介（あなた専用）
```

```
【整体院】
✅ 開示: 症状別の原因解説（一般論）
❌ 開示しない: あなたの症状の原因特定（あなた専用）

✅ 開示: セルフケアの基本（一般論）
❌ 開示しない: あなたに合ったセルフケア（あなた専用）

✅ 開示: 施術の流れ（一般論）
❌ 開示しない: あなたに必要な施術プラン（あなた専用）
```

## 注意点

### 1. 一般論は惜しみなく出す
- 「これ無料で教えていいの？」レベルまで出す
- 出し惜しみすると、従来のLPと変わらない
- **ただし、「あなた専用」は出さない**

### 2. 具体的な数値を入れる
- 「年収アップ」→「年収+150万円」
- 「多くの人が」→「78%の人が」

### 3. 「一般論」と「あなた専用」を明確に分ける
- LPで出すのは一般論
- 登録後に出すのはあなた専用
- **この境界線を意識して設計する**

### 4. 信頼できるトーンで書く
- 煽りすぎない
- 事実ベースで書く
- 「これが現実です」というスタンス

### 5. 登録動機を残す
- 「一般論は分かった。で、自分の場合は？」と思わせる
- 「あなた専用の○○を知りたい方は登録」で誘導
- **開示しすぎて「もう十分」と思わせない**

## まとめ

**中身開示型LPの本質:**

```
従来: 「すごいよ！詳しくは登録して」→ 怪しい
中身開示型: 「これが現実です」（先に見せる）→ 信頼される
```

**開示すべき4要素:**
1. 相場・データ
2. 判断基準・チェックリスト
3. 業界のリアル・裏側
4. 成功事例の詳細

**流れ:**
一般論を開示 → 「あなた専用は登録後」に誘導

**結果:**
読むだけで価値がある → 信頼構築 → 登録ハードルが下がる
