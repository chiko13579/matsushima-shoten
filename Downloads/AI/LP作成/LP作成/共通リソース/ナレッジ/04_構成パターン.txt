# LP構成パターン・ライブラリ

## パターンA：【王道】PASONAの法則（問題解決型）
* 特徴: 悩み（Problem）→ 煽り（Agitation）→ 解決策（Solution）
* 向いているケース: 明確な深い悩みがある場合。コンプレックス商材。

## パターンB：【物語】ストーリーテリング型（ヒーローズ・ジャーニー）
* 特徴: 販売者の「失敗→復活」の物語で共感を得る。
* 向いているケース: コンサル、スクール、コーチングなど「人」を売る場合。

## パターンC：【結論】BEAFの法則（メリット先行型）
* 特徴: メリット（Benefit）→ 証拠（Evidence）→ 詳細
* 向いているケース: 指名買い、知名度がある商品、ターゲットが既に買う気満々の場合。

## パターンD：【深い悩み】問題教育2段階型（価格サンドイッチ型）

### 特徴と狙い
* **特徴**: 問題①(さらっと) + 解決策の概要 + やや理想の未来 → 価格 → 問題②(丁寧に深掘り) → 解決策(詳細)
* **向いているケース**: **悩みが深い商品**。継続課金・高単価・効果が見えにくい・競合が多い商材。

### 構成の狙い
- **問題①+解決策の概要**: 「何の商品か」を明確にして離脱を防ぐ（FV直後、価格前）
  - 問題: 「こんな悩みの方に」
  - 解決策の概要: 「○○成分が○○に効く」「12週間で改善報告」
  - やや理想の未来: 「気軽に生活改善」
- **価格**: ここで一度クロージング（「お得だし試そう」で即決層を獲得）
- **問題②**: 「本当に必要?」と迷う人を引き戻す、購入動機を強化（価格後）
  - あるある → 本音 → 葛藤の3レイヤーで深く刺す
- **解決策(詳細)**: 納得した状態で再度クロージング
  - 独自性、開発ストーリー、不安払拭、お客様の声

### 使える商品の条件
1. **継続課金** or **単価が高い**(5,000円以上)
2. **効果が見えにくい** or **競合が多い**
3. **ターゲットが慎重**(情報収集をしっかりする層)

### 具体例
サプリ・青汁・育毛剤・スキンケア定期購入・オンライン英会話・美容機器・コーチング

## パターンD: 詳細構成テンプレート

### 全体の流れ（8ステップ）

```
1. FV: ターゲットワード + 簡単さ + 理想の未来
   ↓
2. 権威: 専門家 + 実績 + エビデンス
   ↓
3. 問題①+解決策の概要+やや理想の未来: 対象者 + 成分・効果・エビデンス + 簡単さ（さらっと5〜10行）
   ↓
4. 価格①: 日割り + 割引 + 特典 + 定期の柔軟性 + 社会的証明
   ↓
5. お客様の声（簡易版）: 3〜5個の短い口コミ
   ↓
6. 問題②: あるある → 本音 → 葛藤（丁寧に深掘り）
   ↓
7. 解決策(詳細): 提案(権威) + 解決策一言+理想 + 特徴 + 独自性 + エビデンス + +α + 不安払拭 + 口コミ詳細
   ↓
8. 価格②+追伸: オファー再提示 + 最後のひと押し + 緊急性 + FAQ
```

---

### ステップ1: FV

**やること:**
1. ターゲットワード（悩みをそのまま言葉に）
2. 簡単さ（「気軽に」「たった○○で」）
3. 理想の未来（どうなれるか）
4. 実績・権威を1つ

**参考:** [03_6_2大教育の参考フレーズ集.txt - 目標の教育 > ファーストビュー]

---

### ステップ2: 権威

**やること:**
1. 専門家の推薦（プロフィール具体的に）
2. 実績（売上、ランキング）
3. エビデンス（科学的根拠、試験データ）

**参考:** [03_6_2大教育の参考フレーズ集.txt - 信頼の教育]

---

### ステップ3: 問題①+解決策の概要+やや理想の未来

**やること（5〜10行でさらっと）:**
1. 対象者（「こんな悩みの方に」）
2. 成分・機能（「○○成分が○○に効く」）
3. エビデンス（「12週間で改善報告」）
4. 簡単さ（「気軽に」「○○するだけ」）

**コツ:** 深く説明しない。専門用語を使いすぎない。

---

### ステップ4: 価格・オファー【投資の教育①】

**やること:**
1. 日割り価格（「1日あたり○○円」で安く見せる）
2. 割引率（「通常○○円→今だけ○○円（○○%オフ）」）
3. 特典（「○○プレゼント」）
4. 定期の柔軟性（「お届けペース変更OK」）
5. 社会的証明（「○人に○人が選んでいます」）

**参考:** [03_6_2大教育の参考フレーズ集.txt - 投資の教育 > 価格提示]

---

### ステップ5: お客様の声（簡易版）【社会的証明】

**やること:**
1. 3〜5個の短い口コミ
2. 「簡単」「美味しい」「続けられる」を中心に
3. 感情の変化を入れる（「心も体も穏やかに」）

**コツ:** ここでは長く書かない（詳細は後で）

---

### ステップ6: 問題②(丁寧に深掘り)【問題の教育 - 本編】

**やること（パターンBの3レイヤー）:**

**レイヤー1: あるある（共感）**
- アンケートで共感（「95%がはい」）
- あるあるを箇条書き

**レイヤー2: 本音（自覚）**
- 危機感の演出（「やばいかも...」）
- データで現実を突きつける

**レイヤー3: 葛藤（無力感）**
- 理想を提示（「1日1万歩」など）
- 葛藤で締める（「分かってはいるけど...」）

**参考:** [02_6大教育マニュアル.txt - 問題の教育 > パターンB（共感型）]

---

### ステップ7: 解決策(詳細)【手段の教育 - 本編 + +αの価値 + 不安払拭】

#### 7-1. 解決策の提示 + 理想の未来

**やること:**
1. 提案(権威)（「○○会社からご提案」）
2. 解決策(一言で) + 理想の未来（「○○で気軽に○○を始めませんか?」）
3. 解決策の特徴・簡単さ（「さっと溶ける」「お茶代わりに」）
4. お客様の声（簡易版）（「こんなふうに変わったよ」を先に見せる）

**コツ:** 変化の実感を先に見せてから、理屈を説明する

#### 7-2. 独自性・仕組み・エビデンス

**やること:**
1. 独自性（「ダブルの○○成分配合」）
2. 仕組みの説明（「○○が○○に効く理由」）
3. エビデンス（「アメリカで話題」「12週間で改善報告」）

**参考:** [02_6大教育マニュアル.txt - 手段の教育 > 独自性の演出]

#### 7-3. +αの価値

**やること:**
1. 追加感の演出（「さらに」「そしてもう一つ」）
2. サポート成分を箇条書きで見せる

#### 7-4. 不安払拭

**やること:**
1. 疑問の先回り（「でも○○って○○そう」）
2. 共感（「美味しくなければ続きませんよね」）
3. こだわりの提示（「とことん○○にこだわりました」）
4. 開発ストーリー（「20回を超える試作」）

**コツ:** 疑問を先に言い、共感してから解決策を見せる

#### 7-5. お客様の声（詳細版）

**やること:**
1. 否定→肯定の驚き（「好きではなかった→でも飲みやすかった」）
2. 具体的な変化（「血圧が下がった」「体脂肪が減った」）
3. 継続意欲（「頑張って続けたい」）
4. 家族への推奨（「主人にもオススメ」）

**コツ:** 半信半疑からの驚きを演出

---

### ステップ8: オファー再提示 + 追伸【投資の教育② + 行動の教育】

#### 8-1. オファー再提示（投資の教育②）

**やること:**
1. 価格・特典を再度提示（ステップ4と同じ内容でOK）
2. リスクリバーサル（「全額保証」など）

#### 8-2. 追伸（行動の教育）

**やること:**
1. 最後のひと押し（「たった○○からできる○○、あなたも始めてみませんか?」）
2. 緊急性（「キャンペーンは予告なく終了」）
3. 背中を押す（「私も半信半疑でした」）

**参考:** [03_6_2大教育の参考フレーズ集.txt - 行動の教育 > クロージング]

#### 8-3. FAQ（よくある質問）

**やること:**
1. 不安を先回りして潰す（5〜8個程度）
2. アレルギー、副作用、併用、飲み方など

## パターンDまとめ: チェックリスト

LPを作る時に、このチェックリストを使って構成を確認:

```
□ 1. FV: ターゲットワード + 簡単さ + 理想の未来
□ 2. 権威: 専門家の推薦 + 実績 + エビデンス
□ 3. 問題①+解決策の概要+やや理想の未来: 対象者(問題) + 成分・効果・エビデンス(解決策の概要) + 簡単さ(やや理想の未来)（さらっと）
□ 4. 価格①: 日割り + 割引 + 特典 + 定期の利便性 + 社会的証明
□ 5. お客様の声（簡易版）: 3〜5個の短い口コミ
□ 6. 問題②: あるある → 本音 → 葛藤（丁寧に）
□ 7. 解決策(詳細): 独自性 + 仕組み + 開発ストーリー + +αの価値 + 不安払拭 + お客様の声（詳細）
□ 8. オファー再提示 + 追伸: 最後のひと押し + 緊急性 + FAQ
```

---

## なぜこの構成が効くのか?

### 即決層と慎重層の両取り
- **即決層**: 問題①→価格で「お得だし試そう」→購入
- **慎重層**: 問題②で「やっぱり必要だ」→解決策で「これが最良」→購入

### 離脱ポイントの防止
- **価格前に離脱**: 問題①で「何の商品か」を明確化
- **価格後に離脱**: 問題②で購入動機を強化

### 納得の積み重ね
1. 問題①: 「何の商品か」分かる
2. 価格①: 「お得だ」と思う
3. 問題②: 「やっぱり必要だ」と思う
4. 解決策: 「この商品が最良だ」と納得
5. 価格②: 「今すぐ買おう」と決断

## パターンE：【記事LP型】体験共感×権威×教育
* 特徴: 体験者視点の記事風LP。広告感を薄め、友人のおすすめのように感じさせる
* 基本構成: FV → 理想の未来 → 権威 → 目標の教育(体験談) → なぜ？ → 問題+手段の教育 → 投資+行動の教育（6ステップ）
* **6大教育と完全連動した構成**

### この型が強い商材
✅ 悩みが深い（コンプレックス系、美容・健康）
✅ 変化が視覚的（ビフォーアフター可能）
✅ 独自性がある（新技術、新成分、独自メソッド）
✅ ターゲットが広告嫌い（40〜60代女性、情報リテラシー高め）
✅ 説明が必要（なぜ効くのか、科学的根拠が重要）

### この型が弱い商材
❌ BtoB、論理的判断が必要な商材
❌ 変化が見えにくい、即効性がない
❌ 差別化ポイントがない（コモディティ）
❌ 比較検討が長い商材（不動産、車など）

---

### 基本構成（6ステップ + 6大教育マッピング）

#### 1. FV（ファーストビュー）
- **3要素の組み合わせ**:
  - 解決策: どうやって解決するか（1本で完了、貼って寝るだけ等）
  - 理想の未来: どうなれるか（〜の味方、〜に革命、人生が変わる等）
  - ターゲットワード: ターゲットが普段思っている言葉（ズボラ、眉間シワ、ほうれい線等）

#### 1-2. 理想の未来セクション（FV直後）【目標の教育 - 導入部】
**基本構成（ほぼ全ての商材で共通）:**

1. **ターゲット呼びかけ**（「〜な人必見」「〜でお悩みの方へ」）
2. **悩みの共感**（「めんどくさい」「気になる」等、ターゲットの言葉で）
3. **話題の演出**（「大ニュース」「話題になっている」）
4. **理想の未来を箇条書きで羅列**（視覚的に「こんなに解決」を見せる）

**※深刻な悩み vs 軽い不便での違い:**
- **深刻な悩み系**: トーンは深刻に、ビフォーアフター画像を入れる
- **軽い不便系**: トーンはカジュアルに、「あるある」感を出す
- **構成は同じ、トーンと表現が変わる**

#### 2. 権威・社会的証明【信頼の教育】
- 売上実績、ランキング、専門家の推薦
- 「雑誌などで話題になっている」感の演出
- 信頼性の構築

#### 3. 理想の未来【目標の教育 - 体験談形式】
**体験者の物語で「こうなれる」を見せる（3ステップ）**

**3-1. 共感（私も悩んでいた）**
- ターゲットの日常的な悩みを代弁
- 「私も同じでした」の親近感
- 悩みの具体化（箇条書き3〜5個）

**3-2. 発見→試す（半信半疑で使ってみた）**
- 偶然の出会い感を演出
- 「でも半信半疑だった」の正直さ
- 実際に試してみる決断

**3-3. 驚き（予想を超える体験）**
- 予想を超える結果
- 感情の起伏を描写
- ビフォーアフターの提示
- **理想の未来を手に入れた証明**

#### 4. 問題の教育【問題の教育】

**※商材によって2つのアプローチがある**
- **深刻な悩み系（パターンA）**: 原因を科学的に説明、自力では無理を証明
- **軽い不便系（パターンB）**: 不便を列挙、「あるある」共感

**※詳細は02_6大教育マニュアル.txt「問題の教育」を参照**
- パターンA/Bの判断基準
- 切り口パターン（「なぜ？」疑問型、「ご存知ですか？」知識提示型、「調べてみた」探求型）
- やること（商材によって異なる）
- 表現方法（Q&A形式、科学的説明、箇条書き、あるある共感など）

#### 5. 手段の教育【手段の教育】
- 独自の解決策を提示（この商品が最良である理由）
- 科学的根拠・開発ストーリー
- 競合との違いを暗に示す（批判はNG）

**※詳細は02_6大教育マニュアル.txt「手段の教育」を参照**

#### 5-2. +αの価値（手段の教育の後）
- 「解決策だけでも十分なのに、さらにこんな良いことも！」
- メイン機能の副次的メリットを提示
- 「そしてもう一つ」「さらに驚いたことに」の追加感

**※詳細は02_6大教育マニュアル.txt「+αの価値」を参照**

#### 6. オファー＋追伸【投資+行動の教育】

**※これは分離不可能なセット構成**

**オファーセクション（投資の教育）の役割:**
- 価格提示＋割引（「通常4,422円→今だけ1,982円」）
- 特典・プレゼント（「雫型スポンジ無料」）
- 保証・リスクリバーサル（「全額返金保証」「色交換無料」）
- 支払い方法の柔軟性（「後払いOK」「クレカ不要」）
- → 価格に対する抵抗を減らす

**追伸セクション（行動の教育）の役割:**
- 緊急性の演出（「キャンペーン予告なく終了」「完売の可能性」）
- 背中を押す共感（「私も半信半疑でした」「もっと早く使えばよかった」）
- 最後のひと押し（「半額以下の"今"が絶対おすすめ」）
- → 「今すぐ」行動させる

**なぜセットなのか:**
- オファーだけ → 「お得だけど、後で考えよう」で離脱
- 追伸だけ → 「急かされてる」と警戒、価格納得なし
- セットで機能 → 「お得×今しかない」で購入決断

**基本の流れ:**
```
オファー（価格納得）→ 追伸（今すぐ行動）→ CTA
```

---

### 成功要因
1. **一人称の体験談形式** → 広告感が薄い
2. **感情の起伏設計** → 読者を引き込む
3. **反論処理が組み込まれている** → 疑問を先回りして潰す
4. **権威と共感の両立** → 信頼性×親近感
5. **教育の充実** → 納得して買える
6. **リスクリバーサル** → 安心して試せる
