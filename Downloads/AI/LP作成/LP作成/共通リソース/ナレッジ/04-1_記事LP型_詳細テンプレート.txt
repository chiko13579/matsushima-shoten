# 記事LP型 詳細テンプレート（北達人コーポレーション型）

## このファイルの使い方
* 04_構成パターン.txt のパターンD【記事LP型】を選んだ場合の詳細テンプレート
* 北達人コーポレーションが複数商材で実証した12ステップの構成
* 美容・健康商材、40〜60代女性向けに特に強い
* 基本の6ステップ（04参照）をさらに細分化した実践版

## 全体構成（12ステップ）

### フェーズ1: 信頼構築（ステップ1〜2）
### フェーズ2: 共感と発見（ステップ3〜4）
### フェーズ3: 体験と探求（ステップ5〜6）
### フェーズ4: 教育（ステップ7〜9）

## ステップ1: ファーストビュー（圧倒的ベネフィット）

### 目的
3秒で「これ私のことだ!」と思わせる

### 構成要素（3要素必須）
1. **解決策**: どうやって解決するか（1本で完了、貼って寝るだけ等）
2. **理想の未来**: どうなれるか（〜の味方、〜に革命、人生が変わる等）
3. **ターゲットワード**: ターゲットが普段思っている言葉（ズボラ、眉間シワ、ほうれい線等）

### 書き方のコツ
- 数字を入れる（「たった10秒」「1本5役」）
- 専門性を出す（「○○専用」「眉間のシワ専用パッチ」）
- 劇的な変化を約束（「貼って寝るだけでシワが激変」）
- 3要素すべてを必ず含める

### 実例（北達人LP）
```
「たった1本でパウダーまで完了。ズボラの味方ファンデ。」
→ 解決策「1本で完了」+ 理想の未来「味方」+ ターゲットワード「ズボラ」

「顔のたるみを引き上げて、寝ている間にじっくり浸透」
→ 理想の未来「たるみを引き上げて」+ 解決策「寝ている間に」+ ターゲットワード「顔のたるみ」
```

## ステップ1-2: 理想の未来セクション（FV直後）

### 目的
FVで興味を持った読者に、さらに「こうなれる」を具体的に見せる

### 基本構成（ほぼ全ての商材で共通）

**4つの要素:**
1. **ターゲット呼びかけ**（「〜な人必見」「〜でお悩みの方へ」）
2. **悩みの共感**（「めんどくさい」「気になる」等、ターゲットの言葉で）
3. **話題の演出**（「大ニュース」「話題になっている」）
4. **理想の未来を箇条書きで羅列**（視覚的に「こんなに解決」を見せる）

### 書き方のコツ

**共通テクニック:**
- ターゲットの口語で書く
- 「でも○○しないといけない」のジレンマを見せる
- 箇条書きで視覚的に「多い→解決」を見せる
- 「話題」で記事感を演出

**悩みの深刻度による調整:**
- **深刻な悩み系**: トーンは深刻に、ビフォーアフター画像を入れる
- **軽い不便系**: トーンはカジュアルに、「あるある」感を出す

### 実例

**実例1（軽い不便 - ズボラ向けファンデ）:**
```
## 手軽に綺麗になりたい人必見  ← ターゲット呼びかけ

化粧ってほんとめんどくさい。でもすっぴんでは外に出れないし。  ← 悩みの共感

実はそんな女性に大ニュースが。
今はズボラの味方ファンデっていうのが話題になっているんです。  ← 話題の演出

なんとこれ1本でベースメイクは丸ごと完了。  ← 理想の未来
- 日焼け止め
- 化粧下地
- ファンデーション
- コンシーラー
- パウダー
```

**実例2（深刻な悩み - 眉間シワ）:**
```
## 眉間の年齢感でお悩みの方へ  ← ターゲット呼びかけ

鏡を見るたび気になる眉間のシワ。メイクでも隠せない...  ← 悩みの共感

そんな方に朗報です。
今、美容業界で革命的なケアが話題になっています。  ← 話題の演出

寝ている間にこんな変化が:  ← 理想の未来
- 眉間のシワが目立たなくなる
- 年齢感が消える
- 自信を持って笑顔になれる
（+ ビフォーアフター画像）
```

## ステップ2: 権威・実績による信頼構築

### 目的
「怪しくない」「信頼できる」と思わせる

### 構成要素
1. **ランキング実績**: 楽天・Amazon・Yahoo 1位
2. **販売実績**: ○○万個突破、ギネス認定
3. **専門家の推薦**: プロのメイクアップアーティストも大絶賛
4. **権威付け**: 医学誌掲載、東証プライム上場企業

### 書き方のコツ
- 具体的な数字を入れる
- 第三者の評価を入れる（ギネス、医学誌など）
- 「世界一」「業界初」など最上級表現を使う（根拠があれば）

### 実例（北達人LP）
```
「世界で一番売れて、ギネス世界記録にも認定されているシリーズの最新作」
「この技術、医学誌に論文が掲載されるほどの実力なんです」
「楽天1位、シリーズ累計790万個突破」
```

## ステップ3: 共感パート（体験者=語り手の登場）

### 目的
「この人、私と同じだ」と感じてもらう

### 構成要素
1. **呼びかけ**: 「突然ですが、○○って本当に面倒ですよね」
2. **悩みの代弁**: 「私も○○に悩んでいました」
3. **悩みの具体化**: 箇条書き（3〜5個）
4. **希望の提示**: 「そんな私たちを救う革命的な○○が話題に」

### 書き方のコツ
- ターゲットの言葉で書く（40代女性なら40代女性の言葉）
- 「私も」「私たち」で仲間意識を作る
- 悩みは具体的に（抽象的な表現は避ける）

### 実例（北達人LP）
```
「突然ですが、毎朝の化粧って時間かかって本当にめんどくさいですよね。
あれこれ塗り重ねるのがめんどくさくてストレス。
- 化粧下地を塗って
- コンシーラーも
- パウダーもしなきゃ
- プライマーいるかなぁ
- ファンデ塗って

そんな私たちを救う革命的なファンデーションが今話題になっているんです。」
```

## ステップ4: 半信半疑→試してみる（感情の起伏）

### 目的
読者の「でも本当?」という疑念を先に言語化する

### 構成要素
1. **疑念の表明**: 「でも正直、これだけで本当?と半信半疑でした」
2. **理由の説明**: 「だって今まで○○しないと無理だったから」
3. **決断の瞬間**: 「でもあまりに話題なので、ついに注文しちゃいました」

### 書き方のコツ
- 「私も疑ってた」の正直さが信頼を生む
- 過去の失敗体験を入れる（「今まで○○してもダメだった」）
- 背中を押された理由を入れる（「でも楽天1位と聞いて」）

### 実例（北達人LP）
```
「だから1本でシミ・くすみ・小じわのカバーができるのか、誰でもプロ級の仕上がりになるって本当?と半信半疑だけど、気になりすぎてついに注文しちゃいました。」

「口コミが盛り上がっているとはいえ、半信半疑な私。さすがに劇的な変化があるわけないし、気休めよね、と思っていたら…」
```

## ステップ5: 衝撃の体験（感動の瞬間）

### 目的
読者に「私も使ってみたい!」と思わせる

### 構成要素
1. **驚きの表現**: 「嘘でしょ?あまりにも良すぎる」
2. **感情の描写**: 「鏡の前で思わず声が出ちゃいました」
3. **具体的な使用感**: サラサラ、ピン、厚塗り感ゼロなど五感で表現
4. **ビフォーアフター**: 画像＋体験者のコメント

### 書き方のコツ
- 感情を大げさに表現（「びっくり」「感動」「衝撃」）
- 五感で表現する（見た目、触感、香りなど）
- 予想との対比を入れる（「思っていたより○○だった」）

### 実例（北達人LP）
```
「嘘でしょ?あまりにも良すぎる
厚塗り感ゼロでとても自然な仕上がり。
プロが仕上げたような均一でムラのない滑らかな印象に、鏡の前で、え?綺麗すぎる!と思わず声が出ちゃいました。」

「びっくり。えっ….これすごい
気休め程度と思っていたのに感動。早く使っておけばと後悔しました。」
```

## ステップ6: 「なぜ?」の疑問提示（興味喚起）

### 目的
教育パートへの自然な流れを作る

### 構成要素
1. **疑問の提示**: 「でもなんでこんなに○○なの?」
2. **調べたい欲求**: 「このファンデ、何者?」
3. **答えの予告**: 「どうやらすごい秘密があるらしい」
4. **次への橋渡し**: 「ネットで調べても分かりにくい。プロに聞くしかない!」

### 書き方のコツ
- 読者の代わりに疑問を言語化
- 「調べたくなる」気持ちを喚起
- 「秘密」「真実」など知りたくなる言葉を使う

### 実例（北達人LP）
```
「でも、さっと塗っただけなのに、なんでこんなに目の大きさが変わるの?と疑問が。
なんでこんなにパッチリ目になれるの?
ネットで調べてもわかりにくい。もうプロに聞くしかない。」

「でもこのオールインワンファンデのすごいところはこれだけじゃなかったんです」
```

## ステップ7: 専門家インタビュー（問題の教育）

### 目的
悩みの「本当の原因」を科学的に理解させる

### 形式の選択
- **パターンA**: インタビュー形式（Q&A）← 教育色が強い時
- **パターンB**: 「調べてわかった驚きの事実」形式 ← 記事感を出したい時

### 構成要素（インタビュー形式の場合）
1. **企業・専門家の紹介**: 東証プライム上場企業、○○のプロ
2. **Q&A形式で教育**:
   - Q. そもそも○○の原因は?
   - A. ○○の減少が原因です
3. **自力では解決できない理由**: 科学的に説明
4. **新しいアプローチの提示**: 「そこで全く別のアプローチを開発したんです」

### 書き方のコツ
- 専門用語を使った後、必ず噛み砕いて説明
- 「自力では無理」を強調（商品の必要性を高める）
- 図解・イラストがあると理解しやすい

### 実例（北達人LP）
```
「訪れたのは北の達人コーポレーションという東証プライム上場企業。

Q: そもそも眉間に年齢感が出る原因は何なんでしょうか?
A: ズバリ、加齢によるヒアルロン酸の減少が原因です
減少は誰も避けられない。でも肌のヒアルロン酸減少を自力で止めることは、今の科学では不可能だったんです。

Q: じゃあ解決はできないんですね
A: そこで全く別のアプローチ方法を開発したんです」
```

## ステップ8: 解決策の教育（手段の教育）

### 目的
「なぜこの商品で解決できるのか」を納得させる

### 構成要素
1. **独自技術・成分の説明**: 図解付きが理想
2. **メカニズムの解説**: ステップごとに説明
3. **他商品との違い**: 比較表を使う
4. **開発ストーリー**: 製薬会社との共同開発など

### 書き方のコツ
- ステップを数字で示す（1→2→3）
- ビジュアルで補完（図解、イラスト）
- 「業界初」「史上初」など革新性を強調

### 実例（北達人LP）
```
「仕上げのフェイスパウダー不要
1. カバー力の高いバーム
2. 肌の上でとろけてリキッドに変化
3. 肌表面がパウダーに再変化

バームからパウダーに変化する、奇跡のテクスチャーが生まれたんです。」

「ヒアルロン酸の針
針一本一本にたっぷり
- ナイアシンアミド
- グルコマンナン
持ち上げた部分にこの針を刺すことで、寝ている間に美容成分が溶け出します。」
```

## ステップ9: 開発ストーリー・こだわり（商品の教育）

### 目的
「この会社・商品なら信頼できる」と思わせる

### 構成要素
1. **お客様の声から生まれた背景**
2. **完売実績**: 何度も完売した（社会的証明）
3. **品質へのこだわり**:
   - 「コスト度外視」「利益より満足度優先」の姿勢
   - 約300項目チェック、国際規格認証など
4. **成分へのこだわり**: 6つのフリー、パッチテスト済みなど

### 書き方のコツ
- 「お客様のために」の姿勢を強調
- 具体的な数字を入れる（300項目、国際規格など）
- 「妥協しない」姿勢を見せる

### 実例（北達人LP）
```
「もともと弊社では目の下に貼るマイクロニードル化粧品を開発して、おかげさまで3度も完売するほど大人気のロングセラー商品になったんです。
するとご愛用者から
『目の下用だけど眉間にも貼ってます』
『眉間専用の商品を作ってほしい』
という声が届くようになりました。」

「コスト度外視です。利益より使う方の満足度が優先。
お客様に使っていただく以上、品質には妥協したくないので
約300項目のチェックをクリアした環境で生産
国際規格の認証を取得した工場から全国にお届けしています。」
```

## ステップ10: 追加の証拠（信頼の積み上げ）

### 目的
「みんな使ってる」「実際に効果が出ている」と思わせる

### 構成要素
1. **口コミ**: 実名+年齢+具体的な感想（5〜10件）
2. **リピート率**: ○○%
3. **感謝の手紙**: 届いている写真
4. **販売数**: ○○万個突破

### 書き方のコツ
- 口コミは具体的に（「良かった」だけはNG）
- ターゲット層に近い年齢の口コミを選ぶ
- ネガティブな感想も少し入れると信頼度UP（「最初は半信半疑だった」など）

### 実例（北達人LP）
```
「サラッとしていてベタつかない。びっくりしました。」

「朝本当に時間がない中でサッとメイクできる。画期的だなと思いました。」

「年齢的に肌ツヤ感が落ちてきたりベースメイクの方が大事。あんまり期待しなかったけど本当に感動しました。メイクするのも楽しくなる。」

「ありがたいことに楽天やAmazonで1位をいただき、総合販売は170万個突破、大ヒットとなりました。リピート率90%」
```

## ステップ11: オファー（抵抗できない条件提示）

### 目的
「今買わない理由がない」と思わせる

### 構成要素
1. **価格の予想外し**: 「1万円すると思いきや4,422円」
2. **割引オファー**: アンケートモニター募集で55%オフ
3. **価値の再定義**: 「1回37円で使える」
4. **特典プレゼント**: スポンジ、サポーター等
5. **リスクリバーサル**:
   - 全額返金保証（使用後でもOK）
   - 色交換無料（ファンデの場合）
   - 定期縛りなし、1回で解約OK
   - 後払い対応
6. **希少性・緊急性**:
   - 「注文殺到中」「キャンペーン予告なく終了」
   - 「過去2度完売した」

### 書き方のコツ
- 価格は必ず「予想より安い」演出をする
- リスクリバーサルは詳しく説明
- 緊急性は煽りすぎない（嘘っぽくなる）

### 実例（北達人LP）
```
「1本でパウダーまで完了できるから1万円以上するのかと思いきや、約2ヶ月分、通常価格4,422円。
これはすごく節約になりそう。

なんとたった1分で終わる簡単なアンケートに協力すると55%オフ。
通常4,422円のハイテクファンデが最安値の1,982円
それだけで1回あたり37円で使えるなんてすごくお得ですよね。

しかも全額返金保証付き
開封したり使ったりしていても全額戻ってくるんだとか。

後払いも選べるから、クレジットカードの入力が不安という方でも心配無用

ただし注意点があって
申し込みが殺到しているため、キャンペーンは予告なく終了する可能性があります。」
```

## ステップ12: 追伸（最後のひと押し）

### 目的
迷っている人の背中を押す

### 構成要素
1. **呼びかけ**: 「まだ少し迷っている方へ」
2. **共感**: 「私も半信半疑でした」
3. **体験談**: 「でも使ってみて本当にびっくり」
4. **後悔の回避**: 「もっと早く使えばよかったって後悔したんです」
5. **緊急性**: 「発売から1年で2度も完売したみたいで、またすぐ売り切れるかもしれません」
6. **最後のひと押し**: 「半額以下の"今"が絶対におすすめです!」

### 書き方のコツ
- 「迷ってる時間がもったいない」を伝える
- 「後悔」ワードを入れる（損失回避の法則）
- 優しいトーンで（説得ではなく、友人のアドバイス風に）

### 実例（北達人LP）
```
「追記:まだ少し迷っている方へ

本当にいいのかなって私も正直半信半疑でした。
でも使ってみて本当にびっくり。
もっと早く使えばよかったって後悔したんです。

実はこれ、発売から1年で2度も完売したみたいで、またすぐ売り切れるかもしれません。
もしほんの少しでも迷っているなら、半額以下の"今"が絶対におすすめです!」
```

## 応用のコツ

### コツ1: インタビュー形式は省略可能
瞼、ファンデのLPはインタビューなし。代わりに「調べてわかった驚きの事実」形式で教育パートを作っている

### コツ2: ターゲット別に表現を変える
- ズボラ向けファンデ: 「化粧ってほんとめんどくさい」
- 通常ファンデ: 「毎朝の化粧って時間かかって本当にめんどくさい」
同じ商材でも、ターゲットの心理状態に合わせて調整

### コツ3: 商材に合わせてステップを調整
全12ステップを必ず守る必要はない。商材の特性に合わせて:
- ステップを統合する
- 順番を入れ替える
- 強調するステップを増やす

### コツ4: 反論処理を組み込む
各ステップで想定される反論を潰していく
- ステップ2: 「本当に効くの?」→ 権威で潰す
- ステップ4: 「信じられない」→ 「私も疑ってた」で潰す
- ステップ11: 「高い」「リスク」→ 割引と保証で潰す

詳細: 05_反論処理フレームワーク.txt 参照
