```
### **4. 問題の教育(なぜ治らないのか)**

**【見出し】**
```
なぜ、腰痛や頭痛が慢性化してしまうのか?
```

**【慢性化のメカニズム】**
```
実は、頭痛・肩こり・腰痛が慢性化してしまうのは
"自律神経の乱れ"が原因です。

【慢性化の流れ】

ストレス・疲労・睡眠不足
 ↓
自律神経が乱れる
 ↓
血流が悪くなる
 ↓
筋肉が固まる
 ↓
頭痛・肩こり・腰痛

ここで、

「整体行こう」
「薬飲もう」

そう思って、一時的に楽になる。

でも、自律神経の乱れはそのまま。

 ↓

また血流が悪くなる
 ↓
また筋肉が固まる
 ↓
また頭痛・腰痛

この繰り返し。

これが「慢性化」の正体です。


だから、

表面(筋肉)だけほぐしても、
薬で痛みを抑えても、

根本の原因(自律神経の乱れ)が治っていないから、
すぐ戻ってしまう。

そして、放置すればするほど、
自律神経の乱れが悪化して、
どんどん慢性化していくんです。
```

**【比較図】**
```
【一般的な整体】
表面(筋肉)だけ → すぐ戻る → 何年も通う

【当院の根本治療】
表面(筋肉) + 内側(自律神経) → 戻りにくい → 6回で改善
```

**【恐怖訴求】**
```
「このまま放置すると…」

実は、慢性化すると取り返しがつかなくなります。

【悪化の流れ】

1年前: 数ヶ月に1回くらいの頭痛
 ↓
今: 月2,3回は頭痛
 ↓
このまま放置すると...
 ↓
半年後: 週1回は頭痛
 ↓
1年後: 毎日頭痛・めまい・吐き気
 ↓
最悪の場合: 仕事も家事もできなくなる


そして、慢性化すればするほど、
治すのに時間とお金がかかります。

【数値で見ると】
早期: 6回で改善(○○円)
慢性化: 18回かかる(○○円 = 3倍)

「まだ我慢できる」

そう思ってる今が、
実は最後のチャンス。


だから、当院は、接骨院×鍼灸で
「表面」と「内側」の両方から治します。

### **5. 手段の教育(当院の解決策)**

**【3ステップ】**

**【STEP1: 接骨院で「表面」を整える】**
```
まずは表面(筋肉)の緊張をほぐします。

・首・肩・腰の筋肉の硬さを緩める
・体のバランスを整える
・即効性あり: 施術後すぐ楽になる

「あ、ラクになった」

施術直後、そう実感できます。
```

**【STEP2: 鍼灸で「内側」を整える】**
```
次に内側(自律神経)を整えます。

・鍼灸で自律神経を調整
・血流を改善
・体の「治る力」を引き出す

※鍼は髪の毛より細いので、痛くないです

「え、これで終わり?」
そう思うくらい、あっという間。
でも、じわじわ効いてきます。
```

**【STEP3: 定期的なメンテナンス】**
```
受けた後のイメージ。

・1回目: 楽になる(即効性)
・3回目: 「あれ?頭痛減ったかも」
・6回目: 「最近頭痛ない」

その後は月1回のメンテナンスで
良い状態をキープ。

「もう頭痛に悩まないんだ」

そう実感できる人が、たくさんいます。
```
