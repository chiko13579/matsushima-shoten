# LF8分析 確定版

## 🎯 **確定したLF8**

### **メインLF8: LF8-7(愛する人を守る・家族)**

**核心インサイト:**
```
「自分のため」では動かない
「家族のため」「大切な人のため」なら動く
```

---

## 🔥 **なぜこれが最強の切り口なのか**

### 1. **ターゲットの行動原理と一致**

```
【現状】
自分のことは後回し
    ↓
家族・仕事を優先
    ↓
「自分はまだ我慢できる」
```

**でも、逆に言えば…**

```
【訴求】
体が悪化すると、家族に迷惑かける
    ↓
家族との時間が奪われる
    ↓
「家族のために、今動かなきゃ」
```

### 2. **罪悪感を刺激する(良い意味で)**

```
「自分のことは後回しでいい」
と思ってる人に対して…

「でも、あなたが倒れたら、
 一番困るのは家族ですよ」

→ ハッとする
→ 「そうだ、家族のためにも元気でいなきゃ」
```

### 3. **優先度を変える**

```
【今までの優先度】
1位: 家族
2位: 仕事
3位: 自分(後回し)

【訴求後の優先度】
1位: 家族
2位: 自分の体(家族のため)← ここが変わる
3位: 仕事
```

**「自分の体を整える = 家族のため」と再定義**

## 💡 **LF8-7を最大化する要素**

### 1. **具体的なシーン描写**

```
【NG: 抽象的】
「家族との時間を大切に」

【OK: 具体的】
「日曜の朝、"パパ、公園行こう!"
 その誘いに、笑顔で"行こう!"と答えられる。
 そんな週末を取り戻しませんか?」
```

### 2. **家族の声を入れる**

```
「パパ、最近元気だね!」(子供)
「ありがとう、楽しかった」(妻)
「お父さん、若返ったね」(家族)

→ 他者の反応が、行動を後押しする
```

### 3. **罪悪感と希望のバランス**

```
【罪悪感(恐怖)】
「"パパ疲れてるから無理"
 また断りますか?」

【すぐに希望】
「でも、今なら間に合います。
 6回で、元気なパパに戻れます」
```
