## 📊 **コンセプト3要素**

Knowledge [00_私の哲学.md:96-141](../../共通リソース/ナレッジ/00_私の哲学.md#L96-L141)

### **Q1: 誰に売るのか?(ターゲット)**

```
30〜50代、家族持ち
頭痛・肩こり・腰痛などの体の不調を抱えている

【心理状態】
・頭痛いのに、我慢している
・体の不調は感じてるけど、「まぁいっか」と思ってる
・忙しいから、後回しにしてる
・「これ、治らないもんだ」と諦めている

【行動パターン】
・行きたい気持ちはある
・でも他のこと(家族・仕事)を優先しちゃう
・きっかけがなくて、行動に移せない
```

**ペルソナ例:**
```
田中太郎(42歳・会社員)
妻・子供2人
朝から頭痛がするけど「まぁいいか」で我慢。
整体行きたいけど、仕事が忙しくてタイミングがない。
休日は家族サービスで、自分のことは後回し。
「頭痛なんて治らないもんだ」と諦めてる。
```

---

### **Q2: 何を売るのか?(商品・価値)**

```
【商品】
接骨院 × 鍼灸

【機能】
・接骨院: 骨格・筋肉を整える
・鍼灸: 自律神経を整える

【独自性】
接骨院と鍼灸を組み合わせることで
「表面」と「内側」の両方から根本治療
```

**数値的証拠:**
```
早期治療: 平均6回で改善
慢性化後: 平均18回かかる
→ 3倍の差
```

---

### **Q3: どう売るか?(切り口・ポジショニング)**

```
【切り口】
「治らないと諦めている頭痛・不調を、
 体の内側から根本治療」

【ポジショニング】
単なる整体・接骨院ではなく、
「内側から治す根本治療院」

※「家族のため」は全面に出さず、匂わせる程度
```

## 🎯 **LF8分析(Knowledge準拠)**

### **メインLF8: LF8-5(快適でいたい)**

**欲求:**
```
【基本の欲求】
頭痛を根本から治して、体が軽くなりたい
```

**証拠:**
- 頭痛を我慢している
- 体の不調を感じている
- 快適な状態に戻りたい

「あっ、今日ラクだ」
そう思える朝が、増えていく。

---

### **サブLF8①: LF8-7(愛する人を守る・家族)**

**核心インサイト:**
```
「自分のため」では動かない
「家族のため」「大切な人のため」なら動く
```

**証拠:**
- 忙しくて他のこと(家族・仕事)を優先
- 自分のことは後回し
- でも、家族のために元気でいたい

---

### **サブLF8②: LF8-1(生存・人生を楽しむ)**

**欲求:**
```
【奥の欲求】
生活の質を上げたい
疲れにくくなりたい
人生を楽しみたい
```

**証拠:**
- 体が重くて楽しめない日がある。とか仕事に集中できない日がある
- 家族との時間を充実させたい

### **3. 手段の教育(独自性)**

Knowledge [02_6大教育マニュアル.txt:403-506](../../共通リソース/ナレッジ/02_6大教育マニュアル.txt#L403-L506)

#### **他との違いを"暗に"示す**
```
【一般的な整体・接骨院】
表面の筋肉をほぐすだけ
    ↓
その場は楽だけど、すぐ戻る
    ↓
自律神経の乱れは放置

【一般的な鍼灸院】
自律神経を整える
    ↓
でも骨格・筋肉の歪みは残る
    ↓
根本解決にならない

【当院:接骨院×鍼灸】
骨格・筋肉を整える + 自律神経を整える
    ↓
表面も内側も整うから、戻りにくい
    ↓
根本から治る
```

#### **科学的説明 + わかりやすい比喩**
```
【なぜ「接骨院×鍼灸」が必要なのか?】

両方同時にアプローチしないと、根本から治らない。

例えば、お花に水をあげるだけでは枯れてしまう。
土の栄養も必要ですよね。

体も同じ。
表面(筋肉)だけほぐしても、
内側(自律神経)が乱れてたら、すぐ戻る。

両方同時に整えるから、
根本から治るんです。
```
