# 6大教育・具体化戦略

**商品:** ホームページ制作サービス（45万円）
**ターゲット:** ホームページ制作経験がある中小企業経営者
**メインLF8:** LF8-3(恐怖回避) - "また失敗したくない"
**サブLF8:** LF8-8(社会的承認) - "会社の顔としてちゃんとしたサイトが欲しい"

---

## 📋 教育戦略の全体フロー

```
FV(興味) → 問題提起(共感) → 実績(希望) → 問題の教育(自覚)
→ 手段の教育(解決策) → 信用の教育(なぜ私から) → +α(お得感)
→ 投資の教育(価格納得) → 行動の教育(今動く理由)
```

---

## 🎯 1. 目的の教育の配置戦略

### 配置箇所
- **FV (1行目)**: 理想の未来を1フレーズで
- **実績紹介セクション**: 具体的な成功事例で"自分もできそう"と思わせる
- **制作の流れセクション**: "3ステップで完成"でハードルを下げる

### 具体的メッセージ
**FVでの表現:**
「ちゃんと作る意味があった」そう思えるホームページを、作りませんか？

**リニューアルで、ダウンロード数が1.52倍に**
「資料ダウンロード数が1.52倍に増加しました。入ってくれる人も増え、ホームページをリニューアルして本当に良かったです」
<自動車学校様 （50代女性・経営者）>

**ホームページとチラシで、初日から満席に**
「オープン前なのに、ホームページとチラシを見て満席になりました。開業初日から予想以上の集客で、大成功でした」
<ケーキ屋様（40代男性・オーナー）>

**ホームページ作ってから、質のいいお問合せが増えた**
「検索で1位になり、そこからの問い合わせが着実に増えています」
<システム会社様 社長>



「集客動線」「競合に勝てる切り口」を整理すると、
ホームページが「ビジネスに使えるもの」

## 🎯 2. 問題の教育の配置戦略

### パターン選択
**パターンA(深刻な悩み系)** を採用
- 理由: LF8-3(恐怖回避)がメインのため
- トーン: 深刻だが、希望も見せる
- 構成: ポジティブ体験談 → 疑問 → 問題の教育

### 配置箇所
- **問題提起セクション**: 失敗経験を代弁(導入)
- **問題の教育本編セクション**: 原因を科学的に説明

**問題の教育本編での表現:**

導入:
```
ホームページって、集客の根幹となる部分。
そもそも、なぜ、集客につながらないホームページがあるのか？

よくあるのが、
「綺麗に作ること」だけに集中してしまうパターン。

**「どうビジネスに役立てるか」この視点が抜け落ちているんです。**
```
ビジネスに役立てるために、大事な視点2つ

原因1: 集客動線を確保する
- 誰も見に来なければ意味がない
- SEO対策、広告設計、SNS連携の設計
- 作っただけで終わる

原因2: 競合に勝てる切り口を作る
- お客さんは必ず競合と比較する
- 「なぜあなたの会社を選ぶのか」が伝わらなければ、選ばれない
- デザインが綺麗なだけでは、心を掴めない

## 🎯 3. 手段の教育の配置戦略

### 配置箇所
- **手段の教育セクション**: メイン展開
- **+αの価値セクション**: 副次的メリット追加



**核心メッセージ:**
```
だからこそ、私たちは「作ること」からではなく、
**「戦略」から始めます。**
```


### 競合との違い(ポジティブに自分の強みを語る)

```
私は、経営者として実際にビジネスをやってきました。
飲食店経営 1店舗で年商1億円まで成長させた経験があります。

そこで、ホームページを何回も改善し、チラシも何回も試行錯誤して、
結果につながりました。

だからこそ、デザインだけでなく、
「どうやって集客するか」「どうやって成果を出すか」
をビジネス視点で設計が可能です。


### +αの価値(追加感の演出)

**「そしてもう一つ」で追加:**
- +α1: 採用活動にも使える
- +α2: 営業資料としても活用できる
- +α3: 会社の信頼性が上がる

**メイン価値との相乗効果:**
```
集客できるホームページがあれば、営業が楽になる。
営業が楽になれば、経営に集中できる。
経営に集中できれば、会社が成長する。
```

## 🎯 4. 信用の教育の配置戦略

### 配置箇所
- **実績紹介セクション**: お客様の声
- **信用の教育セクション**: 開発者ストーリー
- **LP全体に散りばめる**: 権威性・実績

### 開発者ストーリー(共感 → 克服 → 情熱)

**ストーリー構成:**
```
1. 共感: 私も飲食店経営で、最初はお客さんが全く来なかった
2. 克服: チラシとホームページの見せ方を工夫して、年商1億円に
3. 学び: 「集客動線」「競合との差別化」の重要性を身をもって学んだ
4. 情熱: だから、同じことをホームページ制作でも実践している
```

**具体的メッセージ:**
```
私は元々、飲食店を経営していました。
1店舗で年商1億円まで成長させることができましたが、
最初から順調だったわけではありません。

オープン当初、お客さんが全く来ない日々が続きました。
「このままでは潰れる...」

そこで必死に考えたのが、チラシとホームページの見せ方。
どうやってお客さんに来てもらうか。
どうやって心を掴むか。
何度も試行錯誤を繰り返しました。

その結果、ようやくお客さんが来るようになり、
年商1億円まで成長することができました。

この経験から、
「集客動線を設計すること」
「競合と差別化すること」
の重要性を身をもって学びました。

だから、ホームページ制作でも同じことをしています。
デザインだけを作るのではなく、
「どうやってお客さんに来てもらうか」
「どうやって心を掴むか」
を本気で考えて設計する。

それが、私のホームページ制作です。
```


### リターン提示(投資回収の見える化)

```
月に3件、問い合わせが増えたら？

1件の成約で15万円の売上なら、**月45万円の売上アップ**。

**たった1ヶ月で投資回収できます。**
```

### 他の出費と比較

```
営業マン1人雇うと、年間500万円。

ホームページは45万円で、**24時間365日働いてくれます**。
```

### 失敗コストを見せる

```
また失敗したら、また50万円かかります。

それなら最初から、ちゃんと成果が出るものに投資しませんか？
```

### 特典・保証でリスク軽減

**今だけの特典:**
- ✅ 初回相談無料
- ✅ 競合分析レポート プレゼント
- ✅ SEO対策チェックリスト プレゼント

**安心の保証:**
- ✅ 分割払い対応
- ✅ 納品後の支払いでもOK
- ✅ 修正回数無制限（納品前）

### 反論処理の組み込み
- 反論: "45万円は高くないですか？"
- 処理: リターン提示、他の出費と比較、失敗コスト提示
```

## 🎯 LF8との連動戦略

### メインLF8-3(恐怖回避)への訴求

**全体トーン:**
- 深刻だが、希望も見せる
- 過去の失敗を代弁 → 共感
- 原因を説明 → 納得
- 解決策を提示 → 安心

**具体的フレーズ:**
- "次は失敗したくない"
- "お金を無駄にしたくない"
- "また失敗したら、また50万円かかります"
- "あの時やっておけばと思う前に"

### サブLF8-8(社会的承認)への訴求

**具体的シーン:**
- 取引先に「御社のサイト、しっかりしてますね」と言われる
- 名刺交換の時に自信を持ってURLを渡せる
- 会社の印象が変わる
