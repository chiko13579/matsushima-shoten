# 🎁 超簡単！友達にツールを渡す方法

## 方法1: フォルダをそのまま渡す（一番簡単）

### 📂 渡すもの
`my_list_project` フォルダをまるごとコピー

### 🚫 削除してもOKなファイル
- `node_modules` フォルダ（友達が後で作る）
- `.log` で終わるファイル
- `test-` で始まるファイル

### 📋 友達がやること
1. フォルダを受け取る
2. ターミナルを開く
3. 以下のコマンドを実行：

```bash
# 1. フォルダに移動
cd my_list_project

# 2. 必要なものをインストール（最初だけ）
npm install

# 3. ツールを起動
node server.js

# 4. ブラウザで開く
# http://localhost:3000
```

## 方法2: Googleドライブで共有

### 1️⃣ Googleドライブにアップロード
1. [Google Drive](https://drive.google.com) を開く
2. 「新規」→「フォルダのアップロード」
3. `my_list_project` フォルダを選択

### 2️⃣ 共有リンクを作成
1. アップロードしたフォルダを右クリック
2. 「共有」→「リンクを取得」
3. 「リンクをコピー」

### 3️⃣ 友達に送る
コピーしたリンクをLINEやメールで送る

## 方法3: GitHubを使う（エンジニア向け）

### 準備（あなたがやること）
```bash
# 1. GitHubでリポジトリを作成
# 2. 以下のコマンドを実行

git init
git add .
git commit -m "Web制作会社調査ツール"
git remote add origin あなたのGitHubリポジトリURL
git push -u origin main
```

### 友達がやること
```bash
git clone あなたのGitHubリポジトリURL
cd my_list_project
npm install
node server.js
```

## 🔰 友達が初心者の場合

### 必要なソフトをインストール
1. **Node.js** をインストール
   - https://nodejs.org/ から「LTS版」をダウンロード
   - インストーラーを実行（全部「次へ」でOK）

2. **テキストエディタ**（必要なら）
   - Visual Studio Code がおすすめ
   - https://code.visualstudio.com/

### ターミナルの開き方
- **Mac**: Spotlight検索で「ターミナル」
- **Windows**: スタートメニューで「cmd」または「PowerShell」

## ⚠️ よくある質問

**Q: npm installでエラーが出る**
A: Node.jsがインストールされているか確認

**Q: localhost:3000 が開けない**
A: サーバーが起動しているか確認（ターミナルに「サーバーが起動しました」と出てるか）

**Q: Google Sheetsに保存できない**
A: 共有設定を確認（メールアドレスと権限）
