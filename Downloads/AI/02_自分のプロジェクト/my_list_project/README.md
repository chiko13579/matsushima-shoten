# Web制作会社調査ツール

転職活動用のWeb制作会社情報を自動収集するツールです。

## 🚀 クイックスタート

### 1. ツールの実行
```bash
npm start
```

### 2. Google Sheets連携（オプション）
```bash
npm run setup
```

## 📋 取得できる情報

- ✅ URL
- ✅ 会社名  
- ✅ 社長の名前
- ✅ お問合せのURL
- ✅ その会社の強み
- ✅ 制作実績ページの有無

## 💾 保存形式

1. **ローカルファイル** (JSON/CSV)
2. **Googleスプレッドシート**
3. **両方**

## 🛠️ セットアップ

### 基本機能（ローカル保存）
追加設定不要。すぐに使用できます。

### Google Sheets連携
```bash
npm run setup
```

自動セットアップウィザードが起動し、以下を支援します：
- Google Cloud Console の操作ガイド
- ブラウザの自動起動
- 認証ファイルの検証
- 接続テスト

## 📖 使用方法

### 基本的な流れ
1. 場所を入力（例：東京、大阪、名古屋）
2. 自動でWeb制作会社を検索
3. 各社の詳細情報を取得
4. 結果を表示・保存

### Googleスプレッドシート
- **新規作成**: スプレッドシートIDを空白でEnter
- **既存に追加**: スプレッドシートIDを入力

## 🎯 特徴

### 自動化
- Google検索から企業発見
- 各サイトから情報自動抽出
- フォーマット済みデータ出力

### 柔軟な保存
- JSON/CSVファイル
- Googleスプレッドシート直接保存
- 既存シートへの追加

### 使いやすさ
- 対話式ウィザード
- 自動ブラウザ起動
- エラー検証機能

## ⚠️ 注意事項

### 使用制限
- 適切な間隔でアクセス（2秒間隔）
- 企業サイトの利用規約を遵守
- 取得データの適切な使用

### セキュリティ
- `credentials.json` を共有禁止
- Gitリポジトリにコミット禁止
- 認証情報の適切な管理

## 🆘 トラブルシューティング

### よくある問題

#### Google Sheets接続エラー
```bash
npm run setup
```
設定ウィザードで再設定

#### 企業情報が取得できない
- ネットワーク接続確認
- サイトのアクセス制限確認
- 数時間後に再試行

#### ブラウザ起動失敗
- 手動でURLにアクセス
- ブラウザ設定確認
