# お問い合わせ自動送信システム

複数のお問い合わせフォームに自動で入力・送信するツールです。

## 📁 ファイル構成

- `contact_config.json` - 入力内容と設定を管理
- `auto_contact_sender.js` - メインの送信スクリプト
- `send_from_list.js` - URLリストから複数送信
- `analyze_form.js` - フォーム構造を分析（デバッグ用）
- `contact_send_log.csv` - 送信結果のログ
- `contact_urls_sample.csv` - URLリストのサンプル

## 🚀 使い方

### 1. 入力内容を設定

`contact_config.json` を編集して、送信する内容を設定します：

```json
{
  "入力内容": {
    "会社名": "株式会社サンプル",
    "お名前": "山田太郎",
    "メールアドレス": "example@test.com",
    "電話番号": "090-1234-5678",
    "問い合わせ内容": "ホームページ制作について..."
  },
  "設定": {
    "送信間隔秒数": 10,
    "reCAPTCHA待機時間秒数": 60,
    "ヘッドレスモード": false
  }
}
```

### 2. URLリストを準備

CSVファイルにお問い合わせページのURLリストを用意します：

```csv
会社名,URL,備考
サンプル株式会社,https://example.com/contact,
テスト会社,https://test.com/inquiry,
```

**重要:** URL列の名前は「URL」「url」「ホームページ」「サイト」のいずれかを含めてください。

### 3. 送信実行

#### 単一URLにテスト送信
```bash
node auto_contact_sender.js https://example.com/contact
```

#### リストから複数送信
```bash
node send_from_list.js urls.csv 10
```
→ urls.csvから10件送信します

## 🤖 reCAPTCHA対応

- **reCAPTCHAが検出された場合**、自動的にブラウザが開きます
- 設定した待機時間（デフォルト60秒）内に手動で解決してください
- 解決後、自動的に送信処理が続行されます

## 📊 ログ確認

送信結果は `contact_send_log.csv` に記録されます：

- タイムスタンプ
- 送信先URL
- ステータス（成功/エラー）
- メッセージ
- スクリーンショットのパス

## 🔍 フォーム構造の分析（デバッグ用）

フォームの構造を確認したい場合：

```bash
node analyze_form.js
```

※ スクリプト内のURLを変更して実行してください

## ⚙️ 設定項目

### 送信間隔秒数
次のURLに進むまでの待機時間（デフォルト: 10秒）

### reCAPTCHA待機時間秒数
reCAPTCHAを手動で解決する待機時間（デフォルト: 60秒）

### ヘッドレスモード
- `false`: ブラウザを表示（デバッグ・reCAPTCHA対応時に推奨）
- `true`: バックグラウンドで実行（高速だが、reCAPTCHAには非対応）

## 📸 スクリーンショット

送信前後のスクリーンショットが自動保存されます：
- `screenshot_before_[タイムスタンプ].png`
- `screenshot_after_[タイムスタンプ].png`

## ⚠️ 注意事項

1. **利用規約の遵守**: 各サイトの利用規約を必ず確認してください
2. **送信間隔**: サーバーに負荷をかけないよう、適切な間隔を設定してください
3. **メールアドレス**: 実在する連絡可能なアドレスを使用してください
4. **テスト送信**: 本番前に必ず少数でテスト送信してください

## 🐛 トラブルシューティング

### フォームが見つからない
→ `analyze_form.js` でフォーム構造を確認してください

### 入力フィールドが見つからない
→ サイト固有のフィールド名の場合、`auto_contact_sender.js` のキーワードを追加する必要があります

### reCAPTCHAで失敗する
→ `reCAPTCHA待機時間秒数` を長くしてください

## 📝 カスタマイズ

特定のサイトに対応する場合、`auto_contact_sender.js` の `fillFormField` 関数のキーワードを編集してください。
