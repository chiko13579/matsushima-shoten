# 📝 投稿作成テンプレート

このフォルダには、ちこの投稿作成に必要な全ファイルが含まれています。

## 🔄 メインファイル（必須）

### `01_投稿作成フロー.md`
**投稿作成の8ステップを詳細に説明したメインガイド**
- STEP 1-8の詳細な手順
- 全ファイルの参照タイミング
- **投稿作成時は最初にこのファイルを読み込んでください**

### `02_投稿作成マスターガイド.md`  
**投稿作成時に必要な全情報を統合したワンストップガイド**
- 甘い情報・希望の例文集（40個以上）
- デザイン業界特有情報の例文集（30個以上）
- 冒頭強化ガイド（詳細版）
- 最終チェックリスト
- **STEP 4・5・8で参照する重要ファイル**

## 📚 データベースファイル（参照用）

### `03_投稿パターン詳細.md`
**6つの投稿パターンの詳細な構造と必須要素**
- パターン1-6の詳細説明
- 各パターンの必須要素
- STEP 1で投稿種類・パターン選択時に参照

### `04_質問データベース.md`
**ターゲットの質問・悩み・不安の一覧（150個以上）**
- 学習、スキル、営業、単価、メンタル面の質問
- ターゲットの現実的背景情報
- STEP 1で質問選択時に参照

### `05_理想の未来データベース.md`
**ターゲットが憧れる理想の状態一覧**
- 継続案件、高収入、時間の自由など
- 「理想の未来と解決策」パターン使用時に参照

### `06_解決策データベース.md`
**具体的な解決策・手法の一覧**
- 営業、スキルアップ、マインドセットの解決策
- 「理想の未来と解決策」パターン使用時に参照

### `07_不安・共感フレーズデータベース.md`
**ターゲットの内なる声・深い共感フレーズ（100個以上）**
- シチュエーション別の不安フレーズ
- 冒頭の深い共感要素として使用
- より詳細な共感表現が必要な時に参照

## 👤 ちこ関連ファイル

### `08_ちこ体験談データベース.md`
**ちこの実体験・エピソード・価値観**
- 過去の苦労、転換点、現在の成果
- 核となる価値観・信念
- 実体験ベースの制約チェック時に参照

### `09_ちこ口調データベース.md`
**ちこらしい関西弁・表現パターン**
- 「〜なぁって」などの語尾パターン
- 関西弁の自然な使い方
- より詳細な口調調整が必要な時に参照

## 🔗 使用順序

1. **`01_投稿作成フロー.md`** を読み込み、8ステップに従って進行
2. **`02_投稿作成マスターガイド.md`** をSTEP 4・5・8で参照
3. 必要に応じて各データベースファイルから詳細情報を取得

## ⚠️ 重要な注意点

- **投稿作成マスターガイドが最重要**：甘い情報、業界特有情報、冒頭強化、チェックリストが全て含まれる
- **1ファイル完結を優先**：できるだけマスターガイドで完結させ、詳細が必要な時のみ他ファイルを参照
- **ファイル間の一貫性**：全ファイルの情報は相互に矛盾しないよう設計されている
