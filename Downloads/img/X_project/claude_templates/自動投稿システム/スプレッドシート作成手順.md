# 📊 Googleスプレッドシート作成手順

## 🎯 STEP 1: 新しいスプレッドシート作成

### 1-1. Google Sheetsにアクセス
1. **https://sheets.google.com** にアクセス
2. 「空白のスプレッドシート」をクリック
3. ファイル名を「完全自動化SNS管理システム」に変更

### 1-2. シート構成の作成

#### シート1: 投稿管理
以下の列を作成：

| A | B | C | D | E | F | G | H |
|---|---|---|---|---|---|---|---|
| ID | 投稿内容 | 投稿種類 | ステータス | 投稿予定日時 | 実際投稿日時 | 投稿URL | メモ |

#### シート2: リプライ管理
以下の列を作成：

| A | B | C | D | E | F |
|---|---|---|---|---|---|
| ID | 対象ツイートURL | リプライ内容 | ステータス | 実際日時 | メモ |

#### シート3: アカウント管理
以下の列を作成：

| A | B | C | D | E | F | G |
|---|---|---|---|---|---|---|
| アカウント名 | API Key | API Secret | Access Token | Access Secret | 月間投稿数 | 制限状況 |

## 🔧 STEP 2: シートの詳細設定

### ヘッダー行のフォーマット
1. 1行目を選択
2. 太字にする（Command + B）
3. 背景色を薄いグレーに設定
4. 中央揃えに設定

### データ検証の設定

#### D列「ステータス」の設定
1. D列全体を選択
2. データ > データの入力規則
3. 条件: リストから選択
4. アイテム:
   - 承認済み
   - 投稿完了
   - エラー
   - 削除

## 📱 STEP 3: スプレッドシートID取得

### URLからIDを抽出
スプレッドシートのURLは以下の形式：
```
https://docs.google.com/spreadsheets/d/SPREADSHEET_ID/edit#gid=0
```

`SPREADSHEET_ID`の部分をコピーして保存してください。

例：
```
https://docs.google.com/spreadsheets/d/1ABC123XYZ789/edit#gid=0
↓
SPREADSHEET_ID: 1ABC123XYZ789
```

## 🔐 STEP 4: Google Sheets API有効化

### 4-1. Google Cloud Consoleでの設定
1. **https://console.cloud.google.com** にアクセス
2. 新しいプロジェクト作成または既存プロジェクト選択
3. 「APIとサービス」→「ライブラリ」
4. 「Google Sheets API」を検索して有効化

### 4-2. サービスアカウント作成
1. 「APIとサービス」→「認証情報」
2. 「認証情報を作成」→「サービスアカウント」
3. サービスアカウント名: 「n8n-sheets-access」
4. 作成後、キーをJSONでダウンロード

### 4-3. スプレッドシートの共有設定
1. スプレッドシートで「共有」をクリック
2. サービスアカウントのメールアドレスを追加
3. 権限: 「編集者」

## 📝 STEP 5: テストデータ入力

### サンプルデータの追加
「投稿管理」シートに以下のテストデータを入力：

| ID | 投稿内容 | 投稿種類 | ステータス | 投稿予定日時 | 実際投稿日時 | 投稿URL | メモ |
|----|----------|----------|------------|---------------|---------------|----------|------|
| 1 | テスト投稿です | 不安払拭 | 承認済み | 2024/01/15 10:00 | | | テスト用 |
