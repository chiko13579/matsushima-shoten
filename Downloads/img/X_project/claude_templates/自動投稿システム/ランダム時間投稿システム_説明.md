# ⏰ ランダム時間投稿システム

## 🎯 システム概要

### 基準時間とランダム変動
```
基準時間: 7時、11時、20時
ランダム変動: ±5分（-5分〜+5分）
```

### 実際の投稿時間例
```
7時の投稿:
- 6:55 〜 7:05 の間でランダム

11時の投稿:
- 10:55 〜 11:05 の間でランダム

20時の投稿:
- 19:55 〜 20:05 の間でランダム
```

## 🔧 システムの仕組み

### STEP 1: 定期チェック
- **5分毎**に投稿タイミングをチェック
- 6時〜22時の間で動作

### STEP 2: ランダム時間計算
- 各基準時間に対して**±5分**のランダム変動を生成
- 例: 7時 → 6:58、7:03、7:01 など

### STEP 3: 投稿実行判定
- 現在時刻がランダム時間の**±2分以内**なら投稿実行
- 1日1回のみ各時間帯で投稿

## ✨ メリット

### 自然な投稿パターン
- **機械的でない**: 毎回異なる時間で投稿
- **人間らしい**: 多少の時間のズレが自然

### アルゴリズム対策
- **パターン検知回避**: 定時投稿の機械的パターンを回避
- **アカウント安全性向上**: より人間らしい行動

### フォロワー体験向上
- **予測困難**: フォロワーが投稿時間を予測しにくい
- **サプライズ要素**: 意外な時間の投稿で注目度アップ
