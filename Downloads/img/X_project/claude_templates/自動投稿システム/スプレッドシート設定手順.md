# 📊 Googleスプレッドシート設定手順

## 1. 新しいスプレッドシート作成
1. https://sheets.google.com にアクセス
2. 「空白」をクリックして新規作成
3. ファイル名を「X投稿管理システム」に変更

## 2. テンプレートデータのインポート
1. 「ファイル」→「インポート」
2. 「アップロード」タブで `スプレッドシート_テンプレート.csv` をアップロード
3. 「データをインポート」をクリック

## 3. データ検証（プルダウン）の設定

### D列（ステータス）のプルダウン設定
1. D2からD1000まで選択（D2をクリック→Shift+Ctrl+Endで範囲選択）
2. 「データ」→「データの入力規則」
3. 条件：「リストから範囲」を選択
4. 項目に以下を入力：
   ```
   下書き
   承認待ち
   承認済み
   投稿完了
   エラー
   ```
5. 「完了」をクリック

### C列（投稿種類）のプルダウン設定
1. C2からC1000まで選択
2. 「データ」→「データの入力規則」
3. 項目に以下を入力：
   ```
   不安払拭
   質問回答
   理想の未来
   テスト
   その他
   ```
4. 「完了」をクリック

## 4. 日時フォーマットの設定
1. E列（投稿予定日時）を選択
2. 「表示形式」→「数字」→「日時」
3. フォーマット：「yyyy/mm/dd hh:mm」を選択

## 5. 条件付き書式の設定（見やすくするため）
1. D列全体を選択
2. 「表示形式」→「条件付き書式」
3. 以下の条件を追加：
   - 承認済み → 薄い緑色
   - 投稿完了 → 濃い緑色  
   - エラー → 薄い赤色
