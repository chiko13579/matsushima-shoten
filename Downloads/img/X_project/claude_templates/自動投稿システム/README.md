# 🤖 自動投稿システム

このフォルダには、X投稿の自動化に必要なファイルがまとまっています。

## 📁 ファイル構成

### 🔧 設定ファイル
- **`スプレッドシート_テンプレート.csv`** - Googleスプレッドシート用テンプレート
- **`n8n_workflow.json`** - n8nワークフロー設定（インポート用）

### 📖 設定手順書
- **`スプレッドシート設定手順.md`** - Googleスプレッドシートの詳細設定
- **`X_Developer設定手順.md`** - X Developer アカウント・API設定  
- **`n8n設定手順.md`** - n8n基本設定・認証設定
- **`n8nワークフロー設定手順.md`** - ワークフロー構築・テスト手順

### 📚 運用資料  
- **`運用マニュアル.md`** - 日常運用・トラブルシューティング・保守管理

## 🎯 システム概要

### 自動化の流れ
```
投稿作成 → スプレッドシート入力 → 承認 → n8n自動実行 → X投稿 → 結果更新
```

### 機能
- ✅ 15分間隔の自動チェック
- ✅ 承認ワークフロー管理
- ✅ エラーハンドリング
- ✅ 投稿履歴の自動記録

## 🚀 セットアップ手順

### 1. 事前準備（30分）
1. `X_Developer設定手順.md` → X Developer アカウント作成
2. `スプレッドシート設定手順.md` → Googleスプレッドシート作成

### 2. n8n設定（20分）  
1. `n8n設定手順.md` → n8nアカウント作成・基本設定
2. `n8nワークフロー設定手順.md` → ワークフロー構築

### 3. テスト・運用開始（10分）
1. テストデータで動作確認
2. ワークフローを「Active」に設定
3. `運用マニュアル.md` に従って運用開始
