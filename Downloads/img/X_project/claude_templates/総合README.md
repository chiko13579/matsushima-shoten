# 🚀 X投稿完全自動化システム

このフォルダには、投稿作成から自動投稿まで、X運用を完全自動化するシステムが含まれています。

## 📁 フォルダ構成

### 📝 投稿作成システム
**高品質な投稿コンテンツを効率的に作成**
- 8ステップの投稿作成フロー
- 6つの投稿パターン詳細設定
- 豊富なデータベース（質問・共感フレーズ・体験談など）
- 自然で読者に刺さる投稿が短時間で作成可能

### 🤖 自動投稿システム  
**作成した投稿を完全自動でX投稿**
- Googleスプレッドシートでの投稿管理
- n8nによる15分間隔の自動チェック・投稿
- 承認ワークフローで誤投稿防止
- エラーハンドリング・投稿履歴管理

## 🎯 システム全体の流れ

```
1. 投稿作成システムで高品質コンテンツ作成
   ↓
2. スプレッドシートに投稿内容を入力
   ↓  
3. ステータスを「承認済み」に変更
   ↓
4. n8nが自動でチェック・X投稿
   ↓
5. 投稿結果を自動でスプレッドシートに記録
```
