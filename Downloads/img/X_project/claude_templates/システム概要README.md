# 🚀 X自動投稿システム 完全パッケージ

このフォルダには、n8nを使ったX(Twitter)自動投稿システムの構築に必要な全ファイルが含まれています。

## 📁 ファイル構成

### 🔧 設定ファイル
- `スプレッドシート_テンプレート.csv` - Googleスプレッドシート用テンプレート
- `n8n_workflow.json` - n8nワークフロー設定（インポート用）

### 📖 手順書
- `スプレッドシート設定手順.md` - Googleスプレッドシートの詳細設定
- `X_Developer設定手順.md` - X Developer アカウント・API設定  
- `n8n設定手順.md` - n8n基本設定・認証設定
- `n8nワークフロー設定手順.md` - ワークフロー構築・テスト手順

### 📚 運用資料  
- `運用マニュアル.md` - 日常運用・トラブルシューティング・保守管理

## 🎯 システム概要

### 機能
- Googleスプレッドシートでの投稿管理
- 承認ワークフロー（下書き→承認待ち→承認済み→投稿完了）  
- 15分間隔の自動チェック・投稿
- エラーハンドリング・ログ記録
- 投稿URL・実行時刻の自動記録

### 処理フロー
```
投稿作成 → スプレッドシート入力 → 承認 → n8n自動実行 → X投稿 → 結果更新
```

## 🚀 クイックスタート

### 1. 事前準備（30分）
1. `X_Developer設定手順.md` に従ってX Developer アカウント作成
2. `スプレッドシート設定手順.md` に従ってGoogleスプレッドシート作成

### 2. n8n設定（20分）  
1. `n8n設定手順.md` に従ってn8nアカウント作成・基本設定
2. `n8nワークフロー設定手順.md` に従ってワークフロー構築

### 3. テスト実行（10分）
1. スプレッドシートにテストデータ入力
2. n8nでワークフロー手動実行  
3. X投稿・スプレッドシート更新を確認

### 4. 本格運用開始
1. ワークフローを「Active」に設定
2. `運用マニュアル.md` に従って日常運用開始
