# 📝 投稿作成システム

このフォルダには、X投稿のコンテンツ作成に必要なファイルがまとまっています。

## 📁 ファイル構成

### 🎯 メインファイル
- **`01_投稿作成フロー.md`** - 投稿作成の8ステップ手順
- **`02_投稿作成マスターガイド.md`** - 甘い情報・冒頭強化・チェックリスト
- **`03_投稿パターン詳細.md`** - 6つの投稿パターンの詳細設定

### 📊 データベース
- **`04_質問データベース.md`** - ターゲットからの質問集
- **`05_理想の未来データベース.md`** - 読者の憧れる理想の状態
- **`06_解決策データベース.md`** - 問題解決のための具体的方法
- **`07_不安・共感フレーズデータベース.md`** - 読者の心に刺さる共感フレーズ
- **`08_ちこ体験談データベース.md`** - ちこの実体験エピソード
- **`09_ちこ口調データベース.md`** - ちこらしい自然な語り口

### 📄 作成した投稿
- **`new_post.md`** - 最新の投稿内容（作成後に更新される）

## 🚀 使い方

### 基本の流れ
1. **`01_投稿作成フロー.md`** で8ステップの手順を確認
2. **`03_投稿パターン詳細.md`** で投稿の種類を選択
3. 各データベースから必要な要素を選択
4. **`02_投稿作成マスターガイド.md`** でチェックリスト確認
5. **`new_post.md`** に完成した投稿を保存

### 投稿の種類
- **不安払拭・励まし系**: 120-200文字
- **質問回答系**: 300-500文字  
- **理想の未来系**: 250-400文字

### 投稿作成のポイント
✅ 読者の心理に深く共感する冒頭
✅ 具体的で実践的な内容
✅ 「甘い情報」で希望を与える
✅ 制作会社への配慮を含める
✅ 自然な会話調の文章

## 🎨 投稿作成例

### STEP 1: 投稿種類選択
「ターゲットの不安を払拭する」を選択

### STEP 2: 共感フレーズ選択  
「営業しなきゃって分かってるけど断られるのが怖すぎて手が動かない...」

### STEP 3: 体験談構成
過去の同じ悩み → 今振り返っての気づき → 希望のメッセージ

### STEP 4: 完成
199文字の自然な励まし投稿が完成！
