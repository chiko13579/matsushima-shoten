# ターゲット心理テンプレ（冒頭1文目の深い共感用）

このファイルは投稿の冒頭1文目で使用する「深い共感フレーズ」のデータベースです。
ターゲットが心の中でつぶやいている本音を、そのまま冒頭に使用できる形で収録しています。

  - 「みんなのと比べて明らかにレベル低すぎる」

### 人間心理パターン

【楽して稼ぎたい心理】
- 「私だけが知らない秘密の方法があるのかも」
- 「もっと効率的なやり方があるはず、私は遠回りしてる」
- 「コツコツ頑張ってるのに報われない」

【優越感を得たい心理】
- 「他の人は気づいてないけど実は私もわかってた」
- 「みんな苦労してるけど私だけいい方法見つけた」

【不安を解消したい心理】
- 「私だけじゃなかった、みんな同じこと思ってるんだ」
- 「失敗してもいいんだ、普通のことなんだ」
- 「今のやり方で合ってるって言ってもらいたい」

### 認知的不協和パターン

【行動できない時の本音】
- 「営業しなきゃって分かってるけど断られるのが怖すぎて手が動かない」
- 「値上げしなきゃ生活できないけど『高い』って言われたら終わりな気がする」
- 「毎日勉強してるのに成果が出ないけど何が間違ってるのか分からない」

## 活用ポイント

### 投稿作成時のフロー（強化版）
1. **シチュエーション選択**：ターゲットに合う状況を選ぶ
2. **禁断情報でフック**：「実は〜」「本当のところ〜」で注意を引く
3. **内なる声で共感**：「みんな思ってるけど言えないこと」を代弁
4. **解決情報提供**：「それ知りたかった！」と思わせる具体的情報
5. **優越感提供**：「これを知ってる人は少ない」感を演出
6. **希望ある締め**：ちこらしい優しい応援で終わる

### バズ投稿での注意点
- **冷たい分析ではなく、体験談ベース**で書く
- **攻撃的になりすぎない**：ちこの優しさを保つ
- **具体的な数字やエピソード**を含める
- **最初の2行にターゲットキーワード**を必ず入れる
- **「楽して稼ぎたい」心理に寄り添う**：努力論より効率論
- **「みんな知らない情報」感**を演出
